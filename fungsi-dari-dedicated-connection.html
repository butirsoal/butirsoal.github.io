11. Jenis sambungan komunikasi data (internet) kita mengenal istilah dedicated dan broadband. yang dimaksud dengan dedicated adalah...


Bagi Anda pengguna dedicated server, Anda tidak perlu menghawatirkan resiko gangguan atau tindakan negatif dari pengguna lainnya. Karena biasanya, resiko ini akan diterima oleh mereka yang menggunakan shared hosting atau mereka yang berbagi server. Sehingga tidak menutup kemungkinan pengguna lain akan mengacaukan pengaturan yang sudah dilakukan.

Berbeda dengan dedicated server, karena satu server hanya bisa dikontrol oleh satu orang atau perusahaan. Resiko ini dijamin tidak akan pernah dirasakan.

 

Bisa dimanfaatkan untuk mengontrol data

Salah satu manfaat yang satu ini bisa dikatakan sebagai manfaat yang sangat menguntungkan pengguna dedicated server. Mengingat user bisa dengan leluasa mengolah data, maka mereka akan secara otomatis bisa mendapatkan akses penuh dalam pengelolaan dan juga pengaturan database yang akan dan sudah diinput.

Biasanya penyedia layanan dedicated server akan memberikan pelayanan berupa fitur remote akses yang bertujuan memudahkan user dalam melakukan kontrol. Sehingga mereka tidak perlu lagi datang untuk mengawasi rak server fisik mereka secara langsung.

 

Bisa memiliki akses administrasi

Pada user yang menggunakan dedicated server, mereka akan mendapatkan akses root administrator yang biasanya diberikan oleh si penyedia layanan sebagai salah satu keuntungan terbaik.

Sehingga user dapat memanfaatkan fasilitas ini untuk melakukan installing pada program aplikasi server yang mana biasanya dilakukan bersamaan dengan konfigurasi custom. Jadi user juga bisa melakukan berbagai pengaturan yang dibutuhkan dan diinginkan.

 

Akses lebih aman dan juga efektif

Tentu kita tahu bahwa hal yang harus kita perhatikan ketika kita mempertimbangkan penyedia layanan dedicated server yang akan kita pilih, adalah dari tingkat kenyamanan dan keamanan yang mereka tawarkan.

Salah satunya adalah kita bisa mendapatkan keuntungan berupa pemecahan masalah apabila dikemudian hari terdapat gangguan yang tidak bisa diselesaikan sendiri. Sehingga kita bisa meminta pertanggung jawaban dari mereka, karena sudah menjadi fasilitas yang akan kita dapatkan.

 

Bisa mendapatkan dukungan technical support yang memadai

Ketika kita melakukan penyewaan pada dedicated server, kita secara otomatis akan mendapatkan prioritas dalam hal technical support. Apalagi biasanya, mereka akan memberikan dukungan teknisi yang lebih berpengalaman dibandingkan dengan mereka yang menyewa shared hosting. Sehingga melalui teknisi yang berpengalaman ini, Anda akan mendapatkan berbagai solusi yang lebih bermanfaat.

 

Terhindar dari gangguan pengguna lain

Bagi Anda pemilik perusahaan besar, sangat dianjurkan jika Anda ingin memilih dedicated server sebagai salah satu layanan yang akan dimanfaatkan untuk mengatur website perusahaan.

Hal ini dilakukan agar Anda tidak merasa resah apabila website tiba-tiba mengalami penurunan kualitas / down. Karena melalui dedicated server, Anda tidak mungkin diganggu oleh user nakal yang akan menyerang server perusahaan menggunakan file-file berbahaya seperti malware.

Sekian informasi yang bisa diberikan seputar dedicated server dan keuntungan yang akan Anda dapatkan sebagai user. Jangan ragu untuk memilih, karena dedicated

22. perbedaan dedicated dan embedded


Dedicated Computer adalah sebuah komputer yang dibuat untuk tujuan tertentu saja, sebagai contoh, video game. Video game dibuat dengan berbagai ukuran, mulai dari yang terkecil, yang dapat dipegang oleh tangan dengan mudah dan menggunakan baterai, maupun ukuran besar, yang harus diletakkan di rumah dengan ukuran sebesar televisi. Umumnya video game dilengkapi dengan joystick dan beberapa tombol. SEDANGKAN Embedded computer adalah komputer yang berfungsi sebagai unit kontrol dan ditempatkan di peralatan yang membutuhkan pengendalian otomatis. Biasanya, embedded computer mempunyai sebuah mikroprosesor tunggal. Beberapa peralatan yang mempunyai embedded computer di dalamnya adalah telepon, videotape recorder, pendingin ruangan, mobil yang dilengkapi sistem perapian yang dikendalikan, dan sebagainya. Selain itu, embedded computer juga membantu untuk mengendalikan penerbangan pesawat, mengendalikan satelit-satelit agar tetap pada orbitnya, dan mengendalikan peluru kendali agar menemukan target yang dituju. Embedded computer juga ditempatkan untuk mengendalikan gerakan dari robot-robot yang digunakan di industri.

34. Connection oriented protocolnya apa ? dan connection less protocolnya apa ?


Connection-Oriented Protocol (COP) adalah protokol jaringan yang digunakan untuk membuat sesi komunikasi data di mana perangkat endpoint menggunakan protokol awal untuk membangun koneksi ujung ke ujung dan kemudian aliran data berikutnya disampaikan dalam mode transfer sekuensial.

Protokol tanpa koneksi (connectionless protocol) adalah bentuk transmisi data di mana sinyal IT keluar secara otomatis tanpa menentukan apakah penerima siap, atau bahkan apakah ada penerima.

36. 1. Perbedaan dedicated file server dengan non dedicated file server?.


Jawaban:

Perbedaan utama antara file server yang didedikasikan dan yang tidak didedikasikan adalah ketersediaan sumber daya. File server yang didedikasikan memiliki seluruh sumber daya yang ditentukan untuk tujuannya, sementara file server yang tidak didedikasikan berbagi sumber daya dengan tugas lain dalam jaringan.


Penjelasan:

File server yang didedikasikan hanya digunakan untuk menangani transaksi dan kegiatan file server, dan memiliki seluruh sumber daya yang diperlukan untuk melakukannya. Sementara itu, file server yang tidak didedikasikan berbagi sumber daya dengan aplikasi dan tugas lain dalam jaringan, membatasi performa dan pertumbuhan file server.
