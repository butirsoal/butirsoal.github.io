1. Jelaskan kegunaan dari perintah shortcut Ctrl+C, ctrl+P, ctrl+N, ctrl+V, dan Ctrl+S!


1) Ctrl+C (Copy) digunakan untuk mengcopy (menyalin) data, biasanya
   diiringi dengan Ctrl+V (Paste) untuk menempel (mewujudkan) salinan.
2) Ctrl+P (Print) digunakan ketika ingin memunculkan perintah print untuk
     memgeprint data
3) Ctrl+N (New) digunakan untuk membuka (membuat) data baru
4) Ctrl+S (Save) digunakan untuk menyimpan data baru atau data yang
   sudah diperbarui

Semoga Bermanfaat, Trim's

22. Bagaimana cara untuk memotong dokumen slide dengan menggunakan keyboard maka menekan … ctrl s ctrl a ctrl v ctrl l ctrl p


Jawaban:

Untuk memotong dokumen slide dalam program presentasi seperti Microsoft PowerPoint atau Google Slides, Anda dapat menggunakan fitur pemotongan (cut) yang tersedia di dalam program. Caranya adalah dengan memilih teks atau elemen yang ingin dipotong, lalu tekan tombol keyboard "Ctrl + X" untuk memotong dan "Ctrl + V" untuk menyisipkan (paste) ke tempat lain.

Namun, jika Anda ingin memotong dokumen slide di luar program presentasi, misalnya dengan menggunakan software pengolah gambar atau editor teks, caranya mungkin berbeda-beda tergantung pada program yang Anda gunakan.

Penjelasan:

Biasanya, Anda dapat menggunakan kombinasi tombol "Ctrl + X" untuk memotong, "Ctrl + V" untuk menyisipkan, dan "Ctrl + A" untuk memilih seluruh dokumen slide. Namun, pastikan untuk memeriksa dokumentasi atau menu bantuan program yang Anda gunakan untuk mengetahui cara yang tepat.

Pembahasan

Di dalam menjalankan sebuah aplikasi misalnya kita ambil contoh pada microsoft excel, maka akan terdapat banyak toolbars yang akan kita gunakan untuk membuat sebuah action atau perintah.

Pada umumnya kita menggunakan mouse dan melakukan perintah dengan klik kanan atau pun klik kiri. Namun untuk sebagian orang maka kita dapat menggunakan jalan pintas (Shortcut) untuk beberapa action sebagaimana contoh berikut di bawah ini:

Ctrl + c = Untuk menduplikat data.Ctrl + v = Untuk menempel (paste) data yang tadi kita copy.Ctrl + a = Untuk melakukan seleksi pada semua bagian.Ctrl + s = untuk menyimpan file.Ctrl + o = untuk membuka file.Ctrl + n = untuk membuat document baru.

Selain pada microsoft excel maka shortcut tersebut bisa diaplikasikan pada aplikasi lainnya.

39. macam macam ctrl dan kegunaannya


Ctrl + A : [ALL] Fungsi mem-Blok keseluruhan teks
Ctrl + B : [BOLD] Fungsi Menebalkan teks
Ctrl + C : [COPY] Fungsi meng- Copy teks
Ctrl + D : Fungsi membuka menu/kotak dialog Font
Ctrl + E : [EQUATOR] Fungsi rata tengah teks
Ctrl + F : [FIND] Fungsi membuka menu/kotak dialog Find and Replace
Ctrl + G : [GO TO] Fungsi membuka menu/kotak dialog Find and Replace
Ctrl + H : Fungsi membuka menu/kotak dialog Find and Replace (tab Replace)
Ctrl + I : [ITALIC] Fungsi membuat teks menjadi miring
Ctrl + J : [JUSTIFY] Fungsi rata kanan & kiri pada teks
Ctrl + K : Fungsi membuka menu/kotak dialog Insert Hyperlink
Ctrl + L : [LEFT] Fungsi rata kiri pada teks
Ctrl + M : Fungsi menggeser First Line Indent dan Left Tab pada mistar
Ctrl + N : [NEW] Fungsi membuka dokumen baru
Ctrl + O : [OPEN] Fungsi membuka menu/kotak dialog Open
Ctrl + P : [PRINT] Fungsi membuka menu/kotak dialog Print
Ctrl + Q : Fungsi menghilangkan nomor urut
Ctrl + R : [RIGHT] Fungsi membuat teks menjadi rata kanan
Ctrl + S : [SAVE] Fungsi menyimpan dokumen
Ctrl + T : Fungsi menggeser Left Tab pada mistar dokumen
Ctrl + U : [UNDERLINE] Fungsi menggarisbawahi teks
Ctrl + V : [PASTE] Fungsi menampilkan hasil teks yang sudah di Copy
Ctrl + W : Fungsi menyimpan dan langsung menutup program ms. Office
Ctrl + X : [CUT] Fungsi menghapus teks (Teks terlebih dahulu di-Blok)
Ctrl + Y : [REDO] Fungsi mengulang kembali aktivitas yang dibatalkan
Ctrl + Z : [UNDO] Fungsi membatalkan aktivitas terakhir
