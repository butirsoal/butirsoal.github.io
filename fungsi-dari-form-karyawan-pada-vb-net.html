2. Sebutkan struktur program dalam VB net​


Jawaban:

Sebelum kita mempelajari blok bangunan dasar bahasa pemrograman VB.Net, mari kita lihat struktur program VB.Net yang minimal sehingga kita dapat menganggapnya sebagai referensi dalam bab-bab yang akan datang.

Program VB.Net pada dasarnya terdiri dari beberapa bagian berikut:

Deklarasi namespace

Kelas atau modul

Satu atau lebih prosedur

Variabel

Prosedur utama

Pernyataan & Ekspresi

Komentar

Mari kita lihat sebuah kode sederhana yang akan mencetak kata-kata "Hello World":

1

2

3

4

5

6

7

8

Imports System

Module Module1

'This program will display Hello World

Sub Main()

Console.WriteLine("Hello World")

Console.ReadKey()

End Sub

End Module

Bila kode diatas dikompilasi dan dieksekusi, maka menghasilkan hasil sebagai berikut:

Hello, World !

Mari kita lihat berbagai bagian dari program di atas:

Baris pertama dari program Imports System digunakan untuk memasukkan namespace System dalam program ini.

Baris berikutnya memiliki deklarasi Modul, Modul Modul1. VB.Net benar-benar berorientasi objek, jadi setiap program harus berisi modul kelas yang berisi data dan prosedur yang digunakan program Anda.

Kelas atau Modul umumnya mengandung lebih dari satu prosedur. Prosedur berisi kode yang dapat dieksekusi, atau dengan kata lain, mereka mendefinisikan perilaku kelas. Prosedur bisa berupa salah satu dari berikut ini:

Function

Sub

Operator

Set

AddHandler

RemoveHandler

RaiseEvent

Baris berikutnya ('Program ini) akan diabaikan oleh compiler dan telah ditambahkan komentar tambahan di program ini.

Baris berikutnya mendefinisikan prosedur utama, yang merupakan titik masuk untuk semua program VB.Net. Prosedur Utama menyatakan apa yang akan dilakukan modul atau kelas saat dijalankan.

Prosedur Utama menentukan perilakunya dengan pernyataan tersebut.

Console.WriteLine ("Hallo World") WriteLine adalah metode kelas Console yang didefinisikan dalam namespace Sistem. Pernyataan ini menyebabkan pesan "Hallo, World!" Untuk ditampilkan di layar. Baris terakhir Console.ReadKey () adalah untuk Pengguna VS.NET. Ini akan mencegah layar agar tidak berjalan dan menutup dengan cepat saat program diluncurkan dari Visual Studio.NET.

Compile & Execute Program VB.Net:

Jika Anda menggunakan Visual Studio.Net IDE, lakukan langkah-langkah berikut:

Mulai Visual Studio.

Pada bilah menu, pilih File, New, Project.

Pilih Visual Basic From Template

Pilih Console Application

Tentukan nama dan lokasi proyek Anda dengan menggunakan tombol Browse, kemudian pilih tombol OK

Proyek baru muncul di Solution Explorer.

Tulis kode di Editor Kode.

Klik tombol Run atau tombol F5 untuk menjalankan proyek. Jendela Command Prompt muncul yang berisi baris Hello World.

Anda dapat mengkompilasi sebuah program VB.Net dengan menggunakan command line daripada Visual Studio IDE:

Buka editor teks dan tambahkan kode yang disebutkan di atas.

Simpan file sebagai helloworld.vb

Buka alat command prompt dan masuk ke direktori tempat Anda menyimpan file.

Ketik vbc helloworld.vb dan tekan enter untuk mengkompilasi kode Anda.

Jika tidak ada kesalahan dalam kode Anda, command prompt akan membawa Anda ke baris berikutnya dan akan menghasilkan file eksekusi helloworld.exe.

Selanjutnya, ketik helloworld untuk menjalankan program anda.

Anda akan dapat melihat "Hello World" yang tercetak di layar.

Apa Itu Visual Basic .NET?

vb .net adalah

Seperti yang sudah disebutkan sebelumnya, Visual Basic .NET merupakan bahasa pemrograman yang bersifat object-oriented.

Menurut Funtech, pemrograman object-oriented berbasis konsep objek.

Jadi, struktur data atau objek dalam pemrograman ini memiliki atributnya masing-masing.

Bahasa pemrograman VB .NET dikembangkan dari Visual Basic (VB) yang sudah terlebih dahulu diciptakan dan digunakan oleh Microsoft.

Awalnya, VB .NET dikembangkan pada tahun 2002 untuk menyempurnakan Visual Basic 6.0.

Meskipun baru belajar, kamu akan bisa membuat aplikasi yang berfungsi dengan baik.

Kelebihan dan Kekurangan VB .NET

Kelebihan

Guru99 mengatakan bahwa hal-hal berikut ini adalah kelebihan dari menggunakan VB .NET:

Kekurangan

VB .NET alias Visual Basic .NET memiliki beberapa kekurangan yang perlu dipahami saat dipelajari, yaitu:

Pointer sulit digunakan untuk proses pemrograman.

Coding tambahan bisa menghabiskan cycle CPU lebih banyak sehingga proses jadi lambat.

VB .NET mudah di-decompile (reverse engineer) karena menggunakan Intermediate Language (IL) compilation.

Library-nya yang lengkap justru membuat waktu komputasi jadi lebih lama.

Microsoft Visual Basic .NET adalah sebuah alat untuk mengembangkan dan membangun aplikasi yang bergerak di atas sistem .NET Framework, dengan menggunakan bahasa BASIC. Dengan menggunakan alat ini, para programmer dapat membangun aplikasi Windows Forms, Aplikasi web berbasis ASP.NET, dan juga aplikasi command-line. Alat ini dapat diperoleh secara terpisah dari beberapa produk lainnya (seperti Microsoft Visual C++, Visual C#, atau Visual J#), atau juga dapat diperoleh secara terpadu dalam Microsoft Visual Studio .NET. Bahasa Visual Basic .NET sendiri menganut paradigma bahasa pemrograman berorientasi objek yang dapat dilihat sebagai evolusi dari Microsoft Visual Basic versi sebelumnya yang diimplementasikan di atas .NET Framework. Peluncurannya mengundang kontroversi, mengingat banyak sekali perubahan yang dilakukan oleh Microsoft, dan versi baru ini tidak kompatibel dengan versi terdahulu.

Apakah Visual Basic .NET dianggap sebagai sebuah versi Visual Basic atau benar-benar bahasa yang berbeda merupakan sebuah topik perdebatan yang hangat. Hal ini dikarenakan sintaksis bahasa Visual Basic .NET tidak mengalami perubahan yang sangat drastis, dan hanya menambahkan beberapa dukungan fitur baru seperti penanganan eksepsi secara terstruktur dan ekspresi yang bisa di-short-circuit-kan. Dua perubahan tipe data pun terjadi saat berpindah ke Visual Basic .NET. Dibandingkan dengan Visual Basic 6.0, tipe data Integer yang dimiliki oleh Visual Basic .NET memiliki panjang dua kali lebih panjang, dari 16 bit menjadi 32 bit. Selain itu, tipe data Long juga sama-sama berubah menjadi dua kali lipat lebih panjang, dari 32 bit menjadi 64 bit. Bilangan bulat 16-bit dalam Visual Basic .NET dinamakan dengan Short. Lagi pula, desainer GUI Windows Forms yang terdapat di dalam Visual Studio .NET atau Visual Basic .NET memiliki gaya yang sangat mirip dengan editor form Visual Basic klasik.

Jika sintaksis tidak banyak yang berubah, lain halnya dengan semantik, yang berubah secara signifikan. Visual Basic .NET merupakan sebuah bahasa pemrograman yang mendukung fitur "Bahasa Pemrograman Berorientasi Objek" secara penuh, karena memang didukung oleh arsitektur Microsoft .NET Framework, yang mengandung kombinasi dari Common Language Runtime dan Base Class Library. Visual Basic klasik, hanya merupakan sebuah bahasa pemrogaman berbasis objek, yang berjalan di atas arsitektur Component Object Model (COM).

Perubahan ini telah mengubah banyak asumsi tentang hal yang benar yang harus dilakukan dengan mempertimbangkan performa dan kemudahan untuk dipelihara. Beberapa fungsi dan pustaka perangkat lunak, yang ada di dalam Visual Basic klasik, kini tidak terdapat di dalam Visual Basic .NET; mungkin masih banyak yang masih terdapat di dalam Visual Basic .NET, tetapi tidak seefisien apa yang ditawarkan oleh .NET Framework. Bahkan jika program Visual Basic klasik bisa dikompilasi dengan benar, sebagian besar program Visual Basic klasik harus melalui beberapa proses refactoring untuk mengadopsi fitur bahasa baru secara keseluruhan. Fungsi net dalam permainan bulu tangkis adalah sebagai pembatas area permainan. Dengan adanya net, pemain bisa menentukan tinggi rendahnya pukulan, servis, dan serangan kepada lawan.

Pembahasan

Dalam permainan bulu tangkis, terdapat berbagai macam peralatan dan perlengkapan untuk dapat melakukan permainan bulu tangkis. Diantaranya adalah.

1. Lapangan.

Lapangan yang dilakukan dalam permainan bulu tangkis berberntuk persegi panjang, dengan lebar 6,10 meter dan panjang 13,40 meter. Lapangan bulu tangkis biasanya berada di dalam ruangan yang tidak terdapat angin agar laju bola bisa stabil.

2. Net.

Net adalah pembatas area permainan berupa jaring. Dengan adanya net, pemain bisa menentukan tinggi rendahnya pukulan. servis, dan serangan terhadap awan.

3. Bola atau kok.

Kok atau bola yang digunakan dalam permainan bulu tangkis terbuat dari bulu angsa dengan berat 5, 67 gram. Terdapat sebuah gabus dibagian bawah sebagai tempat pukulan.

4. Raket.

Raket dalam permainan bulu tangkis terbuat dari almunium, grafit, maupun karbon. Raket dibuat seringan mungkin tapi kuat agar memudahkan pemain untuk melakukan pukulan.

5. Dekker.

Dalam permainan bulu tangkis, deker merupakan peralatan penting yang digunakan untuk menghindarkan pemain dari cidera. Namun, bagi sebagian pemain, dekker tidak harus digunakan karena dapat mengganggu pergerakan pemain.
