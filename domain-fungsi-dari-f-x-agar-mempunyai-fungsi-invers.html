2. Domain fungsi (Df) dari f(x) = x² + 2x - 3agar mempunyai fungsi invers adalah....Mohon bantuannya ya​


Domain fungsi [tex]f(x)=x^2+2x-3[/tex] agar mempunyai fungsi invers adalah E. {x|x, x ≤ -1, x ∈ R} dan {x|x, x ≥ -1, x ∈ R}.

PEMBAHASAN

Fungsi invers adalah sebuah fungsi yang berkebalikan dari fungsi asalnya. fungsi invers dari [tex]f(x)[/tex] dilambangkan dengan [tex]f^{-1}(x)[/tex]. Suatu fungsi memiliki invers jika fungsi tersebut adalah fungsi bijektif (fungsi satu-satu), dimana setiap anggota di domain memiliki tepat satu pasangan di kodomain.

.

DIKETAHUI

[tex]f(x)=x^2+2x-3[/tex]

.

DITANYA

Tentukan domain fungsi f(x) agar mempunyai fungsi invers.

.

PENYELESAIAN

Kita cek terlebih dahulu apakah fungsi kuadrat f(x) termasuk fungsi bijektif atau bukan. Misal kita ambil x = -3 dan x = 1.

[tex]f(-3)=(-3)^2+2(-3)-3[/tex]

[tex]f(-3)=0[/tex]

.

[tex]f(1)=(1)^2+2(1)-3[/tex]

[tex]f(1)=0[/tex]

.

Karena f(-3) = f(1) maka f(x) bukan fungsi bijektif, sehingga kita perlu membatasi domain fungsinya agar menjadi fungsi bijektif. Kita batasi pada garis sumbu simetrinya. Mengapa? Karena sumbu simetri membagi parabola menjadi 2 bagian yang sama. Sehingga domain fungsi f(x) akan tepat memiliki satu pasangan di kodomain nya.

[tex]f(x)=x^2+2x-3[/tex]

a = 1

b = 2

c = -3

Diperoleh sumbu simetrinya :

[tex]x_p=-\frac{b}{2a}[/tex]

[tex]x_p=-\frac{2}{2(1)}[/tex]

[tex]x_p=-1[/tex]

.

Sehingga domain fungsinya kita batasi menjadi x ≤ -1 dan x ≥ -1.

.

KESIMPULAN

Domain fungsi [tex]f(x)=x^2+2x-3[/tex] agar mempunyai fungsi invers adalah E. {x|x, x ≤ -1, x ∈ R} dan {x|x, x ≥ -1, x ∈ R}.

.

Jadi, invers f(x) adalah f-¹(x) = 2 ± √(4 - x) dengan domain x ≤ 4.

Jawaban:

Diberikan dua fungsi yaitu f(x) = 1+x² dan g(x) = 1/(1-x), tentukan:

a. fungsi komposisi fog dan domainnya

b. fungsi invers dari f dan g

## Penyelesaian

a. Fungsi komposisi fog dapat dicari dengan mengganti nilai x pada fungsi g(x) dengan fungsi f(x). Sehingga, fungsi komposisi fog adalah:

fog(x) = f(g(x)) = f(1/(1-x)) = 1 + (1/(1-x))^2

Untuk menentukan domain dari fungsi komposisi fog, kita perlu memperhatikan domain dari fungsi f dan g. Domain dari f adalah seluruh bilangan real, sedangkan domain dari g adalah semua bilangan real kecuali 1. Oleh karena itu, domain dari fungsi komposisi fog adalah semua bilangan real kecuali 1.

b. Fungsi invers dari f(x) dapat dicari dengan menukar nilai x dan y pada fungsi f(x), kemudian menyelesaikan persamaan tersebut untuk y. Sehingga, fungsi invers dari f(x) adalah:

f^-1(x) = √(x-1)

Untuk menentukan fungsi invers dari g(x), kita perlu menukar nilai x dan y pada fungsi g(x), kemudian menyelesaikan persamaan tersebut untuk y.

x = 1/(1-y)

1-y = 1/x

y = 1 - 1/x

Sehingga, fungsi invers dari g(x) adalah:

g^-1(x) = 1 - 1/x

22. Diketahui fungsi grafik tersebut adalah f(x) = x² - 8x + 20 dan titik puncaknya berada di titik (4,4). Agar grafik fungsi tersebut mempunyai invers maka tentukan syarat domain dari fungsi tersebut.


Untuk memastikan bahwa fungsi tersebut memiliki invers, fungsi tersebut harus bersifat satu-satu (injektif) di seluruh domainnya. Dalam kasus ini, fungsi kuadrat f(x) = x² - 8x + 20 memiliki puncak di (4, 4), yang menunjukkan bahwa fungsi tersebut terbuka ke atas.

Untuk memastikan sifat injektif, kita harus memastikan bahwa fungsi tersebut selalu meningkat atau selalu berkurang di seluruh domainnya. Fungsi kuadrat terbuka ke atas (seperti f(x) = x²) akan selalu meningkat di setiap sisi titik puncaknya, yang dalam hal ini adalah (4, 4).

Sehingga, syarat domain untuk memastikan fungsi tersebut memiliki invers adalah seluruh himpunan bilangan real, yaitu (-∞, ∞).

34. tentukan domain (Df) dan range fungsi (Rf) dari f(x) = x²-x, dengan menggunakan invers​


Jawaban:

Jadi, domain (Df) dari f(x) = x² - x adalah x ≥ -0.25 dan range (Rf) adalah y ≤ 0.5 atau y ≥ 0.

Penjelasan dengan langkah-langkah:

Untuk menentukan domain (Df) dan range (Rf) dari fungsi f(x) = x² - x menggunakan invers, kita perlu mengambil invers dari fungsi tersebut terlebih dahulu.

Langkah pertama adalah mengganti f(x) dengan y:

y = x² - x

Langkah kedua adalah menukar tempat antara x dan y:

x = y² - y

Langkah ketiga adalah menyelesaikan persamaan kuadrat tersebut untuk mendapatkan invers dari fungsi f(x).

x = y² - y

y² - y - x = 0

Kita dapat menggunakan rumus kuadrat atau melengkapi kuadrat untuk menyelesaikan persamaan ini:

(y - 0.5)² - (0.5)² - x = 0

(y - 0.5)² = x + 0.25

y - 0.5 = ±√(x + 0.25)

Menggunakan tanda plus:

y - 0.5 = √(x + 0.25)

y = √(x + 0.25) + 0.5

Menggunakan tanda minus:

y - 0.5 = -√(x + 0.25)

y = -√(x + 0.25) + 0.5

Sehingga invers dari fungsi f(x) adalah:

g(x) = √(x + 0.25) + 0.5 dan h(x) = -√(x + 0.25) + 0.5

Sekarang kita dapat menentukan domain (Df) dan range (Rf) dari fungsi f(x) menggunakan invers ini.

Untuk g(x) = √(x + 0.25) + 0.5, domainnya adalah x ≥ -0.25 karena nilai di dalam akar kuadrat harus ≥ 0.

Untuk Rf, tidak ada batasan pada akar kuadrat sehingga Rf adalah semua bilangan real positif y ≥ 0.

Untuk h(x) = -√(x + 0.25) + 0.5, domainnya juga adalah x ≥ -0.25.

Untuk Rf, akar kuadrat negatif memberikan nilai negatif, jadi Rf adalah semua bilangan real y ≤ 0.5.

Jadi, domain (Df) dari f(x) = x² - x adalah x ≥ -0.25 dan range (Rf) adalah y ≤ 0.5 atau y ≥ 0.
