Fungsi database

21. Cara menggunakan fungsi statistik database


Praktikum_6 (Fungsi STATISTIK DATABASE © Ari Wibowo 200921FUNGSI STATISTIK DATABASE Apa Fungsi Statistik Database?Adalah suatu fungsi yang antara lain digunakan untuk menghitung, menjumlah atau mencari nilai tertentu pada suatu database dengan menggunakan syarat-syarat (criteria) tertentu. Fungsi ini berbeda dengan fungsi statistik yang telah dibahas sebelumnya. Perbedaanya adalah jika pada fungsi statistik biasa tidak menggunakan syarat atau kriteria tertentu. Rumus Statistik Database1.Fungsi DSUM Rumus: =DSUM(database,field,criteria) 2.Fungsi DCOUNT Rumus: = DCOUNT(database,field,criteria) 3.Fungsi DMAX Rumus: = DMAX(database,field,criteria) 4.Fungsi DMIN Rumus: = DMIN(database,field,criteria) 5.Fungsi DAVERAGE Rumus: = DAVERAGE(database,field,criteria) 6.Fungsi SUMIF Rumus: = SUMIF (range,criteria, [sum_range]) 7.Fungsi COUNTIF Rumus: = COUNTIF(range,criteria) Keterangan: database : tabel databasenya (termasuk judul), field : nomor kolom yang dikenai fungsi, dan kriteria : tabel syarat Keterangan: range : jangkauan criteria : syarat [sum_range] : jangkauan yang dijumlahkan
di gunakan untuk berhitung atau lain sebagainya seperti layaknya calculator ataupun alvalink

33. Jelaskan perbedaan pembuluh alteri dan vena


1. Vena;

Terletak dipermukaan kulit, mengalirkan darah ke jantung (darah mengandung karbondioksida)

, dinding pembuluh tipis dan tidak elastis, jika terluka, darah menetes, warna kebiru-biruan, memiliki katup di sepanjang pembuluh.

2. Arteri:

Berdinding tebal, kuat dan elastis, terletak terembunyi, denyut terasa, jika terluka, darah memancar, mengalirkan darah dar jantung (darah mengandung oksigen).

Fungsi Agregasi/Agregates

Fungsi yang dihasilkan dari sebuah group data. 

1.        SUM : Untuk menjumlah suatu kolom tertentu yang telah definisikan dalam perintah select.

       Sintak : SELECT SUM (nama_kolom) FROM nama_tabel

Ex: SELECT SUM (SKS) FROM Mata_Kuliah

2.        COUNT : Untuk menghitung jumlah baris yang telah diinputkan dalam sebuah tabel

       Sintak : SELECT COUNT (nama_kolom) FROM nama_tabel
