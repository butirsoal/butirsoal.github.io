4. tiga buah mesin yaitu A, B dan C bekerja sehari dapat memproduksi 233 koper. jika yang bekerja hanya mesin A dan B dapat memproduksi 170 koper sehari. jika yang bekerja mesin B dan C dapat memperoduksi 158 koper sehari. jika mesin A dan C yang bekerja, banyak koper yang diperoduksi sehari adalah


Kelas : 10
Mapel : Matematika
Kategori : Sistem Persamaan Linear
Kata Kunci : Metode eliminasi dan Substitusi
Kode : 10.2.3 (Kelas 10 Matematika Bab 3 - Sistem Persamaan Linear)

Pembahasan :

Tiga buah mesin yaitu A, B dan C bekerja sehari dapat memproduksi 233 koper. jika yang bekerja hanya mesin A dan B dapat memproduksi 170 koper sehari. jika yang bekerja mesin B dan C dapat memperoduksi 158 koper sehari. jika mesin A dan C yang bekerja, banyak koper yang diperoduksi sehari adalah ....

Dari soal di atas diketahui :
A + B + C = 233 ..... (1)
A + B = 170 ............ (2)
B + C = 158 ............ (3)

Ditanyakan : A + C = .... ?

Jawab :

Eliminasi (1) dan (2)
A + B + C = 233
A + B = 170
------------------------- --
C = 233 - 170
C = 63

Eliminasi (1) dan (3)
A + B + C = 233
B + C = 158
------------------------- --
A = 233 - 158
A = 75

A + C = 75 + 63 = 138

Jadi banyak koper yang diperoduksi sehari jika mesin A dan C yang bekerja adalah 138 koper

Jawab:

Konsep dasar

Soal ini dapat diselesaikan menggunakan konsep perbandingan senilaiPerbandingan senilai adalah perbandingan dari dua variabel atau lebih yang memiliki nilai sama. Memiliki nilai sama artinya ketika nilai suatu variabel dinaikkan maka variabel lain juga akan naik nilainya.Misalkan 2 variabel m dan n, kedua variabel memiliki perbandingan senilai jika memenuhi persamaan:

        [tex]\frac{m_{1} }{m_{2} } =\frac{n_{1} }{n_{2} }[/tex]

Untuk perbandingan berbalik nilai, persamaan yang memenuhi adalah:

         [tex]\frac{m_{1} }{m_{2} } =\frac{x_{2} }{n_{1} }[/tex]

Penjelasan dengan langkah-langkah:

Dari soal diketahui memiliki 3 variabel, yaitu:

  variabel jumlah mesin (M)

  variabel jumlah produksi (P)

  variabel waktu produksi (T)

Agar lebih mudah mengerjakan, ambil 2 variabel terlebih dahulu untuk dibandingkan, misalnya variabel jumlah mesin dengan variabel jumlah produksi.

Kedua variabel tersebut memiliki perbandingan senilai karena secara logika semakin ditambah mesinnya maka produksinya juga akan semakin bertambah.

Buat tabel untuk kedua variabel tersebut:

  Jumlah mesin      jumlah produksi     waktu produksi

       1 mesin                   0,5 kodi                   3 jam

       5 mesin                   x kodi                      3 jam

Nilai x dapat dihitung menggunakan konsep perbandingan senilai:

     [tex]\frac{M_{1} }{M_{2} } =\frac{P_{1} }{P_{2} }[/tex]

     [tex]\frac{1}{5} =\frac{0,5}{x}[/tex]

     [tex]x=5.(0,5)[/tex]

     [tex]x=2,5[/tex]

Jadi untuk 5 mesin dapat diproduksi sebanyak 2,5 kodi selama 3 jam

Jumlah produksi untuk 5 mesin dan selama 5 jam dapat diperoleh dengan membandingkan variabel jumlah produksi dengan jumlah waktu produksi, yang juga menggunakan konsep perbandingan senilai.

Buat tabel lagi untuk kedua variabel tersebut

  Jumlah produksi      waktu produksi

       2,5 kodi                      3 jam

       y kodi                         5 jam

Nilai y dapat dihitung menggunakan konsep perbandingan senilai:

     [tex]\frac{P_{1} }{P_{2} } =\frac{T_{1} }{T_{2} }[/tex]

     [tex]\frac{2,5}{y} =\frac{3}{5}[/tex]

     [tex]3y=5.(2,5)[/tex]

     [tex]y=\frac{12,5}{3}[/tex]

    [tex]y=4\frac{1}{6}[/tex]

Sehingga jumlah produksi untuk 5 mesin dan 5 jam waktu produksi adalah [tex]4\frac{1}{6}[/tex] kodi atau [tex]80\frac{1}{3}[/tex] buah (dibulatkan menjadi 80 buah)
