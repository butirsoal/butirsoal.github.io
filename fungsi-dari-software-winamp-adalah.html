Dengan trik dan tool tertentu, WinAmp dapat me-ripping ke MP3, menjadi alat bursa tukar musik di Internet, bahkan bekerja sama dengan iPod.

27. 7 Software dapat dibagi berdasarkan fungsinya, salah satunya adalah system software tuliskan fungsi dari sistem software​


Jawaban:

Fungsi Software

Software memiliki fungsi dalam mengatur hardware/perangkat keras yang ada pada komputer. Dengan begitu, komputer yang digunakan dapat bekerja dengan baik sesuai pekerjaan apa yang dibutuhkan. 2. Software juga dapat digunakan sebagai penghubung antara beberapa software yang lain dengan hardware komputer

Penjelasan:

Terserah deh :)

Fungsi dari system software adalah sebagai “jembatan“ atau interface antara perangkat keras komputer dengan aplikasi, yang mengendalikan dan mengelola pengoperasian perangkat keras komputer sesuai instruksi dari aplikasi yang digunakan.
 

Pembahasan

System software adalah software yang “menjembatani” atau sebagai penghubung/interface antara perangkat keras komputer dengan aplikasi. System software mengendalikan dan mengelola pengoperasian perangkat keras komputer, sesuai dengan instruksi dari perangkat lunak aplikasi (user application). Dengan demikian, dapat dikatakan pula bahwa system software berfungsi untuk menjamin program-program aplikasi berjalan dengan benar/semestinya.

Contoh dari system software adalah sistem operasi (operating system), seperti Microsoft Windows, OS X, Linux, Unix, dsb.

33. fungsi fungsi software tersebut antara lain...​


Fungsi fungsi software antara lain :

Untuk membuat sebuah komputer dapat menjalankan perintah dari user.Menyediakan fungsi dasar dari sebuah komputer sehingga dapat dioperasikan.  Mengatur setiap perangkat keras yang ada di komputer sehingga dapat bekerja secara simultan.  Menjadi penghubung antara beberapa perangkat lunak lain dengan hardware yang ada di komputer. Sebagai penerjemah suatu perintah perangkat lunak lainnya ke dalam bahasa mesin, sehingga dapat dimengerti oleh hardware. Dapat mengidentifikasi suatu program yang ada pada sebuah komputer.Pembahasan  

Software ialah suatu bagian dari sistem komputer yang tidak mempunyai wujud fisik dan tidak terlihat karena merupakan sekumpulan data elektronik yang disimpan dan diatur oleh komputer berupa yang program sehingga dapat menjalankan suatu perintah.

Software berdasarkan jenisnya :

Operating System atau sistem operasi Programming Language atau Bahasa Pemrograman Application Program atau Program Aplikasi
