Pembahasan

Komputer adalah perangkat elektronik yang berfungsi untuk mengolah data berdasarkan prosedur yang telah dirumuskan agar menghasilkan informasi yang diinginkan oleh penggunanya. Agar komputer dapat berfungsi sebagaimana mestinya maka komputer harus memenuhi elemen-elemennya, yaitu:

Perangkat keras atau Hardware, yaitu elemen komputer yang fisiknya dapat dilihat dan diraba oleh indera manusia. Perangkat keras komputer dibedakan berdasarkan fungsinya, yaitu:Alat masukan atau Input, yaitu perangkat keras komputer yang berfungsi untuk memasukkan data atau informasi yang akan diproses oleh komputer dalam bentuk digital, contohnya keyboard, Mouse, Scanner, Floppy Disk, dan CD-ROM.Alat pemproses data, yaitu perangkat keras komputer yang berfungsi untuk mengolah data atau informasi yang dimasukkan dari alat input, contohnya CPU, Motherboard, Processor, VGA Card.Alat penyimpanan, yaitu perangkat keras komputer yang berfungsi sebagai tempat penyimpanan data pada komputer, contohnya Memori (RAM), Hard Disk, dan Flashdisk.Alat output atau pengeluaran, yaitu perangkat keras komputer yang berfungsi menampilkan data atau informasi yang telah diolah oleh komputer, contohnya Monitor, Printer, dan Speaker.Perangkat lunak atau Software, yaitu elemen komputer yang tidak berbentuk fisik sehingga tidak dapat dilihat dan diraba indera manusia. Perangkat lunak komputer berupa perintah atau instruksi yang bisa dimengerti oleh komputer. Perangkat lunak komputer dibedakan menjadi:Perangkat lunak sistem, yaitu perangkat lunak komputer yang berhubungan langsung dengan komponen perangkat keras komputer. Perangkat lunak sistem dibedakan menjadi Sistem Operasi contohnya Microsoft Windows, Linux, Unix dan lainnya, Bahasa Pemrograman, dan Program Utility.Perangkat lunak aplikasi yaitu perangkat lunak komputer yang berfungsi membantu pengguna komputer untuk menyelesaikan pekerjaannya dalam bidang tertentu, contohnya Microsoft Office, Windows Media Player, Flight Emulator, MYOB, Adobe Photoshop, Corel Draw, Microsoft Access, dan lainnya.Brainware yaitu user atau manusia yang menggunakan komputer untuk menjalankan software dan hardware komputer.

Pembahasan  Perangkat keras (hardware) adalah komponen fisik dari sebuah komputer seperti keyboard, CPU, mouse, RAM (Random Access Memory), Hard disk (HDD), dan lain-lain. Memory bukan termasuk perangkat keras. Namun, termasuk perangkat keras dengan nama khusus misalnya RAM. Perangkat keras komputer terbagi menjadi 4, yaitu perangkat input, perangkat output, perangkat processing, dan media penyimpan. Perangkat input berfungsi untuk memasukkan data dan program ke dalam komputer. Media penyimpan berfungsi untuk menyimpan data atau program komputer. Contoh: Hard disk, CD, DVD, USB Flash disk, dan lain-lain. Mouse berfungsi sebagai alat pointer untuk mengakses melalui layar monitor. Scanner untuk mengubah data hard copy menjadi soft copy yang dapat diolah di komputer. Joystick adalah tuas kendali yang dapat digerakkan ke segala arah sebagai pelengkap permainan video. Mouse dapat digunakan untuk mengoperasikan beberapa perintah seperti click, double klik, right click, dan drag. Drag dioperasikan dengan mengarahkan pointer mouse pada objek, kemudian menekan dan menahan tombol kiri mouse sambil digerakkan ke tempat yang diinginkan lalu dilepaskan. Keyboard berdasarkan fungsi dasarnya dibagi menjadi 4, yaitu Numeric key, function key, typewriter key dan special function key. Numeric key menampilkan angka, tanda baca titik dan operasi hitung. Perangkat output (output device) berfungsi untuk menyampaikan atau menampilkan informasi. Contoh: Monitor, speaker, dan printer.  ROM singkatan dari Read Only Memory merupakan salah satu memori yang ada dalam komputer yang bersifat permanen dan hanya bisa dibaca saja. Contoh: ROM BIOS (program dasar sistem komputer). RAM (Random Access Memory) dan Hard disk adalah media penyimpan data komputer yang dapat berubah tergantung penggunaan. Memory card adalah media penyimpan data digital pada gadget. Monitor merupakan perangkat output berupa gambar digital (softcopy). Printer merupakan perangkat output berupa gambar atau tulisan cetak (hardcopy). Speaker merupakan perangkat output berupa audio. Sedangkan mikrofon merupakan alat masukan (suara). Printer Inkjet mencetak gambar dan tulisan dengan memancarkan tinta pada kertas.
