f(X) = AX + B

Ditanya:

rumus baru untuk f(X) =

Jawab:

(1) substitusi 1 ke AX + B

f(X) = AX + B

f(1) = A + B

(substitusi f(1) jadi 0. Artinya ganti f(1) jadi 0)

0 = A + B (persamaan 1)

(2) substitusi 0 ke AX + B

f(X) = AX + B

f(X) = +B

(substitusi f(0) jadi -2. Artinya ganti f(0) jadi -2)

-2 = B (persamaan 2)

0 = A + B

Lalu ditanya rumus fungsi f(X). Artinya rumus funggsi f(X) yang baru setelah ditemukan A dan B nya. Maka

Fungsi linear memiliki variabel bebas paling tinggi berpangkat satu dan grafik fungsi linear berbentuk garis lurus. Persamaan grafik fungsi linear dari dua buah titik yang diketahui, yaitu (x1, y1) dan (x2, y2). Rumus fungsi linear dua titik yaitu:

[tex]\frac{Y - Y_{1}}{X - X_{1}}= \frac{Y_{2} - Y_{1}}{X_{2} - X_{1}}[/tex]

Penjelasan dengan langkah-langkah:

Diketahui:

Titik pada grafik (-3,0) dan (0,5)

Rumus grafik fungsi

Jawab:

Grafik fungsi dapat dihitung menggunakan rumus berikut ini:

[tex]\frac{Y - Y_{1}}{X - X_{1}}= \frac{Y_{2} - Y_{1}}{X_{2} - X_{1}}[/tex]

[tex]\frac{Y - 0}{X - (-3)}= \frac{5 - 0}{0 - (-3)}[/tex]

[tex]\frac{Y}{X + 3}= \frac{5}{3}[/tex]

3y = 5x + 15

3y - 5x -15 = 0

f(x) = x + 2

g(x) = x + 5

(f o g)(x) = f(g(x))

(f o g)(x) = f(x + 5)

(f o g)(x) = (x + 5) + 2

(f o g)(x) = x + 7

Jawaban:

[tex](fog)(x) = f(g(x)) \\ = f(x + 5) \\ = (x + 5) + 2 \\ = x + 7[/tex]

Jadi, (fog)(x) = x+7
