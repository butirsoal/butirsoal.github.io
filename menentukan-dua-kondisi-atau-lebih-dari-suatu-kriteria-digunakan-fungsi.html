4. jelaskan dua kriteria menentukan kecepatan akses internet


Terdapat dua macam kecepatan akses internet, yaitu downstream dan upstream. Downstream merupakan kecepatan pada saat kita mengambil data – data dari server internet ke komputer kita. Misalnya, saat kita masuk ke search engine, browsing, dan lain – lain. Adapun upstream adalah kecepatan transfer data yaitu saat kita mengirimkan data dari komputer ke server. Baik downstream maupun upstream memiliki satuan kecepatan transfer data yaitu bps bit per sekon. Artinya, banyaknya bit data yang dipindahkan dari satu komputer ke komputer lain tiap detiknya.

Kecepatan akses internet dihitung dari jumlah data yang dikirim dalam satuan waktu. Jika kita mengirim 1kb file/detik, berarti kita telah mengirim 1.000 byte, dengan 1 byte = 8 bit maka data yang dikirim sama dengan 8.000 bit = 8 kbps kilo bit per sekon. Untuk satuan yang lebih besar mengggunakan Mbps mega bit per sekon berarti 1000 kbps.

8. Jika kedua syarat menjadi sebuah kriteria mutlak yg harus di penuhi utk menghasilkan kondisi nilai True dapat dimasukan ke dalam kategori fungsi


Jawaban:

Jika kedua syarat menjadi sebuah kriteria mutlak yang harus di penuhi untuk menghasilkan kondisi nilai True dapat dimasukan ke dalam kategori fungsi AND. Dalam tabel kebenaran boolean maupun logika matematika, suatu nilai dalam operator AND akan bernilai true jika kedua kondisi bernilai true. Sebaliknya, kedua kondisi selain kondisi tersebut akan bernilai false.

Penjelasan:

Logika operator boolean maupun matematika memiliki beberapa jenis diantara OR, AND, NOT, IMP, Bimplikasi, dan XOR. Untuk OR, suatu kondisi dihasilkan False  jika kedua syarat mutlak bernilai false. Sedangkan pada XOR, akanmenghasilkan kondisi True bila memiliki dua kondisi yang berlawanan. Pada implikasi, hasil akan bergantung pada kondisi kedua kecuali pada situasi kedua syarat adalah false
