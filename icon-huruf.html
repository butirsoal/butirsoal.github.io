23. 1.sebutkan icon yg digunakan untuk mengganti jenis huruf... 2.gambar icon untuk mengubah ukuran huruf...3.gambar icon pengganda data... 4.gambar icon pencetak.... 5.gambar icon untuk memberikan warna pada huruf... 6.gambar icon untuk perataan text.. 7.pengertian dan kegunaan icon drop cap... 8.cara penggunaan tombol keyboard (ctrl, f, alt)... 9.cara penggunaan mouse drag&drop... 10.bagaimana cara menentukan ukuran kertas...


Ikon yang digunakan untuk mengganti jenis huruf adalah ikon font.Gambar ikon untuk mengubah ukuran huruf adalah ikon font size.Gambar ikon untuk pengganda data adalah ikon copy.Gambar ikon untuk pencetak adalah ikon print.Gambar ikon untuk memberikan warna pada huruf adalah ikon font color.Gambar ikon untuk perataan teks adalah ikon align right, align left, center, dan justify.Pengertian dan kegunaan dari ikon drop cap adalah ikon untuk membuat huruf kapital besar indah yang digunakan sebagai huruf awal dalam paragraf utama dengan sentuhan yang dekoratif sehingga huruf awalan tersebut akan turun satu atau lebih baris di bawahnya sesuai dengan keinginan pengguna.Contoh cara menggunakan tombol keyboard (ctrl, f, alt) adalah Ctrl+O untuk membuka dokumen, F1 untuk membuka panel tugas bantuan, dan Alt+N untuk menyisipkan gambar, tabel, bentuk, kotak teks, atau header.Cara menggunakan mouse drag & drop adalah dengan cara menahan klik pada mouse di file akan dieksekusi, kemudian seret file ke lokasi lain dan lepas tekanan pada mouse yang diklik sesuai dengan tempat yang diinginkan.Cara menentukan ukuran kertas adalah dengan klik menu File, kemudian pilih Page Layout. Setelah itu pilih Page Setup dan pilih ukuran kertas yang diinginkan.

Pembahasan

Ikon merupakan sebuah tombol yang digunakan sebagai perintah dalam suatu aplikasi yang berupa simbol atau gambar sehingga akan memudahkan dan mempercepat pekerjaan pengguna.

Berikut merupakan beberapa ikon dan fungsi-fungsinya.

Bold, tombol yang digunakan untuk menebalkan tulisan.Italic, tombol yang digunakan untuk memiringkan tulisan.Underline, tombol yang digunakan untuk membuat garis bawah pada tulisan.Font, tombol yang digunakan untuk memilih jenis huruf.Font Size, tombol yang digunakan untuk mengatur ukuran huruf.Grow Font, tombol yang digunakan untuk membesarkan ukuran huruf.Shrink Font, tombol yang digunakan untuk mengecilkan ukuran huruf.Font Color, tombol yang digunakan untuk memberi warna pada huruf.Highlight Color, tombol yang digunakan untuk memberi warna pada latar belakang huruf.Align Left, tombol yang digunakan untuk meratakan tulisan ke sisi kiri.Align Right, tombol yang digunakan untuk meratakan tulisan ke sisi kanan.Center, tombol yang digunakan untuk meratakan tulisan ke sisi tengahJustify, tombol yang digunakan untuk meratakan tulisan ke sisi kanan dan kiri.

Penggunaan keyboard juga tidak kalah penting. Beberapa pintasan keyboard yang dapat digunakan untuk memudahkan pekerjaan pengguna adalah sebagai berikut.

Ctrl + N, digunakan untuk membuat dokumen baru.Ctrl + S, digunakan untuk menyimpan dokumen.Ctrl + W, digunakan untuk menutup dokumen.Ctrl + X, digunakan untuk memotong konten yang dipilih.Ctrl + C, digunakan untuk menyalin konten yang dipilih.F1, digunakan untuk menampilkan panel bantuan.F2, digunakan untuk memindahkan konten yang dipilih. F3, digunakan untuk mengalihkan tulisan yang dipilih antara huruf besar, huruf kecil, dan huruf pada judul.F4, digunakan untuk mengulangi perintah atau tindakan terakhir.F5, digunakan untuk menampilkan kotak dialog Open.Alt + F, digunakan untuk membuka File yang akan menggunakan tampilan Backstage.Alt + H, digunakan untuk membuka tab Home.Alt + N, digunakan untuk untuk menyisipkan tabel, gambar, bentuk, header, atau kotak teks.Alt + G, digunakan untuk menggunakan tema, warna, dan efek, seperti batas halaman.Alt + P,  digunakan untuk memunculkan margin halaman, orientasi halaman, indentasi, dan penspasian.Pelajari lebih lanjutMateri tentang Microsoft Word secara singkat https://brainly.co.id/tugas/832101Materi tentang Microsoft Office secara singkat https://brainly.co.id/tugas/9287559Materi tentang fungsi dan manfaat Microsoft Word https://brainly.co.id/tugas/5075012

3. icon settings ialah icon yang berfungsi menampilkan jendela pensettingan tertentu saat didobel klik dengan mouse.
