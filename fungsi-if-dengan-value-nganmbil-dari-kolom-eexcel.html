10. Sebutkan contoh dari value IF True


Jawaban:Contoh-contoh penggunaan fungsi IF pada Excel:=IF(C6>=70,"Lulus","Gagal"), inilah contoh untuk menyatakan jika nilai dalam C6 lebih besar dari atau sama dengan 70, kembalikan "Lulus". Jika tidak, kembalikan "Gagal".=IF(AND(A1>7,A1<10),"OK",""), inilah contoh untuk menyatakan jika A1 lebih besar dari 7 dan kurang dari 10, kembalikan "OK". Kalau tidak, jangan kembalikan apa-apa ("").=IF(OR(A1="merah",A1="biru"),B1+10,B1), inilah contoh untuk menyatakan jika A1 merah atau biru, kembalikan B1 + 10, jika tidak kembalikan B1.=IF(E3>30,"Ya","Tidak"), inilah contoh untuk menyatakan Jika nilai dalam E3 lebih besar dari 30, kembalikan "Ya", jika tidak kembalikan "Tidak".

Pembahasan tentang soal iniCara pengunaan fungsi IF:

=IF (logical_test, [value_if_true], [value_if_false])

Argumen logical_test - Nilai atau ekspresi logis yang dapat dievaluasi sebagai TRUE atau FALSE. value_if_true - [opsional] Nilai untuk kembali ketika logical_test mengevaluasi ke TRUE. value_if_false - [opsional] Nilai untuk kembali ketika logical_test mengevaluasi ke FALSE.Fungsi Logis IF dan lainnyaIF - Test one condition; direct logical flow .IFS - Test multiple conditions; direct logical flow .NOT - Reverse criteria or results .AND - Test multiple conditions, return TRUE if all are TRUE .OR - Test multiple conditions, return TRUE if at least one is TRUE .XOR - Exclusive OR – return TRUE if one or the other, not both .IFERROR - Trap errors and return alternative results.

Fungsi gelap terang (value)
Unsur gelap terang (cahaya) pada karya senirupa
memberikan nilai ekspresi, misalnya untuk
menampilkan kesan dramatis pada lukisan, seperti
pada tema peperangan dengan ungkapan gelap terang.

memberikan nilai emosi, misalnya cahaya yang
membus jendela kaca patri yang menimbulkan suasana
khidmat pada interior mesjid atau gereja.
Unsur gelap terang (cahaya) pada karya senirupa
memberikan kesan trimatra atau plastis pada benda
yang diterpa oleh cahaya seperti pada bangunan dan
benda. Dalam hal ini gelap terang (cahaya) dapat
memperkuat sifat benda trimatra.

30. sebutkan fungsi value dalam seni rupa​


Jawaban:

Pada pembuatan desain grafis, value memiliki beberapa fungsi, antara lain :

Untuk mengubah cahaya yang mengenai obyek/benda ke dalam bentuk tiga dimensi semu, misalnya untuk membentuk gelap terang pada gambar bentuk. Value terang (tint) untuk bagian yang kena cahaya langsung, value sedang (tone) untuk bagian yang kena cahaya normal, value gelap (shade) untuk bagian yang tidak kena cahaya atau bagian bayangan

Untuk menciptakan atau menilai karya seni, fungsi ini khususnya berkaitan dengan karakter value yang antara lain adalah sebagai berikut :

-Value terang mempunyai karakter positif, bergairah, meriah, ringan

-Value normal mempunyai karakter tegas, jujur, terbuka, galak

-Value gelap mempunyai karakter berat, dalam, muram, mengerikan, menakutkan

-Untuk mengubah cahaya yang mengenai obyek berwarna-warni menjadi bentuk hitam putih

Penjelasan:

Value adalah dimensi mengenai derajat terang gelap atau tua muda warna. Value merupakan perangkat untuk mengukur derajat terang atau gelapnya suatu warna jika dibandingkan dengan skala value.
