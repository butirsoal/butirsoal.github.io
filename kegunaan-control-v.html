18. Sebutkan kegunaan kode Control pada aplikasi scratch​


Jawaban:

Berikut adalah beberapa kegunaan kode Control pada aplikasi Scratch:

1. Kode Control "Forever" : kode ini digunakan untuk mengulang perintah secara berkala.

2. Kode Control "When green flag clicked" : kode ini digunakan untuk mengeksekusi perintah saat flag hijau ditastikan.

3. Kode Control "When space key pressed" : kode ini digunakan untuk mengeksekusi perintah saat tombol spasi diketuk.

4. Kode Control "Wait" : kode ini digunakan untuk menunda eksekusi perintah selama waktu tertentu.

5. Kode Control "Broadcast" : kode ini digunakan untuk mengirimkan pesan ke banyak kumpulan, seperti "Scratch Cat" atau "Scratch Dog".

6. Kode Control "Switch" : kode ini digunakan untuk mengontrol aliran perintah menuju satu atau beberapa perintah dengan menggunakan operator "if" atau "or" berdasarkan kondisi tertentu.

7. Kode Control "If" : kode ini digunakan untuk menentukan jika kondisi keadaan tertentu harus diambil tindakan tertentu.

8. Kode Control "LOOP" : kode ini digunakan untuk mengulang perintah secara berkala dengan kecepatan tertentu.

Contohnya, kode Control "Forever" dapat digunakan untuk menciptakan animasi berputar, sedangkan kode Control "Wait" dapat digunakan untuk menunda eksekusi perintah selama waktu tertentu.

Save,merupakan perintah untuk menyimpan file lembar kerja Excel 2007 yang baru dibuat atau diedit, yang sebelumnya pernah atau belum pernah tersimpan di dalam hardisk, CD/DVDROM, USB Flashdisk, disket, atau media simpan lainnya.
