16. Rumus jarak sebenarnya (JS) pada peta, jika diketahui skala (S) dan jarak pada peta (JP) adalah....a. JS = JP × Sb. JS =S × JPc. JS = JP : Sd. JS = S : JP​


Jawaban:

~Skala

Rumus jarak sebenarnya (JS) pada peta, jika diketahui skala (S) dan jarak pada peta (JP) adalah

a. JS = JP × S

b. JS = S × JP

c. JS = JP : S ✔

d. JS = S : JP

Karena sesuai rumusnya yaitu :

Jp = S × Js

S = Jp/Js

Js = Jp/S

Ingat yaa per (/) itu adalah Bagi jadi kita boleh menggunakan bagi ataupun per.

Jarak pada peta (JP)
Skala (S)

Jarak sebenarnya (JS)
= jarak pada peta : skala
= JP : S

Jadi, JS = JP : S (C)

24. 1. Apabila diketpeta (JP) maka jarak sebenarnya (JS) adalah ....A. JS = Skala ÷ JPB. JS = JP x SkalaC. JS = JP ÷ SD. JS = (JP x Skala) ÷ 1Alasan : ....​


Diketahui:

Jarak pada peta (JP)

Ditanya:

Jarak Sebenarnya (JS) adalah...

Penyelesaian:

Jika dalam peta diketahui jarak pada peta, maka untuk mencari jarak sebenarnya digunakan rumus C. JS = JP ÷SK

Dalam soal yang melibatkan sebuah peta, maka akan digunakan rumus:

JS = JP ÷ SK ( digunakan untuk mencari jarak sebenarnya)

SK = JP ÷ JS (digunakan untuk mencari skala pada peta)

JP = JS × SK (digunakan untuk mencari jarak pada peta)
