12. tentukan domain dan range dari fungsi rasional f(x) = x2 + x -6 / x2 -x -2tolong bantu jawab pertanyaan saya pliss​


Domain dari fungsi rasional f(x) = [tex]\frac{x^{2}+x-6 }{x^{2} -x-2}[/tex] adalah

Df = {x : x > -1, x ∈ R}

Range dari fungsi rasional f(x) = [tex]\frac{x^{2}+x-6 }{x^{2} -x-2}[/tex] adalah

Rf = {y : y [tex]\neq[/tex] 0, y ∈ R}

Penjelasan dengan langkah-langkah:

Diketahui :

f(x) = [tex]\frac{x^{2}+x-6 }{x^{2} -x-2}[/tex]

Ditanya :

Domain dan range dari f(x)?

Jawab :

Tentukan faktor dari pembilang dan penyebut terlebih dahulu

Faktor x²+x-6

b = 1 = 3 + (-2)

c= -6 = 3 × (-2)

maka faktor dari x²+x-6 = (x+3)(x-2)

Faktor x²-x-2

b = -1 = 1 + (-2)

c= -2 = 1 × (-2)

maka faktor dari x²+x-6 = (x+1)(x-2)

Tentukan Domain

f(x)= [tex]\frac{x^{2}+x-6 }{x^{2} -x-2}[/tex]

f(x)= [tex]\frac{(x+3)(x-2)}{(x+1)(x-2)}[/tex]

f(x) = [tex]\frac{x+3}{x+1}[/tex]

Penyebut harus lebih besar dari 0 karena jika nilai penyebut adalah 0 maka menjadi bilangan irrasional

x+1 > 0 (kedua ruas dikurangi 1)

x > -1

Df = {x : x > -1, x ∈ R}

Tentukan Range

Nilai y atau f(x) harus bernilai positif atau negatif karena jika nilai y adalah 0 maka tidak ada nilai x yang memenuhi

f(x) = [tex]\frac{x+3}{x+1}[/tex]

0 [tex]\neq[/tex] [tex]\frac{x+3}{x+1}[/tex]

Rf = {y : y [tex]\neq[/tex] 0, y ∈ R}

1.) y = x² - 5x, -2 ≤ x ≤ 2

Artinya akan digambar grafik fungsi y = x² - 5x dengan nilai x = {-2, -1, 0, 1, 2}

x = -2 → y = (-2)² - 5(-2) = 4 + 10 = 14 (titik (-2, 14))

x = -1 → y = (-1)² - 5(-1) = 1 + 5 = 6 (titik (-1, 6))

x = 0 → y = 0² - 5(0) = 0 (titik (0, 0))

x = 1 → y = 1² - 5(1) = 1 - 5 = -4 (titik (1, -4))

x = 2 → y = 2² - 5(2) = 4 - 10 = -6 (titik (2, -6))

38. Tentukan domain dan range dari fungi : f(x)=3x-1/x2.


Jawab:

Domain dari [tex]f(x)=\frac{3x-1}{x+2}[/tex] adalah {[tex]x|x\neq{-2},x\in{R}[/tex]}.

range dari [tex]f(x)=\frac{3x-1}{x+2}[/tex] adalah {[tex]y|y\neq{3},y\in{R}[/tex]}.

Penjelasan dengan langkah-langkah:

Diketahui:

[tex]f(x)=\frac{3x-1}{x+2}[/tex]

Ditanya:

domain = . . .

range = . . .

Jawab:

▪︎Mencari domain

[tex]f(x)=\frac{3x-1}{x+2}[/tex]

f(x) merupakan fungsi rasional dengan pembilang dan penyebut berupa polinom dan penyebut [tex]\neq{0}[/tex].

[tex]x+2\neq{0}[/tex]

[tex]x\neq{-2}[/tex]

Jadi, domain dari [tex]f(x)=\frac{3x-1}{x+2}[/tex] adalah {[tex]x|x\neq{-2},x\in{R}[/tex]}.

▪︎Mencari range

[tex]f(x)=\frac{3x-1}{x+2}[/tex]

[tex]y=\frac{3x-1}{x+2}[/tex]

[tex]y(x+2)=3x-1[/tex]

[tex]xy+2y=3x-1[/tex]

[tex]xy-3x=-2y-1[/tex]

[tex]x(y-3)=-2y-1[/tex]

[tex]x=\frac{-2y-1}{y-3}[/tex]

Syarat:

[tex]y-3\neq{0}[/tex]

[tex]y\neq{3}[/tex]

Jadi, range dari [tex]f(x)=\frac{3x-1}{x+2}[/tex] adalah {[tex]y|y\neq{3},y\in{R}[/tex]}.

          = 0
