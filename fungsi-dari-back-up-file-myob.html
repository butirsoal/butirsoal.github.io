12. mengapa file data program myob di simpan di disk c?​


Jawaban:

File data program MYOB di simpan di disk C karena secara default, Windows memberikan nama C ke sebuah drive. Drive C ini biasanya digunakan untuk menyimpan file sistem operasi dan aplikasi pendukung.

Pada awal instalasi, MYOB akan membuat folder baru di dalam drive C dengan nama "MYOB Data". Folder inilah yang akan digunakan untuk menyimpan file data MYOB.

Ada beberapa alasan mengapa file data MYOB disimpan di drive C:

* **Untuk memudahkan akses.** File data MYOB adalah file yang sering digunakan, sehingga dianjurkan untuk menyimpannya di drive yang mudah diakses, yaitu drive C.

* **Untuk menghindari kehilangan data.** Jika terjadi kerusakan pada hard disk, file data MYOB yang disimpan di drive C akan lebih mudah untuk dipulihkan.

* **Untuk memenuhi persyaratan sistem.** MYOB memiliki persyaratan sistem yang mengharuskan file data disimpan di drive C.

Namun, ada beberapa hal yang perlu diperhatikan jika menyimpan file data MYOB di drive C:

* **Disk C biasanya berukuran kecil.** Jika file data MYOB berukuran besar, maka dapat menyebabkan kapasitas disk C menjadi penuh.

* **Jika terjadi kerusakan pada hard disk, maka file data MYOB akan ikut rusak.** Untuk menghindari hal ini, disarankan untuk melakukan backup file data MYOB secara berkala.

Jika Anda ingin menyimpan file data MYOB di lokasi lain, Anda dapat mengubah lokasi tersebut melalui pengaturan MYOB. Namun, Anda perlu memastikan bahwa lokasi tersebut memenuhi persyaratan sistem MYOB.

pengertian back up

Kegunaan atau manfaat back up data yaitu kita masih mempunyai cadangan data dari data yang hilang/rusak/terhapus, baik yang disebabkan oleh kesalahan kita sendiri atau faktor lain di luar kemampuan kita, seperti: terkena virus, file rusak (tidak bisa dibuka), perangkat komputer error/bermasalah, mati listrik, bencana, dan lain sebagainya.

Gerakan push up bermanfaat untuk menguatkan otot lengan dan otot perut.  Gerakn sit up bermanfaat untuk menguatkan otot perut dan otot punggung.  Sedangkan gerakan back up bermanfaat untuk menguatkan otot dada, otot perut, dan otot punggung.

Pembahasan

Untuk menjaga kebugaran jasmani atau kebugaran tubuh, terdapat bebarapa gerakan sederhana yang dapat dilakukan. Seperti gerakan push up, sit up, dan back up. Ketiga gerakan tersebut merupakan gerakan sederhana yang dapat meningkatkan kebugaran jasmani seseorang dan dapat menguatkan beberapa anggota tubuh sesuai dengan gerakannya. Berikut penjelasannya.

Push up

Push up adalah gerakan mengangkat tubuh dengan menggunakan tumpuan dan dorongan kedua tangan. Gerakan push up bermanfaat untuk menguatkan otot lengan dan otot perut.

Sit up

Sit up adalah gerakan mengangkat tubuh bagian atas dengan posisi badan terlentang. Gerakn sit up bermanfaat untuk menguatkan otot perut dan otot punggung.

Back up

Back up adalah gerakan mengangkat tubuh bagian atas dengan posisi badan tengkurap. Gerakan back up bermanfaat untuk menguatkan otot dada, otot perut, dan otot punggung.

40. sebutkan salah satu tool yang dapat di gunakan untuk mem back up file?​


Jawaban:

Rsync

Rsync adalah salah satu backup tools yang paling banyak digunakan. Dengan Rsync, kita dapat melakukan proses tahapan backup secara flexible, baik secara lokal maupun remote. Rsync dapat memperbaharui seluruh direktori dan file systems, preserve links, ownerships, permissions, dan privileges dengan menggunakan rsh, ssh ataupun koneksi langsung ke socket, serta mendukung koneksi anonymous. Rsync sendiri adalah program command-line, salah satu nilai plus menggunakan program command-line yaitu kita dapat membuat sebuah script untuk digunakan, terkait dengan cron (penjadwalan tugas) untuk membuat proses backup secara otomatis.
