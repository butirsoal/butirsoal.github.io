[tex]\textbf{Peluang = banyak kejadian yang di maksud / ruang sampel}[/tex]

Jikalau suatu percobaan hanya menghasilkan dua kemungkinan kejadian yang kita asumsikan sebagai "sukses" dan "gagal" , maka percobaan ini bisa menghasilkan distribusi probabilitas Binomial. Distribusi Probabilitas Binomial :

P(X=a) = ⁿCₐ pᵃ ( 1-p ) ⁿ⁻ᵃ

dengan :

p = kemungkinan suatu kejadian sukses

n = banyaknya percobaan / pengambilan

Distribusi Probabilitas Poisson :

[tex]P(X=a)=e^{-\lambda}~\frac{\lambda ^a}{a!}[/tex]

dengan :

λ = parameter kejadian sukses

15. Misal X dan Y adalah dua peubah acak yang saling bebas dan masing-masing memiliki FKP berbentuk: f(x) = 2.e-2x:x>0 = 0, x lainnya g(y) = 3.e-³;y > 0 = 0, x lainnya Jika U = X + Y dan V = X tentukan fungsi densitas gabungan dari U dan V!


Jawaban:

Untuk menentukan fungsi densitas gabungan dari U dan V, kita perlu memperhatikan distribusi masing-masing variabel acak dan hubungan antara U dan V.

Dari informasi yang diberikan, kita memiliki fungsi densitas probabilitas (FKP) untuk X dan Y:

f(x) = 2e^(-2x), untuk x > 0

0, untuk x <= 0

g(y) = 3e^(-3y), untuk y > 0

0, untuk y <= 0

Kita juga memiliki hubungan U = X + Y dan V = X.

Untuk mencari fungsi densitas gabungan fU,V(u,v), kita perlu menentukan rentang di mana fU,V(u,v) bukan nol. Karena X dan Y adalah saling bebas, nilai-nilai positif dari X dan Y akan memberikan kontribusi pada nilai positif U dan V. Oleh karena itu, kita dapat membatasi rentang dengan u > 0 dan v > 0.

Untuk u > 0 dan v > 0, kita dapat menggunakan transformasi variabel. Kita ingin mencari fungsi densitas gabungan fU,V(u,v) dengan mempertimbangkan X = v dan Y = u - v.

Menggunakan transformasi variabel, kita dapat menghitung determinan dari matriks Jacobian:

|dx/du dx/dv|

|dy/du dy/dv|

dx/du = 0 (karena X tidak tergantung pada U)

dx/dv = 1 (karena X = V)

dy/du = 1 (karena Y = U - V)

dy/dv = -1 (karena Y tidak tergantung pada V)

Determinan dari matriks Jacobian adalah -1.

Kita dapat menggunakan transformasi variabel untuk mendapatkan fungsi densitas gabungan fU,V(u,v) berdasarkan fungsi densitas f(x) dan g(y):

fU,V(u,v) = f(x = v) * g(y = u - v) * |-1|

= f(x = v) * g(y = u - v)

= f(v) * g(u - v)

Substitusikan f(v) dan g(u - v) dengan fungsinya:

fU,V(u,v) = (2e^(-2v)) * (3e^(-3(u-v)))

Sehingga, fungsi densitas gabungan fU,V(u,v) adalah:

fU,V(u,v) = 6e^(-2v-3(u-v))

untuk u > 0 dan v > 0.

Dalam kata lain, fungsi densitas gabungan fU,V(u,v) adalah:

fU,V(u,v) = 6e^(-2v-3u+3v)

= 6e^(-3u+v)

untuk u > 0 dan v > 0.

Jawaban:

Untuk menentukan distribusi peluang dan fungsi distribusi dari peubah acak X, yang menunjukkan banyak huruf "BANK INDONESIA" yang muncul saat pelemparan dua buah mata uang Rp.500 yang seimbang, kita perlu mempertimbangkan semua kemungkinan hasil pelemparan.

Ada empat kemungkinan hasil pelemparan mata uang:

1. Kedua mata uang menunjukkan huruf "B" (BB).

2. Kedua mata uang menunjukkan huruf "A" (AA).

3. Kedua mata uang menunjukkan huruf "N" (NN).

4. Kedua mata uang menunjukkan huruf selain "BANK INDONESIA" (SN).

Distribusi peluang adalah probabilitas terjadinya setiap hasil pelemparan. Dalam hal ini, karena setiap mata uang memiliki peluang 1/2 untuk menunjukkan huruf tertentu, maka peluang untuk munculnya huruf "BANK INDONESIA" adalah:

P(X = 0) = P(SN) = 1/2 * 1/2 = 1/4

P(X = 1) = P(BB, AA, atau NN) = 3 * 1/2 * 1/2 = 3/4

P(X = 2) = P(BB) = 1/2 * 1/2 = 1/4

Fungsi distribusi (cumulative distribution function, CDF) adalah jumlah probabilitas terjadi kurang dari atau sama dengan suatu nilai. Dalam hal ini, fungsi distribusi dapat dihitung sebagai berikut:

F(X ≤ 0) = P(X = 0) = 1/4

F(X ≤ 1) = P(X = 0) + P(X = 1) = 1/4 + 3/4 = 1

F(X ≤ 2) = P(X = 0) + P(X = 1) + P(X = 2) = 1/4 + 3/4 + 1/4 = 1

Jadi, distribusi peluang untuk peubah acak X adalah: P(X = 0) = 1/4, P(X = 1) = 3/4, dan P(X = 2) = 1/4.

Sementara itu, fungsi distribusi untuk X adalah: F(X ≤ 0) = 1/4, F(X ≤ 1) = 1, dan F(X ≤ 2) = 1.

26. Dalam sebuah kotak terdapat 6 kapasitor, 4 diantaranya bernilai 0,1µF dan sisanya bernilai 1 µF. Dari kotak tersebut diambil 3 kapasitor secara acak. Jika peubah acak X menyatakan banyaknya kapasitor yang bernilai 1 µF yang terambil , maka tentukan : 1.Daerah hasilnya. 2.Fungsi peluang bagi X 3. Peluang distribusi


Dengan 6 kapasitor yang terdapat 4 kapasitor 0,1 μF dan 2 kapasitor 1 μF, dan akan diambil 3 kapasitor secara acak.

Dengan X sebagai banyaknya kapasitor 1 μF yang terambil.
Untuk p adalah peluang sukses dan q adalah peluang gagal:
p = 4/6 = 2/3
q = 2/6 = 1/3

Dengan n = 6, maka nilai a sebagai banyaknya yang terambil haruslah dalam interval tertutup 0 ≤ a ≤ 3, untuk a bilangan bulat nonnegatif karena merupakan kejadian diskret.

Fungsi peluang untuk X diberikan dalam:
P(X = a) = nCa . pᵃ. qⁿ⁻ᵃ

Sehingga, peluang distribusi yang dihasilkan:
Untuk a = 0
F(X = 0) = ₆C₀ x (2/3)⁰ x (1/3)⁶ = 1/729
Untuk a = 1
F(X = 0) = ₆C₁ x (2/3)¹ x (1/3)⁵ = 12/729 = 4/243
Untuk a = 2
F(X = 0) = ₆C₂ x (2/3)² x (1/3)⁴ = 60/729 = 20/243
Untuk a = 3
F(X = 0) = ₆C₃ x (2/3)³ x (1/3)³ = 160/729

Didapat daerah hasil = {1/729, 4/243, 20/243, 160/729}
