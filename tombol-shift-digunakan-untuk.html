Penjelasan:

Fungsi tombol shift pada tombol keyboard antara lain : Membuat huruf kapital dan menjadikan lowercase pada kondisi capslock on. Sebagai shortcut membuka Recycle Bin. Sebagai cara menghilangkan tampilan loading ketika membuka Microsoft Office.

Jawaban:

Apa Fungsi Shift – Jika kalian sering menggunakan computer atau laptop, tentu kalian sudah terbiasa dengan beragam tombol beserta fungsinya yang ada di keyboard komputer atau laptop kalian. Meskipun begitu, tidak sedikit pula yang belum mengetahui apa saja fungsi dari tombol keyboard yang tersedia. Masing-masing tombol pada keyboard memiliki kegunaan yang berbeda dan pastinya saling berkaitan satu sama lain. Salah satu tombol yang mempunyai banyak fungsi dan pastinya juga banyak dipakai saat mengoperasikan sebuah komputer atau laptop, yakni tombol shift.

Mendengar kata “Shift” pasti kalian tidak akan merasa asing, namun bisa jadi kalian belum tahu apa saja fungsi dari keberadaan tombol tersebut. Tombol shift merupakan satu dari sekian banyak tombol yang keberadaannya cukup penting di keyboard. Tombol ini biasanya sering digunakan untuk membesarkan huruf ataupun untuk pelengkap tombol shortcut. Selain itu, tombol shift ini juga sering digunakan secara berdampingan dengan tombol yang lainnya seperti: Alt, Ctrl, Enter, Tab, Delete, dan tombol lainnya. Adapun contoh umumnya yakni kalian bisa menghapus sebuah file atau dokumen secara permanen, maka kalian pasti akan menekan tombol “Shift + Del”.

 

Apa Fungsi Shift

 

Nah, tak hanya satu pasangan tombol shift yang bisa dipakai untuk mengoperasikan komputer, namun ada banyak. Untuk pemanfaatan tombol shift ini memang tidak bisa mandiri untuk itu selalu ada tombol lain yang perlu dipasangkan dengan shift. Hal ini akan membuat tombol shift memiliki banyak fungsi apabila tombol shift tersebut dikombinasikan bersama tombol lainnya. Untuk itu penting sekali mengetahui apa fungsi shift yang lainnya. Saking pentingnya terdapat dua tombol “SHIFT” pada sebuah keyboard PC. Maka dari itu, jika tombol SHIFT pada keyboard rusak saat kalian sedang mengoperasikan PC, maka kalian pasti akan mengalami beberapa gangguan sehingga aktivitas saat beroperasi dengan komputer tidak bisa lancar.

Apa Fungsi Shift

 

Fungsi Tombol “SHIFT” pada keyboard PC komputer atau Laptop

Karena tombol “shift” ini bersifat universal, maka tombol tersebut juga dapat dipakai pada sejumlah software. Berikut ini fungsi dari tombol “shift” yang bisa kalian ketahui, yakni:

Untuk Menghapus File atau Folder

Adapun fungsi tombol shift bisa dipakai untuk mengahpus file atau folder jika tombol tersebut dikombinasikan dengan tombol “Del”. Gabungan tombol tersebut akan menghapus sebuah file atau folder dan berkas yang terhapus tersebut tidak akan masuk ke Recycle Bin sehingga file dihapus tidak bisa dipulihkan karena penghapusan yang dilakukan bersifat permanen. Caranya kalian bisa menekan secara bersama Tombol SHIFT + DEL.

Mempercepat komputer booting

Jika kalian ingin tahu apa fungsi shift yang lainnya yakni digunakan untuk mempercepat komputer booting. Apabila kalian mengalami masalah computer lelet ketika booting, maka silahkan kalian tekan tombol “SHIFT” dan kalian akan merasakan tambahan kecepatan booting di komputer kalian.

Cegah autorun virus

Untuk fungsi tombol shift selanjutnya ialah mencegah autorun virus pada komputer. Apabila kalian sering kemasukan virus dari flashdisk yang kalian tancapkan di komputer, maka pastikan sebelum menancapkan flashdisk tersebut, kalian tekan serta tahan tombol “SHIFT” untuk mencegah autorun virus berjalan. Hal ini bisa membuat virus tidak akan mudah masuk ke komputer ataupun laptop.

Mengubah File Jadi Tanpa Nama

Fungsi shift selanjutnya yakni untuk mengubah file agar tidak punya nama. Kalian bisa melakukan ini dengan cara menekan serta menahan tombol “SHIFT” saat hendak membuka file dengan melalui Windows Explorer. Kalian bisa melakukan hal tersebut jika kalian benar-benar membutuhkannya.

Memunculkan Menu stickykeys

Fungsi selanjutnya yakni untuk memunculkan menu stickykeys. Kalian mungkin sudah tahu tentang apa fungsi shift yang satu ini. Dalam hal ini kalian bisa tekan berulang ulang tombol “SHIFT”. Rata-rata penekanan dilakukan hingga 4 kali. Kalian bisa melakukannya sampai terdengar bunyi tiiiiit di CPU. Fasilitas stickykeys ini digunakan untuk mempermudah proses pengkombinasian beberapa tombol. Dengan begitu, kalian bisa menjalankan perintah-perintah yang lain tanpa harus menekan tombol yang sebelumnya.

Meniadakan Splash Screen/Loading Screen

Fungsi ke enam yaitu tombol shift bisa dipakai untuk menghilangkan splash screen/ loading screen. Keduanya bisa ditemui saat ingin mengaktifkan microsof Word. Jika kalian mau menghilangkan tampilan tersebut, kalian bisa menekan serta menahan tombol “SHIFT” sebelum kalian menjalankan aplikasi Microsoft Word maupun aplikasi apapun lainnya. Setelah itu, biasanya splashscreen akan menghilang.

6. guna membuat range dengan menggunakan tombol shift dengan cara menempatkan​


Jawaban:

Untuk membuat range dengan menggunakan tombol shift, Anda dapat menggunakan shortcut keyboard pada aplikasi yang Anda gunakan. Pada aplikasi Microsoft Excel, Anda dapat menempatkan cursor pada sel yang ingin Anda jadikan awal range, kemudian tekan tombol Shift dan klik pada sel yang ingin Anda jadikan akhir range. Seluruh sel di antara sel awal dan akhir akan terpilih dan dapat digunakan sebagai range.

Pada aplikasi Microsoft Word, Anda dapat menempatkan cursor pada kata atau paragraf yang ingin Anda jadikan awal range, kemudian tekan tombol Shift dan klik pada kata atau paragraf yang ingin Anda jadikan akhir range. Seluruh teks di antara kata atau paragraf awal dan akhir akan terpilih dan dapat digunakan sebagai range.

Note: cara membuat range dengan menggunakan tombol shift juga tergantung dari aplikasi yang digunakan, mungkin berbeda pada aplikasi lain.

8. Apa fungsi dari tombol Shift +panah kanan dan shift+home? ​


Fungsi Shortcut Shift + Panah kanan:

Fungsinya adalah untuk select setiap huruf yang sudah diketik. Saat menekan tombol Shift + Panah Kanan secara bersamaan, tulisan yang kita ketik akan ter-select setiap hurufnya / satu-satu huruf.

Fungsi Shortcut Shift + Home:

Fungsinya adalah untuk select tulisan sampai ke awal tulisan / teks yang sudah diketik. Begitu pula sebaliknya, saat menekan tombol Shift + End, tulisan yang akan ter-select adalah sampai bagian akhir suatu teks.

Bila ada kekurangan atau bahkan kesalahan, mohon maaf.
