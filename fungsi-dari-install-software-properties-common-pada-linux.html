32. Apakah yang akan terjadi kalau pengguna sering install dan uninstall software


Tidak akan terjadi masalah apapun jika user terus-menerus reinstall suatu software, tapi ada kemungkinan akan ada error. Namun, kemungkinan itu sangat kecil. Biasanya software akan membuat file cache.

File cache ini digunakan untuk menyimpan data yang nantinya akan digunakan untuk memulai kembali aplikasi. Jadi, aplikasi tidak perlu memuat data yang banyak karena sudah tersimpan pada cache.

Bisa saja terjadi error saat memulai kembali aplikasi yang terus-menerus di-reinstall. Misalnya, file cache tersebut menumpuk dan aplikasi terkena freeze atau force close. Hal yang dapat mencegahnya adalah selalu membersihkan file cache sebelum meng-uninstall suatu software.

2. Bila ada konfirmasi y / n, ketik saja y



3. Untuk meremote server via localhost, ketikkan perintah: ssh root @ localhost , kemudian ketikkan yes


4. Kemudian jika Anda ingin mengubah port pada ssh server Anda, ketikkan perintah:
cd / etc / ssh - ls - pico sshd_config atau pico / etc / ssh / ssd_config


5. Lalu ubah port pada baris seperti gambar dibawah:


6. Ubah Port 22 menjadi port yang ingin Anda hendaki, misalnya: 850



7. Setelah selesai, simpan file tersebut dengan CTRL-X, Y
8. Restart konfigurasi ssh Anda: /etc/init.d/ssh restart

34. Common network services pada server telah ditentukan diantaranya: Web Server, Database Server, DNS Server, dan DHCP Server. Jika diimplementasikan pada SO Linux/Debian maka paket apa saja yang cocok diinstall untuk layanan diatas. Jelaskan!


Penjelasan:

Untuk mengimplementasikan layanan Web Server pada sistem operasi Linux/Debian, Anda dapat menginstall paket Apache. Apache adalah salah satu web server yang paling populer dan banyak digunakan di seluruh dunia.

Untuk mengimplementasikan layanan Database Server, Anda dapat menginstall paket MySQL atau MariaDB. Kedua paket ini merupakan sistem manajemen database yang populer dan banyak digunakan pada sistem operasi Linux/Debian.

Untuk mengimplementasikan layanan DNS Server, Anda dapat menginstall paket BIND. BIND adalah salah satu DNS Server yang paling populer dan banyak digunakan pada sistem operasi Linux/Debian.

Untuk mengimplementasikan layanan DHCP Server, Anda dapat menginstall paket ISC DHCP Server. ISC DHCP Server adalah salah satu DHCP Server yang paling populer dan banyak digunakan pada sistem operasi Linux/Debian.
