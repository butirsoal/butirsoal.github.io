7. tombol scroll page pada browser namanya apa


Pada browser, tombol scroll page adalah Page up atau page down yang berada pada sebelah kiri halaman website yang dipakai naik sedangkan page down untuk turun.

Pembahasan

Browser merupakan suatu jenis perangkat lunak atau software yang pada umumnya digunakan bahkan dimanfaatkan untuk membuka suatu halaman di situs internet. Dengan demikian, browser disebut juga sebagai penerima, pengakses bahkan penyaji dalam berbagai informasi diinternet. Ada beberapa fungsi web browser, antara lain:

Untuk mempermudah dalam mencari informasi di internet. Untuk menyimpan data-data di internet. Untuk membuka file dengan ekstensi tertentu. Untuk mendukung pemakai search engine. Untuk menjaga keamanan website.

Ada beberapa macam web browser, antara lain:

Google Chrome, yang hampir 60% pengguna internet di dunia menggunakannya khususnya negara-negara di benua Eropa, Asia bahkan Amerika. Mozilla Firefox, hal ini jangan diragukan lagi, misalnya browser memiiki tingkat pengamanan  super ampuh dibanding yang lain. Opera mini, merupakan salah satu dari macam-macam web browser yang sedang populer dan memiiki setidaknya 6% dari pengguna internet di seluruh dunia untuk perangkat komputer. Internet Explorer, yang meski pengguna internet explorer semakin sedikit, akan tetapi browser ini masih ada yang menngunakannya. Pada tahun 2002-2003, internet explorer pernah digunakan sampai 95 % dalam pengguna internet.

Page up berfungsi untuk menaikkan bahkan berpindah halaman ke arah atas. Dengan demikian, tombol hanya menggerakkan halaman maupun seluruh tampilan layar yang hanya bisa digunakan untuk menaikkan. Sedangkan page down berfungsi untuk berpindah bahkan menggeser halaman ke arah bawah. Sama halnya dengan page up yang hanya page down ke arah bawah sedangkan pageup ke arah atas.

11. sebutkan fungsi - fungsi dari tombol keyboard berikut tombol caps lock, tombol tab, tombol back pace, tombol delete, tombol entar, tombol esc, tombol home, tombol end, tombol Page up, tombol Page Down​


Jawaban:

Misalkan dipakai pada program pengolah data teks (seperti Ms. Word):

Caps Lock: untuk mengunci huruf kapital.

Tab: untuk memindah posisi kursor ke sebelah kanan 1 tabulasi.

Back space: untuk menghapus 1 karakter di sebelah kiri

Delete : untuk menghapus 1 karakter di sebelah kanan

Enter: untuk mengakhiri 1 baris dan mengawali pengetikan teks di baris baru (di bawahnya).

Esc: untuk menghilangkan tampilan menu yg muncul.

Home: untuk memindah posisi kursor ke awal baris teks.

End: untuk memindah posisi kursor ke akhir baris.

Page Up: untuk memindahkan posisi kursor 1 layar ke atas.

Page Down: untuk memindahkan posisi kursor 1 layar ke bawah.

Penjelasan:

Contoh pemakaian tombol Esc: misalkan mengklik kanan teks (dg mouse) lalu muncul suatu menu, maka tombol ini bisa dipakai untuk menghilangkan menu yg muncul tadi.

. Fungsi Tombol Shift

Fungsinya yaitu untuk membuat huruf besar secara manual (dengan menekan shift+a = A), dan untuk memasukkan input yang diatas angka 1,2,3 dan seterusnya (contoh : shift+1 = ! , shift+/ = ?).


8. Fungsi Tombol Back Space 

Fungsinya untuk mendelete satu huruf yang berada disebelah kiri kursor. Jika kita kombinasikan dengan tombol ctrl, ctrl+backspace maka anda menghapus satu kata disebelah kiri kursor.
