15. gambarlah grafik fungsi kuadrat yang ditentukan oleh fungsi fungsi berikut f(x)= x kuadrat -2x-15 kemudian tentukan pembuat nol fungsi (jika ada),persamaan sumbu simetri ,dan titik puncak maksimum/minimum


Grafik Fungsi Kuadrat dan Penentuan Pembuat Nol, Sumbu Simetri, dan Titik Puncak

Dalam kasus ini, kita diberikan fungsi kuadrat f(x) = x² - 2x - 15. Untuk menggambar grafik fungsi ini, kita dapat melihat koefisien dari x², x, dan konstanta.

Dalam fungsi f(x) = x² - 2x - 15:

- Koefisien a pada x² adalah 1. Ini menunjukkan bahwa parabola memiliki bukaan ke atas karena a > 0.

- Koefisien b pada x adalah -2. Ini menunjukkan posisi sumbu simetri.

- Koefisien c adalah -15. Ini menentukan posisi titik puncak.

Untuk menentukan sumbu simetri (x = h), kita menggunakan rumus x = -b/2a. Dalam kasus ini, h = -(-2) / (2*1) = 1.

Untuk menentukan titik puncak (h, k), kita substitusikan nilai h ke dalam fungsi. Dalam kasus ini, k = f(1) = 1² - 2(1) - 15 = -16. Jadi, titik puncak adalah (1, -16).

Selanjutnya, kita dapat menentukan pembukaan parabola dan menemukan pembuat nol dengan memfaktorkan fungsi kuadrat atau menggunakan persamaan kuadrat.

Untuk memfaktorkan fungsi f(x) = x² - 2x - 15, kita akan mencari dua faktor yang ketika dikalikan menghasilkan -15 dan ketika ditambahkan menghasilkan -2.

Mencari faktor dari -15:

-1 x 15 = -15

-3 x 5 = -15

Dari pengamatan di atas, kita dapat mencoba mencari kombinasi faktor yang menghasilkan -2 ketika dijumlahkan:

-3 + 5 = 2

Jadi, kita dapat memfaktorkan f(x) sebagai (x - 3)(x + 5).

Dengan memfaktorkan f(x), kita dapat menemukan pembuat nol dengan mengatur faktor-faktor menjadi nol:

x - 3 = 0 atau x + 5 = 0

Dari persamaan ini, kita dapat menemukan dua pembuat nol:

x = 3 atau x = -5

33. diketahui fungsi f(x)=x²-4x-32 tentukan pembuat nol fungsi​


Penjelasan dengan langkah-langkah:

Untuk mencari akar-akar fungsi f(x)=x²-4x-32, kita dapat menggunakan rumus kuadrat. Rumus kuadrat adalah sebagai berikut:

x = (-b ± √(b² - 4ac)) / 2a

Dalam rumus tersebut, a, b, dan c adalah koefisien-koefisien dari fungsi kuadrat f(x). Dalam hal ini, a = 1, b = -4, dan c = -32. Kita dapat mengganti nilai-nilai tersebut ke dalam rumus kuadrat dan menyelesaikannya:

x = (-(-4) ± √((-4)² - 4(1)(-32))) / 2(1)

x = (4 ± √(16 + 128)) / 2

x = (4 ± √144) / 2

x1 = (4 + 12) / 2 = 8/2 = 4

x2 = (4 - 12) / 2 = -8/2 = -4

Jadi, pembuat nol fungsi f(x) adalah 4 dan -4.

Jawab:

Untuk menentukan pembuat nol fungsi f(x) = x^2 - 4x - 32, kita perlu mencari nilai x yang membuat f(x) = 0.

Langkah-langkahnya adalah sebagai berikut:

Ganti f(x) dengan 0.

f(x) = x^2 - 4x - 32 = 0

Cari nilai x dengan menggunakan rumus kuadrat.

x = [-b ± √(b^2 - 4ac)] / 2a

Dalam hal ini, a = 1, b = -4, dan c = -32.

x = [-(-4) ± √((-4)^2 - 4(1)(-32))] / 2(1)

x = [4 ± √(16 + 128)] / 2

x = [4 ± √144] / 2

x = [4 ± 12] / 2

Sehingga diperoleh dua akar yaitu:

x1 = (4 + 12) / 2 = 8

x2 = (4 - 12) / 2 = -4

Sehingga pembuat nol fungsi f(x) = x^2 - 4x - 32 adalah x1 = 8 dan x2 = -4.
