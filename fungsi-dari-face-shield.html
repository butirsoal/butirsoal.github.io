3. toko A mempunyai 280 face Shield sedangkan toko B mempunyai 560 face Shield . berapa perbandingan face Shield toko A dan BTolong bantu kak (╯︵╰,)​


Perbandingan face Shield toko A dan B adalah 1 : 2.

Diketahui :Toko A mempunyai 280 face ShieldToko B mempunyai 560 face Shield

Ditanyakan :Perbandingan face Shield toko A dan B

PENYELESAIAN.

[tex]\tt{280 : 560}[/tex]

[tex]\tt{=\frac{280}{560}}[/tex]

[tex]\tt{=\frac{280 \:\div 280}{560 \:\div 280}}[/tex]

[tex]\tt{=\frac{1}{2}}[/tex]

[tex]\boxed{\bold{= 1:2}}[/tex]

KESIMPULAN

Jadi perbandingan face Shield toko A dan B adalah 1 : 2.

9. harga 3 masker dan 4 face shield adalah Rp 96.000 sedangkan harga 2 masker dan 5 face shield Rp 99.000. harga 1 masker dan 1 face shield adalah A. Rp 24.000B. Rp 25.000C. Rp 26.000D. Rp 27.000​


Jawaban:

D

Penjelasan dengan langkah-langkah:

langkah pengerjaan ada pada gambar

Jawab:

D. Rp 27.000

Penjelasan dengan langkah-langkah:

*Diketahui :

masker = x

face shield = y

3 masker + 4 face shield = Rp 96.000

3x + 4y = 96.000

2 masker + 5 face shield = Rp 99.000

2x + 5y = 99.000

*Ditanya :

1 masker + 1 face shield = Rp ?

x + y = ?

*Dijawab :

3x + 4y = 96.000

2x + 5y = 99.000

(Karena ingin menghilangkan x nya, jadi cari perkalian yang bisa dikali dan hasilnya sama. Contohnya 3 dan 2 jadi :

3 x 2 = 6

2 x 3 = 6

= 3x(2) + 4y(2) = 96.000(2)

= 6x + 8y = 192.000

= 2x(3) + 5y(3) = 99.000(3)

= 6x + 15y = 297.000

= 6x + 8y = 192.000

= 6x + 15y = 297.000

=        -7y = -105.000

=           y = -105.000/-7 (- : - = +)

=           y = 15.000

(Jika y sudah ketemu, x tinggal dicari dengan memasukkan persamaan awal dan memasukkan nilai y nya)

A. (Persamaan 1)

= 3x + 4y = 96.000

= 3x + 4(15.000) = 96.000

= 3x + 60.000 = 96.000

= 3x = 96.000-60.000

= 3x = 36.000

= x = 36.000/3

= x = 12.000

B. (Persamaan 2)

= 2x + 5y = 99.000

= 2x + 5(15.000) = 99.000

= 2x + 75.000 = 99.000

= 2x = 99.000-75.000

= 2x = 24.000

= x = 24.000/2

= x = 12.000

(Pembuktian bahwa memakai persamaan 1 dan 2 itu hasil nilai x nya pun juga sama )

= x + y

= 12.000 + 15.000

= 27.000

= Rp 27.000 (D)
