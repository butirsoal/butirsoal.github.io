5. Cara menyimpan dokumen dalam flasdiks


1# - Pertama masukkan flashdisk anda pada port USB yang tersedia di komputer atau laptop anda.


2# - Tunggu sampai proses deteksi perangkat baru selesai dan direktori Flashdisk anda muncuk di komputer.


3# - Cari dimana file yang akan anda pindahkan,


4# - Tempatkan kursor pada file, klik kanan, kemudian pilih “send to” kemudian cari direktori flashdisk anda.


5# - Kemudian klik, file secara otomatis akan tersimpan di dalam flashdisk anda.

Atau apabila anda ingin menggunakan cara lain juga bisa, yaitu dengan cara klik kanan pada file atau folder yang akan di pindahkan kemudian pilih "copy". Setelah itu, anda menuju ke direktori flashdisk anda pada windows explorer kemudian klik kanan dan piilih "paste

1. MENYIMPAN DATA KE FLASHDISK DENGAN COPY PASTE

Caranya, arahkan pointer (mouse) pada file / data / dokumen yang ingin disimpan kedalam flashdisk lalu klik kanan, kemudian pilih" Copy".  Setelah itu klik pada flashdisk yang telah terhubung dengan kedalam komputer. Kemudian klik kanan lagi lalu pilih "Paste". Dan file anda sudah berpindah ke flashdisk.


2. MENYIMPAN / MEMINDAH DATA DENGAN DRAG N DROP

Cara kerja drag & drop sebenarnya sama dengan copy paste, namun drag & drop hanya cukup menekan mouse (jangan dilepas) pada file yang ingin disimpan pada flashdisk. Kemudian arahkan file tadi kedalam flashdisk.



3. MENYIMPAN DOKUMEN DENGAN SAVE AS

Apabila ingin menyimpan dokumen seperti dokumen yang masih berjalan pada microsoft word, excel dan lain lain, ini bisa langsung disimpan ke flashdisk. Sebagai contoh anda sedang membuka program microsoft word, maka untuk menyimpan dokumen tersebut cukup klik file, kemudian "Save As". Arahkan pointer ke "Flashdisk" kemudian klik Save, maka dokumen anda sudah tersimpan flasdisk.

20. setelah membuat dokumen baru dokumen akan disimpan di komputer dengan alamat d:data/excel.berikan penjelasan cara menyimpan dokumen baru ​


Jawaban:

Untuk menyimpan dokumen baru di komputer dengan alamat D:data/excel, Anda bisa mengikuti langkah-langkah berikut:

- Buka aplikasi Microsoft Excel atau aplikasi spreadsheet lain yang Anda gunakan.

- Buat dokumen baru dengan mengklik "File" dan pilih "New" atau tekan "Ctrl + N" pada keyboard.

- Setelah dokumen baru dibuat, klik "File" dan pilih "Save As".

- Pada jendela "Save As", pilih drive "D" sebagai lokasi penyimpanan.

- Kemudian, klik folder "data" dan masuk ke dalam folder tersebut.

- Klik folder "excel" dan pastikan folder tersebut dipilih sebagai lokasi penyimpanan.

- Isikan nama untuk dokumen Anda pada kolom "File name" dan pilih format file seperti "Excel Workbook (*.xlsx)" atau format lain yang Anda inginkan.

- Klik "Save" untuk menyimpan dokumen baru pada lokasi yang telah ditentukan.

Sekarang dokumen baru Anda sudah tersimpan di D:data/excel dan siap untuk digunakan atau dibuka kembali di lain waktu.

37. bagaimana cara menyimpan dokumen dengan cara digital​


Jawaban:

Caranya adalah dengan menggunakan media penyimpanan yang disebut dengan cloud storage.

Penjelasan:

Cloud storage merupakan sebuah teknologi penyimpanan data digital yang memanfaatkan adanya virtual server sebagai media penyimpanan. Contohnya seperti Google Drive, DropBox, atau sejenisnya.

Pada umumnya menyimpan dokumen secara digital ini, harus scan masing-masing berkas dokumen terlebih dahulu kemudian barulah disimpan ke berbagai alat digital seperti Laptop/PC, Flashdisk, e-mail dan drive.

Jangan hanya memilih salah satu saja, sebaiknya simpan di semua jenis penyimpanan digital ini.

Jangan lupa berilah nama folder di masing-masing dokumen agar mudah ditemukan jika diperlukan kembali.
