26. jelaskan jenis jenis jquery plugin?​


Jawaban:

jenis - jenis jquery plugin?​

1. Navigation UI

Plugin jQuery Navigation UI adalah plugin jQuery yang di gunakan untuk membuat efek pada bagian menu, agar ketika kita scroll halaman website kita maka bagian menu tersebut akan tetap berada di atas.

2. Content Tabs

Plugin jQuery content tabs adalah proyek open source yag di buat untuk memanipulasi content tab yang ada di website kita.

3. Media Galleries

Media galleries adalah istilah dari jenis plugin jQuery yang di gunakan untuk menampilkan sebuah foto atau video didalam website kita. Salah satu plugin jQuery Media Galleries adalah plugin jQuery lightSlider.js. Sebenarnya ada banyak plugin jQuery yang di gunakan untuk menampilkan foto atau video tinggal kalian memilih mana yang cocok untuk di gunakan di website kalian.

4. Parallax Motion & Scrolling

Parallax adalah plugin jQuery yang biasa di gunakan untuk membuat website seolah-olah hidup atau bisa disebut juga dengan pergeseran yang tampak dari suatu obyek (titk 1) terhadap latar belakang (titik 2) yang timbul akibat perubahan posisi. Di jaman yang modern sekarang ini sudah banyak sekali website yang menerapkan efek  parallax.

27. Bagaimana mengakses dan mendapatkan elemen HTML menggunakan JQuery?


Jawaban:ebagaimana fungsi utama JavaScript, kita memerlukan JavaScript untuk memanipulasi element HTML. Langkah pertama adalah mencari element mana yang akan diubah, apakah itu tag <p>, tag <p> dengan atribut id=”penting”, tag <p> dengan atribut class=”warning”, atau sebuah tombol <button>.

Di dalam JavaScript, untuk mencari element HTML ini kita biasa menggunakan method getElementById(). Method atau fungsi ini berfungsi untuk mencari element HTML berdasarkan atribut id-nya.

Sebagai contoh, jika saya memiliki sebuah kode HTML berikut:

<button id="tombol">Click Me!</button>

Maka untuk mengakses element ini kita bisa menggunakan:

var x = document.getElementById("tombol");

Dengan menggunakan jQuery Selector, kode programnya jauh lebih singkat dan lebih powerfull.
