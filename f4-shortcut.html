18. apa pengertian dari shortcut itu ?


shortcut merupakan suatu jalan pintas untuk melakukan perintah dalam programShortcut di dalam sistem operasi Microsoft Windows, shortcut merujuk pada file yang berukuran kecil yang digunakan untuk membuka program atau file lain pada direktori tertentu. Alih-alih mencari dan menemukan file dalam Windows Explorer, dengan menggunakan shortcut, kita bisa membuka file tanpa perlu menemukan lokasi asli file. Jika bersama-sama menjadi tombol ketik yang ada akan susah menggunakan ke2nya karena Alt dan F4 tidak ada di tombol ketik. Alt di tombol opsi, sedangkan F4 di tombol fungsi, jika ke2 itu pindah bersama-sama akan susah mengoperasikan atau melakukannya dan tidak diperbolehkan atau diperkenankan.

Penjelasan:

A. Alt (tombol fungsi); F4 (tombol ketik)

B. Alt dan F4 sama-sama di kelompok tombol fungsi opsi

Jika bersama-sama menjadi tombol opsi yang ada akan susah menggunakan ke2nya karena Alt di tombol opsi, sedangkan F4 tombol fungsi dan diperbolehkan atau diperkenankan juga memakai atau menggunakan seperti ini. (Jika ke2nya berada pada satu tombol yang ada akan rusak keyboardnya karena ke2 itu lain tombolnya).

Ini baru diperbolehkan atau diperkenankan karena ini yang umu dipakai orang untuk memudahkan menutup item, aplikasi, dan program yang sedang aktif (Ini baru diperkenankan atau diperbolehkan karena yang banyak digunakan atau dipakai).

D. Alt dan F4 sama-sama di kelompok tombol ketik

Jika bersama-sama menjadi tombol ketik yang ada akan susah menggunakan ke2nya karena Alt dan F4 tidak ada di tombol ketik. Alt di tombol opsi, sedangkan F4 di tombol fungsi, jika ke2 itu pindah bersama-sama akan susah mengoperasikan atau melakukannya dan tidak diperbolehkan atau diperkenankan (Jika ke2nya berada pada satu tombol yang ada akan rusak keyboardnya karena ke2 itu lain tombolnya).

34. Shortcut untuk mengakhiri pengoperasian komputer adalah a.Alt+F4 b.Alt+F3 c.Ctrl+F4 d.Ctrl+F3 tolong dibantu ya


Jawaban:

C. Ctrl+F4

Penjelasan:

Pada Windows (semua versi), Alt + F4 berfungsi untuk menutup jendela program yang terbuka ataupun mengambil tindakan untuk komputer yang sedang digunakan.

Saat kita menekan Alt+F4, akan muncul jendela kecil dengan pilihan menu :

1. shutdown = mematikan komputer

2. restart = memulai ulang komputer

3. sleep = menidurkan komputer

4. hibernate = tidur panjang

6. switch user = pindah user

7. log off = akun yang digunakan keluar dari sistem Windows


35. Shortcut untuk menambahkan Sheet adalah .... a. Ctrl + B b. Shift + F11 c. d. Alt + F4 Ctrl + F2​


Jawaban:

b). Shift + F11

Penjelasan:

Shortcut Shift + F11 digunakan untuk menambahkan sheet baru dalam program spreadsheet seperti Microsoft Excel atau Google Sheets. Ketika Anda menggunakan shortcut ini, program akan membuat sebuah sheet baru di sebelah kiri sheet aktif Anda.

Shortcut ini sangat berguna ketika Anda ingin dengan cepat menambahkan sheet baru tanpa harus menggunakan menu atau ikon tambahan. Anda dapat langsung menekan kombinasi tombol Shift + F11 dan sheet baru akan langsung ditambahkan ke dalam workbook Anda.

Namun, penting untuk dicatat bahwa shortcut ini mungkin berbeda tergantung pada program spreadsheet yang Anda gunakan. Oleh karena itu, pastikan untuk memeriksa panduan atau dokumentasi program yang Anda gunakan jika shortcut ini tidak berfungsi untuk Anda.
