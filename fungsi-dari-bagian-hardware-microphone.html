33. Hardware berdasarkan fungsinya dibagimenjadi 4, sebutkan dan berikan contohmasing-masing 4 perangkat !​


Hardware berdasarkan fungsinya dapat dibagi menjadi empat kategori, yaitu:

1. Input Device (Perangkat Masukan)

  Contoh:

  - Keyboard: digunakan untuk memasukkan data dan memberikan perintah melalui tombol-tombol pada papan ketik.

  - Mouse: digunakan untuk menggerakkan kursor di layar komputer dan melakukan seleksi serta interaksi dengan objek grafis.

  - Scanner: digunakan untuk memindai atau mendigitalisasi dokumen fisik menjadi file digital.

2. Output Device (Perangkat Keluaran)

  Contoh:

  - Monitor: menampilkan informasi visual dan grafis yang dihasilkan oleh komputer.

  - Printer: digunakan untuk mencetak dokumen atau gambar dari komputer menjadi bentuk fisik.

  - Speaker: digunakan untuk mengeluarkan suara melalui komputer, baik dalam bentuk musik, suara film, atau suara sistem.

3. Storage Device (Perangkat Penyimpanan)

  Contoh:

  - Hard Drive: merupakan perangkat penyimpanan internal yang digunakan untuk menyimpan data dan program komputer.

  - USB Flash Drive: perangkat portable yang digunakan untuk menyimpan dan mentransfer data dengan mudah.

  - Solid State Drive (SSD): perangkat penyimpanan solid-state yang lebih cepat dan lebih tahan terhadap guncangan dibandingkan hard drive tradisional.

4. Processing Device (Perangkat Pemrosesan)

  Contoh:

  - Central Processing Unit (CPU): otak dari komputer yang melakukan pemrosesan data dan menjalankan program.

  - Graphics Processing Unit (GPU): bertanggung jawab untuk memproses dan menghasilkan gambar dan video pada komputer.

  - Microprocessor: merupakan chip semikonduktor yang mengerjakan tugas pemrosesan dalam perangkat elektronik seperti smartphone dan tablet.

Perlu diingat bahwa ini hanya beberapa contoh perangkat keras dalam masing-masing kategori dan ada banyak perangkat keras lainnya yang dapat masuk ke dalam kategori yang sama berdasarkan fungsinya.

36. Sistem Komputer memilik 2 elemen penting , yaitu hardware dan software . Pada hardware ( perangkat keras ) mempunyai 3 fungsi . Sebutkan 3 fungsi / bagian dari hardware dan berikan contoh dari masing – masing komponen tersebut !​


Jawaban:

1. CPU ( Central Processing Unit ) - CPU adalah bagian inti dari hardware yang bertugas untuk melakukan perhitungan aritmatika dan logika serta mengontrol semua kegiatan yang terjadi di komputer. Contoh : Intel Core i3/i5/i7, AMD Ryzen.

2. RAM ( Random Access Memory ) - RAM adalah bagian hardware yang digunakan untuk menyimpan data dan informasi secara sementara. Contoh : DDR3/DDR4.

3. Hard Disk - Hard Disk adalah bagian hardware yang digunakan untuk menyimpan data dan informasi secara permanen. Contoh : Seagate, Western Digital.
