13. jelaskan fungsi column, row, pointer, sheet, scroll bar, dan status bar pada Ms. Excel! ​


Column atau Kolom - berfungsi untuk menampilkan nama kolom yang diidentifikasi dengan huruf A-XFD. Anda dapat menggunakan CTRL + → untuk menuju kolom terakhir Microsoft Excel 2016.

Row atau Baris - Berfungsi untuk menampilkan index baris pada Microsoft Excel yang diidentifikasi dengan angka 1-1048576. Anda dapat menggunakan CTRL + ↓ untuk menuju baris terakhir Microsoft Excel 2016.

Pointer - sebagai objek atau penunjuk cell, dan untuk operasi lainnya.

Tab Worksheet atau sheet - Berfungsi untuk menambah atau mengurangi worksheet serta memilih worksheet yang akan digunakan pada lembar kerja.

Scroll bar

Horizontal Scroll Bar - berfungsi untuk menggeser lembar kerja ke kanan atau ke kiri.Vertical Scroll Bar - Berfungsi untuk menggeser lembar kerja ke atas atau ke bawah.

Status Bar - Berfungsi untuk menampilkan keterangan terkait proses yang terjadi serta rekomendasi perintah yang akan dilakukan pengguna.

40. Jelaskan perbedaan fungsi delete entire row dengan delete entire column


Sebelum menjawab pertanyaan mari pahami terlebih dahulu pengertian dari opsi Delete entire. Delete entire merupakan sebuah cara untuk menghapus suatu data dalam satu rentang data baik baris maupun kolom dalam software Microsoft Excel.

Dibawah ini adalah penjelasan antara fungsi delete entire row dan delete entire column:

1. Delete Entire Row

Opsi entire row digunakan saat akan menghapus data di excel, satu atau lebih baris dari ujung kiri sampai ujung kanan. Jadi pada saat memilih opsi ini saat melakukan penghapusan data, semua data yang ada pada baris / rentang baris yang dipilih akan dihapus.

Kelebihan dari delete entire row ini adalah dapat menghapus rentetan data dalam suatu baris jika sudah tidak digunakan lagi secara sekaligus.


2. Delete Entire Columns

Opsi entire column ini hampir sama dengan cara delete menggunakan entire row. Hanya saja pada pilihan ini, penghapusan data dilakukan pada kolom dari ujung atas sampai ujung bawah.

Penggunaan opsi delete entire column ini akan dapat menghapus suatu data atau sekelompok data yang sudah tidak digunakan dalam data bentuk vertical.
