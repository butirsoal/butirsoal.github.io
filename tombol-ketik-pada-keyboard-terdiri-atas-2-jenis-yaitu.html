Secara fisik, keyboard terbagi atas 4 jenis, yaitu:

1. Keyboard Serial

Serial

PS 2

Unit pemancar biasanya terdapat pada keyboard itu sendiri, sedangkan penerima biasanya dipasang pada port USB atau serial pada CPU.

Wireless

Komputer terbaru saat ini sudah banyak yang mempergunakan jenis konektorUSB yang menjamin transfer data lebih cepat .

23. Tuliskan 20 jenis tombol keyboard beserta fungsinya


Jawaban:

Berikut adalah 20 jenis tombol keyboard beserta fungsinya:

Tombol ESC: Tombol ini berfungsi untuk menghentikan, membatalkan, atau keluar dari aplikasi secara cepat.Tombol Tab: Tombol ini berfungsi untuk membuat kursor bergeser dengan jarak 1 tabulasi ke arah kanan.Tombol Caps Lock: Tombol ini berfungsi untuk membuat huruf atau tulisan menjadi besar (uppercase).Tombol Windows (Home): Tombol ini berfungsi untuk mengarahkan tampilan pada halaman awal atau halaman start secara cepat.Tombol Space (Spasi): Tombol ini berfungsi untuk menambahkan spasi atau jarak antara 1 kata dengan kata lainnya.Tombol Delete: Tombol ini berfungsi untuk menghapus 1 karakter di kanan/depan kursor.Tombol Enter: Tombol ini berfungsi untuk berpindah baris.Tombol Ctrl: Tombol ini berfungsi sebagai tombol kontrol yang dapat digabungkan dengan tombol lain untuk membuat perintah-perintah tertentu. Misalnya, Ctrl + C untuk menyalin, Ctrl + V untuk menempel, Ctrl + Z untuk membatalkan, dan sebagainya.Tombol Alt: Tombol ini berfungsi sebagai tombol alternatif yang dapat digabungkan dengan tombol lain untuk membuat perintah-perintah tertentu. Misalnya, Alt + F4 untuk menutup program, Alt + Tab untuk beralih antara jendela, Alt + PrtSc untuk mengambil screenshot, dan sebagainya.Tombol Shift: Tombol ini berfungsi sebagai tombol pergeseran yang dapat digabungkan dengan tombol lain untuk membuat perintah-perintah tertentu. Misalnya, Shift + F3 untuk mengubah huruf besar kecil, Shift + Del untuk memotong, Shift + Ins untuk menempel, dan sebagainya.Tombol F1-F12: Tombol-tombol ini berfungsi sebagai tombol fungsi yang dapat digunakan untuk melakukan perintah-perintah tertentu tergantung pada program yang sedang dijalankan. Misalnya, F1 untuk membuka bantuan, F5 untuk menyegarkan halaman web, F12 untuk menyimpan dokumen, dan sebagainya.Tombol PrtSc (Print Screen): Tombol ini berfungsi untuk mengambil screenshot atau gambar dari layar komputer atau laptop.Tombol Num Lock: Tombol ini berfungsi untuk mengaktifkan atau menonaktifkan tombol angka pada bagian kanan keyboard.Tombol Scroll Lock: Tombol ini berfungsi untuk mengaktifkan atau menonaktifkan fungsi scroll pada mouse atau keyboard.Tombol Pause/Break: Tombol ini berfungsi untuk menghentikan sementara atau memutuskan proses yang sedang berjalan pada komputer atau laptop.Tombol Insert: Tombol ini berfungsi untuk memasukkan teks baru tanpa menghapus teks yang ada di belakang kursor.Tombol Home: Tombol ini berfungsi untuk memindahkan kursor ke awal baris teks.Tombol End: Tombol ini berfungsi untuk memindahkan kursor ke akhir baris teks.Tombol Page Up: Tombol ini berfungsi untuk menggulir layar ke atas satu halaman.Tombol Page Down: Tombol ini berfungsi untuk menggulir layar ke bawah satu halaman.

25. Sebutkan jenis-jenis keyboard berdasarkan tombol dan jelaskan pengertiannya​


Keyboard adalah Alat input yang digunakan untuk mengetik informasi kedalam komputer dan menjalankan berbagai intruksi atau perintah ke dalam komputer.

Jenis-Jenis Keyboard

1. Berdasarkan Susunan Tombolnya

Keyboard QWERTY  : dibuat berdasarkan layout mesin tik. Tata letak ini ditemukan oleh Scholes, Glidden dan Soule pada tahun 1878, dan kemudian menjadi standar mesin tik komersial pada tahun 1905. Keyboard QWERTY didesain sedemikian rupa sehingga key yang paling sering ditekan terpisah letaknya sejauh mungkin, sehingga bisa meminimalkan kemacetan pada saatmengetik (pada mesin ketik mekanik). Keyboard DVORAK  : Keyboard ini dibuat pada tahun 1936. Keyboard Dvorak diciptakanberdasarkan prinsip kerja biomekanis dan efisiensi. Susunan letak tombol huruf dengan jenis QWERTY dibuat sedemikian rupa sehingga 56 % ketukan ada padatangan kanan dan jari-jari yang lebih banyak bekerja adalah jari telunjuk, jari tengah, dan jari manis. Huruf-huruf yang ada pada baris tengah lebih seringdiketuk kira-kira sampai 70 % dan perpindahan antar baris hanya sekitar 10 %sehingga kelelahan jari-jari sangat banyak berkurang. Keyboard KLOCKENBERG  : Keyboard ini dibuat dengan maksud menyempurnakan jenis keyboard yang sudah ada, yaitu dengan memisahkan kedua bagian keyboard (bagian kiridan kanan). Bagian kiri dan kanan keyboard dipisahkan dengan sudut 15 derajatdan dibuat miring ke bawah. Selain itu, keyboard KLOCKENBERG mempunyai tombol-tombol yang dibuat lebih dekat (tipis) dengan meja kerja sehingga terasalebih nyaman.

2. Berdasarkan Konektornya

Keyboard Serial  : Menggunakan DIN 5 male dan biasanya digunakan pada computer tipe AT. Keyboard Serial adalah keyboard yang terhubung ke computer menggunakan port Serial. Keyboard PS/2  : Biasanya digunakan pada komputer ATX dan saat ini yang paling banyak dipergunakan. Pemasangan keyboard tipe ini harus dilaksanakan dengan cermat, sebab port yang dimiliki sama dengan port untuk mouse. Keyboard Wireless  : Sesuai dengan namanya, keyboard tipe ini tidak menggunakan kabel sebagai penghubung antara keyboard dengan komputer. Jenis koneksi yang digunakan adalah infra red, wifi atau bluetooth. Untuk menghubungkan keyboard dengan komputer, dibutuhkan unit pemancar dan penerima. Unit pemancar biasanya terdapat pada keyboard itu sendiri, sedangkan penerima biasanya dipasang pada port USB atau serial pada CPU. Keyboard USB  : Komputer terbaru saat ini sudah banyak yang mempergunakan jenis konektor USB yang menjamin transfer data lebih cepat dan lebih mudah dalam pemasangannya.
