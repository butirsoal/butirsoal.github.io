2. fungsi dialog box launcher pada ms power point 2013 ​


Jawaban:

Fungsinya untuk membuka kotak dialog (dialog box) secara lebih cepat.

Penjelasan:

Ms power point 2013 ​memiliki banyak tab ribbon, salah satunya ribbon Home. Dalam ribbon ini terdapat dialog box launcher pada group Font & Paragraph.

Bila tombol itu diklik maka akan tampil kotak dialog (dialog box) Font & kotak dialog (dialog box) Paragraph.

Perhatikan gambar berikut ini:

tanda panah nomer 1 menunjukkan dialog box launcher di group Font

tanda panah nomer 2 menunjukkan dialog box launcher di group Paragraph

Windows tidak kompatibel dengan kontroler penyimpanan massal atau pengandar penyimpanan massal.

Metode 1: Memverifikasi bahwa partisi yang kompatibel dengan Windows

Catatan Saat Anda memformat partisi, semua data yang dihapus dari partisi. Data ini mencakup semua berkas di partisi. Untuk informasi selengkapnya tentang cara menggunakan perintah Convert.exe, klik nomor artikel berikut ini untuk melihat artikel di Pangkalan Pengetahuan Microsoft:

314097 bagaimana cara menggunakan Convert.exe untuk mengubah partisi ke sistem berkas NTFS

Jawaban:

1. Klik tab ribbon Insert

Untuk membuat Drop Cap:

1. Klik tab ribbon Insert

2. Klik Drop Cap dan klik salah 1 opsi yg disediakan

Word Art dan Drop Cap terdapat dalam grup Text (di tab ribbon Insert). Perhatikan gambar berikut ini:

nomor 1 menunjukkan posisi tab ribbon insert

nomor 2 menunjukkan posisi menu Word Art

nomor 3 menunjukkan posisi menu Drop Cap

18. Membuat huruf besar di awal paragraf dapat menggunakan a. Text Box c. Picture b. Drop Caps d. WordArtsoalnya besok dikumpulin​


Jawaban:

B. Drop Caps.

Penjelasan:

Drop Caps adalah teknik dalam pemformatan dokumen yang digunakan untuk menonjolkan huruf pertama di awal paragraf dengan memperbesar huruf pertama tersebut dan menempatkannya di tengah atas dua atau tiga baris pertama paragraf. Teknik ini umumnya digunakan pada buku, majalah, atau dokumen formal lainnya.

Opsi a. Text Box digunakan untuk menampilkan teks atau gambar dalam kotak yang dapat ditempatkan di lokasi tertentu pada halaman. Opsi c. Picture digunakan untuk menyisipkan gambar ke dalam dokumen. Opsi d. WordArt digunakan untuk membuat teks yang terlihat kreatif dengan menambahkan efek khusus seperti bayangan, transparansi, atau efek 3D. Namun, tidak berhubungan dengan memperbesar huruf pertama di awal paragraf.
