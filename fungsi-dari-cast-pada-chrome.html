1. apa fungsi dari google chrome?


Fungsi dari Google Chrome adalah utamanya untuk browsing. Google Chrome sendiri juga memiliki fungsi tambahan lain yang memungkinkan penggunanya mensinkronisasi akun Google dengan browser tersebut, pengguna dapat memasang ekstensi yang menunjang kinerja, membuka beberapa bentuk file didalam browser, dan melakukan inspect element pada suatu webpage.

Pembahasan

Google Chrome adalah salah satu produk hasil pengembangan perusahaan Google yang tersedia sebagai web browser maupun di android.  Pengguna Google Chrome sendiri kian meningkat tiap tahunnya dikarenakan fitur yang semakin lengkap dan interface yang lebih user-friendly.

Fungsi utama browser ini adalah untuk menjelajah berbagai web untuk kebutuhan berkirim email, layanan sosial media, search engine dan sebagainya.

Fungsi tambahan browser ini antara lain :

→ Memungkinan pengguna untuk mensinkronisasi akun Google dengan Chrome.

Biasanya ketika pengguna pertama kali menggunakan Chrome akan ditawarkan untuk menghubungkan akun Google aktif pengguna agar dapat terhubung sehingga memudahkan dalam mengintegrasikan fasilitas google yang ada pada android dengan browser. Contohnya, Google Calendar, rekomendasi keyword pada kolom search Google, dan login otomatis pada layanan Google lainnya

→ Pengguna dapat memasang ekstensi yang menunjang kinerja

Ekstensi yang tersedia di Chrome sangat beragam dan akan sangat mendukung kinerja penggunanya. Contoh : pemblokir iklan, ekstensi tools perusahaan, ekstensi mempercepat download/upload file dan sebagainya.

→ Membuka beberapa bentuk file didalam browser

Chrome selain sebagai browser untuk menjelajah web, browser ini dapat digunakan untuk membuka file seperti .pdf, docx, .mp3(audio), mp4 (video)

→ Melakukan inspect element pada suatu webpage

Hal ini dapat dilakukan di Chrome dengan tujuan mengedit suatu tampilan web pada susunan HTML / CSS namun hal tersebut hanya dapat dilihat pada komputer pengguna yang melakukan perubahan denga inspect element.

Penjelasan:

Film The King's Speech dianugerahi Oscar untuk skenario ...

A. diberikan untuk anggota pemerannya

B. untuk anggota pemerannya

C Anggota pemeran

D. itis untuk anggota pemeran

Jawaban:

B

Penjelasan:

Bila kita perhatikan kalimat 'The movie The King's Speech is awarded Oscar for its screenplay as well as.... merupakan kalimat yang menggunakan frasa “as well as”. Tapi tidak sedikit yang ternyata bingung terutama dengan cara menerjemahkannya, setidaknya itu juga yang saya alami sendiri ketika awal-awal belajar bahasa Inggris. As well as sendiri bisa diartikan dengan dan juga atau begitu juga. As well as merupakan salah satu bentuk conjunctions atau kata penghubung yang dapat menghubungkan dua kata, kalimat atau frase.

Formula yang digunakan untuk subjek tunggal antara lain sbb:

Formula 1

Subject + Verb + Adjective, as well as, Adjective.

Contoh:

– Andy is smart.

– Andy is also handsome.

Jika digabung menjadi:

Andy is smart as well as handsome (Andy pintar dan juga ganteng)

Formula 2

Subject + Verb + Adverb, as well as, Adverb.

Contoh:

– Andrew writes fast.

– He also writes neatly.

Jika digabung menjadi :

Andrew writes fast as well as neatly. (Andrew menulis dengan cepat dan juga rapi).

Formula 3

Subject + Verb + Prepositional phrase, as well as, Prepositional phrase.

Contoh:

– Bill excels in mathematics.

– He also excels in science.

Jika digabung menjadi :

Bill excels in mathematics as well as in science. (Andy menguasai matematikan dan juga IPA)

Formula 4

Subject + Verb + Complement, as well as, Verb + Complement

– Smith plays the piano.

– He composes music.

Jika digabung menjadi:

Smith plays the piano as well as composes music. (Smith bermain piano dan juga mengarang lagu).

Frase for its screenplay as well as.... dalam soal akan seimbang jika diisi dengan for its cast member sesuai Formula 3.

30. apa fungsi google chrome 1.0.154.53 ?​


Jawaban:

Versi Google Chrome yang dipasang di host jarak jauh lebih lama dari 1.0.154.53. Versi tersebut dilaporkan dipengaruhi oleh beberapa kerentanan:

- Browser menggunakan header Host HTTP untuk menentukan konteks respons CONNECT 4xx/5xx dari server proxy. Ini dapat memungkinkan penyerang man-in-the-middle untuk mengeksekusi kode skrip arbitrer dalam konteks server yang sah. (CVE-2009-2060)

- Browser menampilkan sertifikat yang di-cache untuk halaman respons CONNECT 4xx/5xx dari server proxy. Penyerang man-in-the-middle dapat mengeksploitasi ini dengan menampilkan halaman web palsu dengan sertifikat valid dari situs web yang sah.

perbaikan SSL. (Masalah: 8706)

32. deskripsi Google Chrome​


Google Chrome adalah peramban web yang dikembangkan oleh Google. Peramban web ini dirancang untuk memberikan pengalaman penjelajahan web yang cepat, aman, dan mudah digunakan. Google Chrome menjadi salah satu peramban web paling populer di dunia dengan fitur-fitur canggih dan dukungan yang kuat.

Salah satu keunggulan Google Chrome adalah kecepatan dalam memuat halaman web. Peramban ini menggunakan teknologi yang disebut Chrome V8 JavaScript engine yang mengoptimalkan kinerja perangkat keras dan perangkat lunak untuk mempercepat penjalanan kode JavaScript pada halaman web.

Selain itu, Google Chrome juga memiliki antarmuka pengguna yang sederhana dan intuitif. Pengguna dapat dengan mudah membuka tab baru, mengelola tab yang ada, dan melakukan navigasi web dengan mudah menggunakan bilah alamat dan tombol navigasi yang terintegrasi dengan baik.

Fitur keamanan juga menjadi perhatian utama Google Chrome. Peramban ini secara otomatis memperbarui dirinya sendiri untuk menjaga keamanan pengguna dengan menghadirkan pembaruan keamanan terbaru. Fitur sandboxing juga digunakan untuk mengisolasi setiap tab yang berjalan dalam proses terpisah, sehingga melindungi pengguna dari ancaman malware dan serangan keamanan.

Google Chrome mendukung ekosistem ekstensi yang kaya, yang memungkinkan pengguna memperluas fungsionalitas peramban dengan menambahkan ekstensi yang dapat menghadirkan fitur-fitur tambahan seperti blokir iklan, manajemen kata sandi, dan alat pengembang.

Selain itu, Google Chrome juga menyinkronkan data pengguna di berbagai perangkat yang terhubung dengan akun Google. Misalnya, riwayat penjelajahan, bookmark, dan pengaturan dapat diakses dan disinkronkan di perangkat yang berbeda.

Dalam beberapa tahun terakhir, Google Chrome telah menjadi peramban web yang sangat populer dengan banyak pengguna di seluruh dunia.
