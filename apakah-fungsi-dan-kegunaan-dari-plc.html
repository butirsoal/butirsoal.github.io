2. Jelaskan fungsi motor 3 phase pada rangkaian instalasi forward-reverse motor listrik menggunakan PLC!


Motor 3 phase pada rangkaian instalasi forward-reverse motor listrik menggunakan PLC berfungsi untuk menggerakkan mesin atau alat yang membutuhkan putaran yang dapat diatur arahnya. Dalam sistem ini, motor 3 phase digunakan untuk menggerakkan motor listrik secara bolak-balik (forward-reverse) dengan menggunakan PLC sebagai pengontrolnya.

PLC (Programmable Logic Controller) adalah suatu sistem kontrol yang menggunakan program dan logika matematika untuk mengendalikan operasi dari mesin atau peralatan yang terhubung. Dalam rangkaian instalasi forward-reverse motor listrik menggunakan PLC, PLC digunakan untuk mengontrol arah putaran motor 3 phase yang terhubung dengan mesin atau peralatan yang diperlukan.

Dengan menggunakan motor 3 phase dan PLC dalam instalasi forward-reverse motor listrik, pengguna dapat mengendalikan arah putaran motor dengan mudah dan akurat sesuai dengan kebutuhan aplikasi. Hal ini memungkinkan pengguna untuk menghemat waktu dan energi dalam mengoperasikan mesin atau alat yang membutuhkan putaran yang dapat diatur arahnya.

Bagaimana cara kerjanya?

Sebuah PLC menerima informasi dari sensor yang terhubung dan perangkat input. Kemudian, itu memproses data itu dan menciptakan output berdasarkan parameter yang telah ditetapkan. Bergantung pada pengaturan yang tepat, sebuah PLC bahkan dapat memonitor dan merekam data run-time, menganalisis produktivitas lini mesin atau suhu operasi

Karakteristik PLC

Apa yang membedakan PLC dari mikrokontroler, PLC industri skala besar, atau solusi kontrol industri lainnya? Mari kita lihat beberapa karakteristik utama.

Input / Output

PLC termasuk CPU yang menyimpan dan memproses data program. Namun, ia bergantung pada modul input dan output untuk menghubungkannya ke seluruh mesin. Modul I / O ini mengirimkan informasi ke CPU, memicu hasil tertentu.

Kontrol PLC I / O dapat berupa analog atau digital. Input dapat berkisar dari sensor dan sakelar hingga meter, dan output mencakup relay, katup, drive, dan lampu. Satu keuntungan yang ditawarkan PLC adalah bahwa pengguna dapat menyesuaikan I / O mereka untuk memenuhi persyaratan konfigurasi untuk aplikasi mereka.

Antarmuka Mesin Manusia

Ketika manusia ingin berinteraksi dengan PLC secara real-time, mereka bergantung pada Human Machine Interface (HMI) untuk melakukannya. Ini dapat berkisar dalam skala dari tampilan sederhana dengan tombol dan teks hingga panel layar sentuh yang cerdas seperti elektronik konsumen modern.

Komunikasi

Sementara perangkat I / O adalah permulaan, PLC mungkin juga perlu terhubung ke berbagai jenis sistem. Misalnya, pengguna mungkin perlu mengekspor data aplikasi yang direkam dari PLC dan mengirimkannya ke sistem Pengawasan dan Pengadaan Data (SCADA).

Untuk memfasilitasi fungsi ini, kebanyakan PLC menyertakan banyak port dan protokol komunikasi untuk memastikan bahwa perangkat dapat berkomunikasi dengan sistem lain.

15. sebutkan 3 jenis PLC dan jelaskan cara kerja serta spesifikasi dan software yang digunakan​


Jawaban:

Informasi mengenai jenis-jenis PLC beserta fungsinya dapat menjadi dasar bagi Anda yang ingin mempelajari bahasa pemrograman. PLC merupakan singkatan dari programmable logic controller. Programmable sendiri memiliki pengertian kemampuan menyimpan memori.

Sementara logic sama dengan kemampuan memproses input baik secara aritmatika dan logika. Yang dilakukan adalah operasi membandingkan, mengalikan, membagi, menjumlahkan, negasi, mengurangi, AND, OR, dan berbagai fungsi matematika lainnya.

PLC terbangun atas berbagai komponen, diantaranya CPU, memory, power supply, programmer, dan input serta output. Semua komponen tersebut menjalankan fungsi sesuai dengan bagiannya dan menjadi satu kesatuan tidak bisa terpisahkan sebagai bagian dari PLC.

Dari berbagai jenis-jenis PLC di atas, dapat ditemui berbagai macam lagi bagian-bagian kecil yang menjalankan fungsi secara lebih spesifik. Contohnya saja untuk memori terbagi menjadi dua jenis, yakni volatile dan non-volatile. Keduanya memiliki perbedaan spesifikasi.

Untuk jenis memori volatile, penyimpanan akan otomatis hilang jika daya sambungnya dilepaskan. Sementara untuk memori non-volatile penyimpanan tidak berpengaruh terhadap daya sambung tersebut. Banyak hal menjadi dasar jenis kontrol logika terprogram dalam industri.

38. Aplikasi penulisan program PLC Omron CPM1A menggunakan….


Jawaban:

6 Software PLC Gratis :

1. WinProLadder PLC Software

Software pemrograman PLC jenis ini dapat digunakan dengan PLC Fatek. Winproladder adalah perangkat lunak pemrograman PLC yang cocok untuk pemula. Sobat Teknik bisa mengerjakan banyak proyek dengan menggunakan software ini.

WinProLadder ini terdapat tools simulasi. Kita dapat menjalankannya tanpa menggunakan perangkat keras PLC sehingga sangat pas untuk perkuliahan online seperti sekarang ini.

Selain itu, software ini tersedia untuk windows dan linux, jadi sobat bisa memilih sesuai laptop yang dimiliki. Canggihnya lagi, program WinProLadder sekarang mampu diedit secara online.

2. I-TriLoGi

Software yang kedua ada i-TriLoGi. Dengan menggunakan software ini sobat bisa mensimulasikan program PLC secara real-time. I-TriLoGi bisa mendeteksi apakah program yang dibuat berjalan dengan baik atau tidak. Menariknya, perangkat lunak ini mampu menjalankan program tanpa terhubung ke PLC.

Selain itu, i-TriLoGi terdiri dari mode pengalamatan sederhana sehingga mudah digunakan. Tetapi sobat Teknik bisa menggunakan versi berbayar juga kok, cukup dengan PLC kabel FMD dan EZ.

3. WPLsoft

Selanjutnya perangkat lunak pemrograman PLC ini dikembangkan oleh Delta Electtronic yang dikembangkan untuk windows. WPLsoft hanya dapat digunakan dengan PLC delta. Dengan bantuan software jenis ini, sobat teknik bisa melakukan banyak proses otomasi industri.

Terdevelop dengan bahasa pemrograman ladder, fitur utama dari WPLsoft adalah memiliki simulator PLC built-in dan dengan menggunakannya sobat bisa memeriksa program PLC secara real-time.

Selain itu, fitur lainnya ada fitur perbandingan program. Canggihnya, sobat teknik dapat membuat program PLC ini dengan tiga bahasa pemrograman seperti STL, logika tangga, dan teks sekuensial.

4. Micro Ladder

Micro Ladder adalah software pemrograman PLC gratis lainnya untuk windows yang hanya digunakan dengan PLC Siera. Artinya sobat teknik tidak akan bisa menggunakan software ini dengan PLC lain kecuali Siera.

Fitur utama dari Micro Ladder adalah terdiri dari banyak instruksi I/O digital dan analog. Micro Ladder bisa gunakan dengan dua bahasa pemrograman yaitu logika tangga dan C.

5. Open PLC Editor

Software gratis kelima adalah Open PLC Editor. Seperti yang lainnya, masih didevelop untuk OS windows.

Fitur utama dari perangkat lunak ini adalah dapat membuat program dengan salah satu dari lima bahasa pemrograman PLC. Menarik ya, jadi sobat tidak perlu menguasai banyak bahasa pemrograman untuk PLC.

Open PLC Editor terdiri dari semua fungsi pemrograman, dan memiliki simulator bawaan lho.

6. Do-more Designer

Software yang terakhir dengan antarmuka user friendly ini, Do-more Designer hanya menggunakan bahasa pemrograman logika tangga. Perangkat lunak ini terdiri dari simulator bawaan yang berfungsi untuk menguji program yang sobat buat.

Do-more Designer dapat menggunakan browser web apa pun untuk menentukan status PLC. Fitur utamanya adalah memiliki banyak tools pendukung ketika sobat menginstalnya di PC. Tools pendukung ini akan berguna untuk membuat program dan juga untuk komunikasi yang lebih baik.

Itu dia beberapa software untuk memprogram PLC yang mayoritas dapat berjalan di OS Windows dan gratis. Sobat Teknik tinggal pilih mau yang mana dan disesuaikan sesuai kebutuhan. Semoga bermanfaat
