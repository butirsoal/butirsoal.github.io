Public class Mahasiswa implement Manusia B. public class Mahasiswa implements Manusia C. public class Mahasiswa extend Manusia D. public class Mahasiswa extends Manusia


Jawaban:

D. public class Mahasiswa extends Manusia

Penjelasan:

Kata kunci extends digunakan untuk mendeklarasikan kelas turunan (subclass) dari kelas atasnya (superclass). Dengan kata kunci extends, kelas turunan akan mewarisi data field dan metoda yang dimiliki oleh kelas di atasnya.

Sebenarnya tidak semua data field dan metoda diwariskan kepada kelas turunnannya. Data field dan metoda yang dideklarasikan dengan modifier private, tidak diwariskan kepada kelas turunannya. Berikut ini adalah sintaksis untuk mendeklarasikan kelas turunan (subclass) dari superclass:

public class nama-subclass extends nama-superclass {

  // data dan metoda

}

27. Teknik-teknik pengkodean memiliki kegunaan yang berbeda, teknik pengkodean manakah yang digunakan pada Public Telephone Network


Teknik-teknik pengkodean memiliki kegunaan yang berbeda, teknik pengkodean yang digunakan pada Public Telephone Network adalah b. Data digital, sinyal analog

Pembahasan

Pada jaringan telepon umum, device yang dipakai adalah sebuah modem (modulator dan demodulator) yang berfungsi mengubah data digital ke sinyal analog (modulator) dan sebaliknya mengubah sinyal analog ke data digital (demodulator).

Karena operasi modulasi meliputi 1 atau lebih dari 3 besaran yang terlibat pada sinyal pembawa yaitu amplitudo, frequency, dan phase, dimana sinyal yang dihasilkan menempati pusat bandwidth pada frequency pembawa.

Kelemahan komunikasi dengan sinyal digital adalah jarak tempuh pendek akibat pengaruh redaman/derau yang terjadi pada media transmisi.

Sedangakan pengiriman sinyal analog memiliki kemampuan menghantarkan sinyal dengan baik meskipun jarak tempuh jauh.

Untuk itu, dibuatlah metode menggunakan sinyal analog untuk pengiriman sinyal digital.  

Sinyal digital mengenal dua keadaan yaitu high dan low (binary), maka digunakan tehnik modulasi untuk mengolah sinyal digital. Dengan teknik modulasi sinyal digital dapat diubah menjadi sinyal analog supaya bisa dikirimkan dengan jarak yang panjang dan setelah diterima dapat diubah kembali menjadi sinyal digital.  Teknik mengubah digital menjadi analog ini disebut dengan demodulasi. Sedangkan gelombang pembawa sinyal ini disebut carrier dan berbentuk sinusoidal.

29. (Java) Buatlah sebuah class Matakuliah yang berisi fields: String kode, String nama, int sks, double nilai. Kemudian lengkapi class dengan method: constructor, getter, setter, dan toString( )​


Jawaban:

java :

public class Matakuliah {

   private String kode;

   private String nama;

   private int sks;

   private double nilai;

   // Constructor

   public Matakuliah(String kode, String nama, int sks, double nilai) {

       this.kode = kode;

       this.nama = nama;

       this.sks = sks;

       this.nilai = nilai;

   }

   // Getter methods

   public String getKode() {

       return kode;

   }

   public String getNama() {

       return nama;

   }

   public int getSks() {

       return sks;

   }

   public double getNilai() {

       return nilai;

   }

   // Setter methods

   public void setKode(String kode) {

       this.kode = kode;

   }

   public void setNama(String nama) {

       this.nama = nama;

   }

   public void setSks(int sks) {

       this.sks = sks;

   }

   public void setNilai(double nilai) {

       this.nilai = nilai;

   }

   // toString method

   @Override

   public String toString() {

       return "Matakuliah{" +

               "kode='" + kode + '\'' +

               ", nama='" + nama + '\'' +

               ", sks=" + sks +

               ", nilai=" + nilai +

               '}';

   }

}

Penjelasan:

Dalam implementasi di atas, terdapat:

Empat fields dengan tipe data yang sesuai dengan spesifikasi: kode, nama, sks, dan nilai.

Constructor yang menerima empat parameter dan digunakan untuk menginisialisasi nilai dari setiap field.

Getter methods untuk masing-masing field, yang digunakan untuk mengakses nilai dari setiap field.

Setter methods untuk masing-masing field, yang digunakan untuk mengubah nilai dari setiap field.

Method toString() yang mengembalikan string representasi dari objek Matakuliah.
