• Jawaban pendek:

Kenapa komputer generasi pertama di masukan dalam buku MODERN OPERATING SYSTEM sedangkan komputer generasi pertama belum ada sistem operasi atau OS?

Karena generasi pertama dari komputer merupakan awal perkembangan sistem komputasi elektronik sebagai pengganti sistem komputasi mekanik, hal itu disebabkan karena terbatasnya kecepatan manusia untuk menghitungdan manusia sangat mudah untuk membuat kecerobohan, kekeliruan atau kesalahan. Pada generasi ini belum ada sistem operasi, maka dibentuklah sistem komputer yang diberi instruksi yang harus dikerjakan secara langsung. Namun, banyak orang pada masa itu yang menyebutnya menjadi Sistem Operasi Modern (Modern Operating System).

Macam-Macam Sistem Operasi dan Kelebihan Kekurangan

1. Sistem Operasi UNIX

UNIX adalah sistem operasi yang pertama ada. UNIX adalah “mama” dari sistem operasi Linux.

Kelebihan UNIX:

Open Source (FREE alias Gratis)Tersedia program pendukung untuk multimedia (built in)Desain desktop oke punya.

Kekurangan UNIX:

User pemula pasti binggung karena sangat berbeda dengan Windows yang familiarExtensi file (format) yang didukung terbatasTingkat kompatibilitas perangkat lunak rendah

2. Sistem Operasi MS-DOS

MS-Dos adalah nenek moyang dari Windows yang fenomenal saat ini. Coba saja tekan tombol windows + R, lalu ketik “cmd” dan tekan Enter. Jika muncul layar hitam dengan tulisan putih, berarti Anda sudah masuk DOS  Berikut kelebihan dan kekurangan sistem operasi DOS:

Kelebihan MS-DOS:

Ukuran program kecil.Memori yang digunakan ringanCocok dengan semua perangkat baik keras maupun lunak

Kekurangan MS-DOS:

Perintah dimasukkan dengan teks. Jika tidak ingat atau lupa, maka akan jadi kesulitan tersendiri.Kurang mendukung aplikasi yang mengedepankan grafik.

3. Sistem Operasi Novell

Sistem Operasi besutan Novell Corp. Jarang kan bertemu dengan OS ini?

Kelebihan Novell:

Sistem operasi yang dibuat khusus untuk sistem jaringanKoneksi yang cepat antar komputer yang terhubung dengan Local Area Network (LAN)Kontrol jaringan yang gampang.

Kekurangan Novell:

Pengoperasiannya menggunakan teks (DOS).Kurang familiar karena keterbatasan peruntukannya

4. Sistem Operasi Windows

Nah, ini nih, sistem operasi sejuta umat  Windows, versi apa dan berapapun tidak bisa dipungkiri adalah sistem operasi terpopuler di dunia. Silakan tanya orang-orang yang pernah memegang komputer. “komputernya pakai apa?” Pasti jawabannya macam macam sistem operasi windows (seperti yang disebutkan di atas)

Kelebihan Windows:

Sangat familiarVersi jaringan (Windows Server) pun sangat dinamis dan kompatibelPengoperasian yang mudah (tinggal klak-klik saja)Port koneksi yang tersedia beragam.

Kekurangan Windows:

Harga software asli Windows termasuk mahal (meski biaya upgrade ke Windows 10 adalah gratis mengikuti Mac OS X)Hardware untuk menjalankannya memiliki minimun requirement yang terbilang tinggi (Prosesor Pentium)Memori besarLisensi jaringan lebih mahal lagi.Rentan terhadap serangan virus.

5. Mac OS X – Macintosh

Ini dia sistem operasi yang mimin pakai

Kelebihan Mac OS X:

Jarang ada virusCepat, jarang nge-lag atau hangTouch Pad super enakPerforma handal.

Kekurangan Mac OS X:

Tidak bisa diinstall selain di komputer/laptop milik Apple. Ada sih yang bisa kemudian diberinama Hackintosh.Harga komputer (iMac) dan Laptop (Macbook) tergolong mahal.

6. Sistem Operasi Linux

Sistem operasi yang dikembangkan oleh Linus Torvald. Sistem operasi yang mengusung sistem terbuka (open source) sehingga dapat dikembangkan siapapun. Sudah banyak distro (turunan) dari Linux ini. Mulai dari Suse, Debian, Fedora Red Hat, Slackware, Ubuntu dll.

Kelebihan Linux:

Handal dalam jaringanPengoperasian bisa dengan dual mode; teks oke, grafis juga bisaTanpa lisensi alias Gratis.

Kekurangan Linux:

Harus belajar tersendiri cara menggunakannya. Orang awam pasti binggung.Kurang diminati karena ketergantungan terhadap software yang ada pada Windows

Macam-Macam Sistem Operasi SmartPhone/Mobile

1. Windows Phone

Seperti namanya, sistem operasi mobile ini adalah milik Microsoft untuk mempersenjatai smartphone mereka. Saat ini kebanyakan seri Nokia Lumia yang menggunakannya.

Kelebihan Windows Phone:

Interface yang mirip dengan versi komputer atau desktopnya. Akan lebih cepat adaptasinya.Hampir semua software versi desktop juga ada versi mobile-nya.Fasilitas sync dengan Windows Live.

Kekurangan Windows Phone:

Kemunculannya yang agak terlambat membuat banyak aplikasi belum tersedia.Baterai boros diversi awal.

2. Apple iOS

iOS merupakan versi mobile dari Mac OS X. Sistem operasi yang menggebrak dunia ponsel melalui iPhone, iPod hingga digunakan juga diperangkat tablet mereka; iPad. Sebuah sistem operasi yang memanusiakan manusia melalui fitur touch screen yang realistis.

Kelebihan iOS:

Integrasi yang mudah dengan perangkat Apple lainnya.Cepat dan efisien karena sistem yang mereka buat khusus untuk produk mereka sendiri.Memory yang digunakan tidak besar.

Kekurangan iOS:

Tidak ada media eksternal.Memerlukan program tambahan (iTunes) untuk bisa mentransfer data (mp3, video dll).Harga mahal.

3. Google Android

Saat ini, boleh dibilang inilah sistem operasi terpopuler untuk smartphone. Dengan duit 500ribu pun Anda sudah bisa mendapatkan ponsel pintar bersistem operasi Android. Android juga mengusung open source karena pada dasarnya basisnya dari Linux.

1.)Komputer adalah alat yang dipakai untuk mengolah data menurut prosedur yang telah dirumuskan. Kata computer pada awalnya dipergunakan untuk menggambarkan orang yang perkerjaannya melakukan perhitungan aritmetika, dengan atau tanpa alat bantu, tetapi arti kata ini kemudian dipindahkan kepada mesin itu sendiri.

2.)Bagaimana cara mematikan PC melalui Power User Menu? Pertama, klik kanan pada menu Start yang berada di sudut kiri bawah layar atau tekan tombol Win + X pada keyboard Anda untuk membuka Power User Menu. Lalu, pilih Shut down or sign out > Shutdown seperti yang ditunjuk oleh panah kuning pada gambar dibawah ini.

3.)Bisa dikatakan bahwa sistem operasi adalah Resource Manager dalam perangkat komputer. Maksudnya, sistem operasi berfungsi mengontrol fungsi perangkat keras seperti memori, CPU, harddisk, dan perangkat keras lainnya dan juga mengatur fungsi program software agar terhubung dengan perangkat keras tersebut.

4.)Warm booting menggunakan tombol start. a. Klik start. b. Klik Turn Off Computer. ...

Warm booting menggunakan keyboard (cara ini digunakan jika komputer tidak merespon perintah Anda) a. Tekan tombol Ctrl + Alt + Delete secara bersamaan. b. Ulangi langkah a jika perlu.

Warm booting menggunakan tombol reset.
