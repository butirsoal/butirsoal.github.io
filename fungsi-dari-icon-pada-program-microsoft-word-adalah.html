7. Sebutkan 7 icon pada Microsoft Word beserta fungsinya


Jawaban:

7 macam icon pada Microsoft Word dan fungsinya:

1. Icon B (Bold): digunakan untuk menebalkan tulisan yang dipilih. Dapat pula dilakukan dengan shortcur Ctrl + B

2. Icon I (Italic): digunakan untuk memiringkan tulisan yang dipilih ke arah kanan dan dapat pula dilakukan dengan shortcut Ctrl + I.

3. Icon U (Underline): digunakan untuk menggarisbawahi tulisan yang dpilih dan dapat pula dilakukan dengan shortcut Ctrl + U.

4. Icon dua kertas berjajar (Copy): digunakan untuk menyalin objek di dalam lembar kerja pengguna dapat berupa tabel, gambar maupun tulisan ke dalam clipboard.

5. Icon kertas dengan alas clipboard (Paste): digunakan untuk menempelkan objek yang semula disalin atau berada pada clipboard ke dalam lembar kerja

6. Icon gunting (Cut): digunakan untuk memindahkan objek maupun tulisan dalam lembar kerja pengguna ke clipboard.

7. Icon kertas putih (New): digunakan untuk membuka dokumen Microsoft Word yang baru tanpa menutup dokumen sebelumnya. Hal ini dapat dilakukan dengan bantuan shortcut Ctrl + N.

Penjelasan:

Microsoft Word atau Microsoft Office Word atau Word adalah perangkat lunak pengolah kata andalan Microsoft. Pertama di terbitkan pada 1983 dengan nama Multi-Tool Word untuk Xenix, versi-versi lain kemudian dikembangkan untuk berbagai sistem operasi, misalnya DOS, Apple Macintosh, SCO UNIX, OS/2, dan Microsoft Windows.

Jawaban:

1. Membuat surat menyurat

2.Membuat dokumen

3.Membuat dokumen dengan berbagai variasi

4.Menambahkan gambar pada dokumen

5.Membuat tabel

Penjelasan:

1. Membuat surat menyurat

Fungsi Microsoft Word yang pertama adalah membuat surat menyurat. Pada awalnya untuk membuat surat, kamu perlu menulisnya di selembar kerta, atau mengetik dengan menggunakan mesin ketik. Namun setelah muncul Microsoft Word, untuk membuat sebuah tulisan atau surat menyurat, kamu akan lebih dimudahkan.

2.Membuat dokumen

Selain fungsi Microsoft Word sebagai program untuk membuat surat, fungsi lain dari Microsoft Word adalah membuat dokumen. Segala jenis dokumen formal, semi formal, atau tidak formal, dapat dibuat menggunakan program ini. Jika ingin di cetak, kamu hanya perlu menghubungkan komputer atau laptop ke perangkat lain seperti printer.

3. Membuat dokumen dengan berbagai variasi

Jika dulu untuk membuat sebuah dokumen hanya bisa menggunakan mesin ketik atau menulisnya, sehingga variasi atau tampilan dari dokumen tersebut akan biasa saja. Dengan hadirnya Microsoft Word, kamu bisa membuat dokumen yang lebih menarik.

Seperti mengganti fontnya, membuat tulisannya menjadi lebih tebal, lebih kecil, atau lebih besar. Banyak sekali tools yang dapat kamu lakukan menggunakan Microsoft Word. Sehingga membuat dokumenmu menjadi lebih menarik dan bervariasi lagi.

4. Menambahkan gambar pada dokumen

Ini adalah fungsi Microsoft Word yang mempermudah kita dalam melakukan menyisipkan gambar pada sebuah dokumen. Jika dulu, gambar hanya bisa ditempel menggunakan lem, kini gambar bisa langsung disisipkan ke dokumen tanpa harus menempelkannya dengan lem. Kamu juga bisa mengatur gambar agar sesuai dengan apa yang kamu inginkan.

5.Membuat tabel

Fungsi Microsoft Word selanjutnya adalah membuat tabel. Kini membuat tabel dalam sebuah dokumen tidak perlu lagi harus menggambar menggunakan penggaris. Salah satu tools yang ada di Microsoft Word adalah menyisipkan tabel yang dapat kamu ubah-ubah bentuk dan jumlahnya, sangat membantu bukan.

18. apa fungsi icon B / U pada Microsoft word


Fungsi icon B pada Microsoft Word adalah untuk mengatur atau membuat tulisan menjadi tebal (bold)  Fungsi icon U pada Microsoft Word adalah untuk mengatur tulisan bergaris bawah (underline)

Pembahasan:

Microsoft Word atau MS-WORD (sering disebut Word) adalah program pengolah kata dengan tampilan grafis yang dapat membuat dokumen dengan hasil cetak sesuai dengan yang diketik oleh pengguna. Perangkat lunak ini dibuat perusahaan komputer Microsoft. Fungsi utama dari perangkat lunak pengolah kata, adalah untuk membuat dan mengedit dokumen.

Dalam pengaturan teks atau tulisan di Microsoft word, ada tiga icon utama yaitu:

Icon B (bold) untuk mengatur apakah tulisan tebal atau tidakIcon I (Italic) untuk mengatur apakah tulisan miring atau tidakIcon U (underline) untuk mengatur apakah tulisan bergaris bawah atau tidak

Dengan meng-klik tiga icon ini maka bentuk tulisan dapat disesuaikan.
