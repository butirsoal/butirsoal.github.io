11. pengertian sort dan filter


Pengertian sort adalah mengurutkan suatu kumpulan objek menurut urutannya secara ascending maupun descending. Sedangkan, pengertian filter adalah menyaring sekumpulan objek dengan suatu aturan tertentu sehingga hanya objek yang tersaring yang dapat ditampilkan. Baik sort dan filter banyak digunakan ketika kita akan mengelola sekumpulan data.

Pembahasan

Sort

Secara umum, sort merupakan metode untuk mengurutkan sekumpulan data. Fitur sorting ini umumnya digunakan pada software pengolah angka seperti Microsoft Excel dan Google Sheet. Penggunaan sorting dapat dilakukan untuk mengurutkan angka maupun alphabet. Nah, kita menggunakan sort hanya jika menginginkan bentuk data yang terurut dan nilainya dapat diurutkan. Namun, jika ingin menampilkan sekumpulan data secara spesifik, sorting tidak dapat digunakan. Berikut contoh bentuk sorting:

Diketahui data berupa: [3, 4, 6, 2, 1, 9, 12].

Jika data ini dilakukan sorting secara ascending akan menjadi [1, 2, 3, 4, 6, 9, 12]. Jika dilakukan secara descending akan menjadi [12, 9, 6, 4, 3, 2, 1].

Filter

Filter merupakan metode untuk menyaring sekumpulan data dengan kondisi tertentu. Fitur ini sama halnya dengan sorting, banyak digunakan pada software pengolah angka. Penggunaan filter tidak terbatas pada angka dan alphabet saja, tetapi lebih luas dan menyesuaikan kondisi seperti, warna tertentu, teks dengan panjang tertentu, data tidak kosong, dan sebagainya. Berikut contoh bentuk filtering:

Diketahui data berupa nama dan warna kesukaan [ Nama - Warna]:

[Abi-Merah, Candra-Kuning, Budi-Biru, Daniel-Merah, Ami-Kuning, Joe-Kuning, Richard- Hijau]

Jika kita melakukan filtering data nama orang dan warna kesukaan yang warna kesukaannya adalah kuning, maka, hasil filtering akan menampilkan: [Candra - Kuning, Ami - Kuning, Joe - Kuning].Jika kita melakukan filtering data nama orang yang diawali huruf A, maka, hasil filtering menjadi: [Abi, Ami]

14. Cara menggunakan fitur filter yaitu...a. Home-->Sort & Filter-->Filter b. Home-->Sort & Filterc. Insert-->Sort & Filterd. Insert-->Sort & Filter-->Filter​


Jawaban:

a. Home-->Sort & Filter-->Filter

Penjelasan:

Untuk menggunakan fitur filter dalam Microsoft Excel, Anda dapat mengikuti langkah-langkah berikut:

Buka lembar kerja Excel yang ingin Anda filter.

1. Pergi ke tab "Home" di ribbon Excel.

2. Di grup "Sort & Filter", klik tombol "Filter". Ini akan menampilkan panah drop-down di baris header setiap kolom.

3. Klik panah drop-down di salah satu kolom yang ingin Anda filter.

4. Anda dapat memilih opsi filter seperti menampilkan hanya nilai-nilai unik, filter berdasarkan kriteria tertentu, atau mengatur filter kustom.

5. Setelah Anda memilih kriteria filter, lembar kerja Excel akan diperbarui dan hanya menampilkan data yang sesuai dengan kriteria yang Anda tentukan.

Jadi, langkah-langkahnya adalah Home-->Sort & Filter-->Filter.

39. jelaskan fungsi dari menu sort asscending dan sort descending


Sort Ascending bertujuan untuk mengurutkan (sortir) data dengan urutan naik mulai dari nilai terkecil hingga terbesar.

Sort Descending kebalikannya, bertujuan untuk mengurutkan (sortir) data dengan urutan turun mulai dari nilai terbesar hingga terkecil.
