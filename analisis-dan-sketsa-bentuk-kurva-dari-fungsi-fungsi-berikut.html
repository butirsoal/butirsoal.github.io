23. tentukan fungsi eksponen dari sketsa grafik berikut ​


Tentukan fungsi eksponen dari sketsa grafik berikut

a. f(x) = 4ˣ b. f(x) = 6⁻ˣ

Suatu grafik fungsi eksponen jika memotong sumbu y di titik (0, 1) maka bentuk fungsinya adalah

f(x) = aˣ, dengan a > 0 dan a ≠ 1

Pembahasan

Grafik pada gambar a

Memotong sumbu y di (0, 1) dan melalui titik (2, 16)

Karena memotong sumbu y di (0, 1) maka bentuk fungsinya adalah  

f(x) = aˣ, dengan a > 0 dan a ≠ 1

Melalui titik (2, 16) artinya

f(2) = 16

a² = 16

a² = 4²

a = 4

Jadi fungsi eksponen untuk grafik pada gambar a adalah

f(x) = 4ˣ

Grafik pada gambar b

Memotong sumbu y di (0, 1) dan melalui titik (–2, 36)

Karena memotong sumbu y di (0, 1) maka bentuk fungsinya adalah  

f(x) = aˣ, dengan a > 0 dan a ≠ 1

Melalui titik (–2, 36) artinya

f(–2) = 36

a⁻² = 36

[tex]\frac{1}{a^{2}} = \frac{36}{1}[/tex]

[tex]\frac{a^{2}}{1} = \frac{1}{36}[/tex]

a² = [tex]\left(\frac{1}{6} \right)^{2}[/tex]

a = [tex]\frac{1}{6}[/tex]

Jadi fungsi eksponen untuk grafik pada gambar b adalah

[tex]f(x) = \left(\frac{1}{6} \right)^{x}[/tex]

[tex]f(x) = (6^{-1})^{x}[/tex]

[tex]f(x) = 6^{-x}[/tex]

26. sketsa dan analisis limit fungsi di x = -1 dan x = 1


Sketsa dan analisis limit fungsi di x = -1 dan x = 1 untuk fungsi berikut ini adalah :

a. lim {x → 1} f(x)=tidak ada

lim {x → -1} f(x)=tidak ada

b. lim {x → 1} f(x)=4

lim {x → -1} f(x)=0

c. lim {x → 1} f(x)=2

lim {x → -1} f(x)=4

d. lim {x → 1} f(x)=3

lim {x → -1} f(x)=tidak ada

e. lim {x → 1} f(x)=tidak ada

lim {x → -1} f(x)=tidak ada

Sketsa untuk masing-masing grafik bisa di lihat pada gambar di lampiran.

PEMBAHASAN

Limit suatu fungsi di katakan ada jika limit kiri dan kanan dari fungsinya bernilai sama. Artinya dari kiri ataupun kanan grafik nya akan mendekati sebuah nilai tertentu.

Oke marilah kita menggunakan prinsip ini untuk menyelesaikan soal berikut.

Mula-mula kita akan sketsa terlebih dahulu masing-masing grafik fungsi yang ditanyakan seperti pada gambar di lampiran.

Soal a :

Perhatikan gambar A:

Pada titik x = 1 grafik terputus maka :

[tex]\lim_{x \to 1} f(x)=tidak~ada[/tex]

Pada titik x = -1 grafik terputus maka :

[tex]\lim_{x \to -1} f(x)=tidak~ada[/tex]

Soal b:

Perhatikan gambar B:

Pada titik x = 1 grafik tersambung di y = 4 maka :

[tex]\lim_{x \to 1} f(x)=4[/tex]

Pada titik x = -1 grafik tersambung di y = 0 maka :

[tex]\lim_{x \to -1} f(x)=0[/tex]

Soal c:

Perhatikan gambar C:

Pada titik x = 1 grafik tersambung di y = 2 maka :

[tex]\lim_{x \to 1} f(x)=2[/tex]

Pada titik x = -1 grafik tersambung di y = 4 maka :

[tex]\lim_{x \to -1} f(x)=4[/tex]

Soal d:

Perhatikan gambar D:

Pada titik x = 1 grafik tersambung di y = 3 maka :

[tex]\lim_{x \to 1} f(x)=3[/tex]

Pada titik x = -1 grafik terputus maka :

[tex]\lim_{x \to -1} f(x)=tidak~ada[/tex]

Soal e:

Perhatikan gambar E:

Pada titik x = 1 grafik terputus maka :

[tex]\lim_{x \to 1} f(x)=tidak~ada[/tex]

Pada titik x = -1 grafik terputus maka :

[tex]\lim_{x \to -1} f(x)=tidak~ada[/tex]
