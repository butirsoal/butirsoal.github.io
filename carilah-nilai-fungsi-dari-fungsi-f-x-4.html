22. 1)Sebuah fungsi dirumuskan f (x) = 2x. Jika bayangan f (x) adalah 64. Tentukan nilai x! Jawab: 2) fungsi f didefinisikan dengan rumus f(x) = x^2+4 jika f(a) = 40 carilah persamaan dalam a dan besar nilai a 3) fungsi f dengan rumus f(x) = x^2 + x tentukan himpunan bayangan dari daerah asal A {-2,-1,0,1,2,} dengan himpunan pasangan berurutan


1. Nilai x adalah 32.

2. Persamaan dalam a adalah a² - 36 = 0. Sedangkan nilai a adalah 6 atau -6.

3. Himpunan pasangan berurut dari banyangan dan daerah asal adalah {(-2, 2), (-1, 0), (0, 0), (1, 2), (2, 6)}

Simak pembahasan berikut.

Pembahasan

1. Diketahui:

Sebuah fungsi dirumuskan dengan

f(x) = 2x

f(x) = 64

Ditanya: Nilai x

Jawab:

f(x) = 2x

64 = 2x

x = 64/2

x = 32

Jadi nilai x adalah 32.

2. Diketahui:

fungsi f didefinisikan dengan

f(x) = x² + 4

f(a) = 40

Ditanya: persamaan dalam a dan besar a

Jawab:

a. Persamaan dalam a

f(x) = x² + 4, karena f(a) = 40, maka subtitusikan a kedalam fungsi f(x)

f(x) = x² + 4

f(a) = a² + 4

40 = a² + 4

a² + 4 - 40 = 0

a² - 36 = 0

Jadi persamaan dalam a adalah a² - 36 = 0

b. Nilai a

a² - 36 = 0

a² = 36

a = ± √36

a = 6

atau

a = -6

Jadi nilai a adalah 6 atau -6.

3. Diketahui:

Fungsi f dirumuskan dengan

f(x) = x² + x

daerah asal A = {-2, -1, 0, 1, 2}

Ditanya: Himpunan pasangan berurut dari banyangan dan daerah asal

Jawab:

Untuk menentukan bayangan, maka subtitusikan satu persatu nilai dari daerah asal ke dalam fungsi.

Untuk x = -2

f(x) = x² + x

f(-2) = (-2)² + (-2)

f(-2) = 4 - 2

f(-2) = 2

Maka pasangan berurutnya adalah (-2, 2)

Untuk x = -1

f(x) = x² + x

f(-1) = (-1)² + (-1)

f(-1) = 1 - 1

f(-1) = 0

Maka pasangan berurutnya adalah (-1, 0)

Untuk x = 0

f(x) = x² + x

f(0) = (0)² + 0

f(0) = 0 + 0

f(0) = 0

Maka pasangan berurutnya adalah (0, 0)

Untuk x = 1

f(x) = x² + x

f(1) = 1² + 1

f(1) = 1 + 1

f(1) = 2

Maka pasangan berurutnya adalah (1, 2)

Untuk x = 2

f(x) = x² + x

f(2) = 2² + 2

f(2) = 4 + 2

f(2) = 6

Maka pasangan berurutnya adalah (2, 6)

Jadi himpunan pasangan berurut dari banyangan dan daerah asal adalah {(-2, 2), (-1, 0), (0, 0), (1, 2), (2, 6)}
