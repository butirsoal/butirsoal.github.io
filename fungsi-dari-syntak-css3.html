9. Tentukan syntak fungsi-fungsi dibawah ini apakah fungsi menaik atau menurun 1. Y = -x^2 + 6x + -2 2. Y = x^2 - 4x + 8


Jawaban:

1) fungsi naik

2) fungsi turun

Penjelasan dengan langkah-langkah:

1) y' = -2x + 6 = 0

-2x = -6

2x = 6

x = 3 (fungsi naik)

2) y' = 2x -4 > 0

2x - 4 = 0

2x = - 4

x = - 2 (fungsi turun)

Fungsi naik, fungsi turun, dan fungsi diam (stasioner) merupakan kondisi dari turunan pertama suatu fungsi pada suatu interval tertentu. Kondisi yang dimaksud dapat berupa berikut.

Jika f'(x) bertanda positif, atau f'(x) > 0, maka kurva fungsi dalam keadaan naik (disebut fungsi naik).

Jika f(x) bertanda negatif, atau f'(x) < 0, maka kurva fungsi dalam keadaan turun (disebut fungsi turun).

Jika f(x) bertanda netral, atau f(x) = 0 , maka kurva fungsi dalam keadaan tidak turun dan tidak naik, istilahnya kita sebut sebagai stasioner (disebut juga fungsi diam).

17. bagaimana penulisan rumus syntak looping while ??​

while(kondisi)

Contoh PHP :

<?php

$i=1000;

while ($i <= 10)

{

echo "$i";

echo "Tidak akan tampil di browser";

}

?>

Syntax do-while :

do {

statement;

statement;

} while (condition);

Contoh C++ :

#include <iostream>

using namespace std;

int main()

{

int pilihan = 1;

do{

cout<<"Masukan nomer : ";cin>>pilihan;

} while(pilihan<=10);

return 0;

}
