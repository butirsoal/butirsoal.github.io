17. jarak antara depok dan bandung adalah 65 km, dengan skala 1:2.500.000, jarak depok bandung pada peta adalah​


Jawaban :  

2,6 cm

Penjelasan :   → Diketahui :     Jarak sebenarnya ( JS ) = 65 km  Skala ( S ) = 1 : 2.500.000

 

→ Ditanya :    Jarak Pada Peta ( JP ) = ???  

→ Penyelesaian :    

JP = S x JS  

=  1/2.500.000 x 65 km

= 1/2.500.000 x ( 65 x 100.000 )  cm    

=  1/2.500.000 x 6.500.000 cm

= 6.500.000/2.500.000

= 2,6 cm

Jadi,  jarak depok bandung pada peta adalah​ 2,6 cm
