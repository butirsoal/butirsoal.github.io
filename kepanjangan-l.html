7. Luas ABCD = panjang .... x panjang ....= p X ....= .... Satuan luasMaka luas persegi panjang ABCDL = panjang .... x panjang ....L = .... X ...​


Apabila diketahui persegi panjang ABCD sebagai berikut, maka perlu diketahui dulu RUMUSnya.

Rumus Luas Persegi Panjang adalah :

Luas = Panjang persegi  X  Lebar persegi

Maka, apabila tidak diketahui panjang dan lebar persegi. Cara mengetahui luas nya adalah :

L =  Panjang AB  X  Panjang BC

L =  AB  X  BC

L = (hasil dari perkalian itu) Satuan luas.

NOTE :

Kalian bisa menggunakan bagian panjang dan lebar manapun.  Karena persegi panjang mempunyai 2 sisi yang sama, yaitu sisi atas dengan sisi bawah, dan sisi kiri dengan sisi kanan.

9. Kuis ⁽¹⁰⁶⁾ Rumus panjang diagonal persegi panjang adalah? [a.] = √(p² - l²) [b.] = l√p [c.] = √(p² + l²) [d.] = √(p²l²) [e.] = p√l


Penjelasan dengan langkah-langkah:

*gambar dulu biar ga ngang ngong

Diagonal persegi panjang

D² = P²+l²

D = √(p²+l²)

maka opsi jawaban yang benar adalah C

Diagonal (d), panjang (p), dan lebar (l) persegi panjang membentuk segitiga siku-siku. Maka menggunakan pythagoras. Pada pythagoras, kuadrat sisi miring sama dengan jumlah kuadrat kedua bilangan lain.

Diagonal persegi panjang itu sisi miring, jadi:

d² = p² + l²

d = √(p² + l²)

Jawabannya (c).
