3. ajarkan tenses dong..


Present tense --> sekarang
Past tense --> sudah terjadi
Future tense --> akan terjadi

FORMULAS:
- Present tense --> S + Verb 1 + Object
- Past tense --> S + Verb 2 + Object
- Future Tense --> S + will + verb 1 + object
- Present perfect --> S + has/have + V3 + object
- Present continuous --> Subject + verb-ing + object
- Past perfect --> S + had + v3 + object
- Past continuous --> S + had been + verb-ing + object
- Future perfect --> S + will have + V3 + object
- Future continuous --> S + will have been + verb-ing + object

18. apakah toefl dan ielts itu?

IELTS singkatan dari International English Language Testing System, yaitu program tes untuk menguji kemahiran dalam berbahasa Inggris.IELTS biasanya digunakan untuk mengurus visa luar negeri, studi/sekolah ke negara-negara English-British (seperti Inggris dan Australia misalnya), pindah kewarganegaraan, dsb.
