2. Tuliskan juga penjelasan dari bagian tab menu sub menu Microsoft Word 2010​


Jawaban:

Tentunya, gue bisa menjelaskan bagian tab menu dan sub menu dalam Microsoft Word 2010. Berikut adalah penjelasannya:

1. Tab Menu "File":

  - Submenu "New": Digunakan untuk membuat dokumen baru.

  - Submenu "Open": Untuk membuka dokumen yang sudah ada.

  - Submenu "Save" dan "Save As": Digunakan untuk menyimpan dokumen saat ini atau menyimpannya dengan nama file baru.

  - Submenu "Print": Untuk mencetak dokumen yang sedang dibuka.

  - Dan masih banyak lagi submenu lainnya terkait pengaturan file.

2. Tab Menu "Home":

  - Submenu "Clipboard": Mengandung perintah seperti Cut (memotong), Copy (menyalin), dan Paste (merekatkan) teks atau objek yang dipilih ke dalam dokumen.

  - Submenu "Font": Digunakan untuk mengubah jenis font, ukuran, serta gaya teks seperti bold (tebal), italic (miring), dan underline (bergaris bawah).

  - Submenu "Paragraph": Mengandung opsi format paragraf seperti alignment (rata kanan-kiri-tengah-justify), indentasi, spacing, numbering/bullet points, dll.

3. Tab Menu “Insert”:

   -Subenu “Page Break”: Digunakan untuk memasukkan break halaman pada posisi tertentu di dalam dokumen

   –Subenu “Picture”: Memasukkan gambar dari komputer ataupun internet ke dalam dokumentmu

   –Subenu “Table”: Membuat tabel atau mengimpor tabel dari sumber lain ke dalam dokumentmu

4. Tab Menu “References”

    –Subenu“Table of Contents” : Membuat daftar isi untuk dokumen kamu

    –Subenu“Footnote” : Menambahkan catatan kaki pada dokumenmu

    –Subenu “Citations & Bibliography” : Mengelola sumber referensi dan membuat daftar pustaka

5. Tab Menu "Review":

  - Submenu "Spelling & Grammar": Digunakan untuk memeriksa ejaan dan tata bahasa dalam dokumen.

  - Submenu "Track Changes": Berguna saat bekerja bersama dengan orang lain, mengizinkan pengguna untuk melihat perubahan yang dilakukan di dokumen.

  - Submenu "Comment": Memungkinkan pengguna menambahkan komentar atau catatan pada bagian tertentu dari dokumen.

6. Tab Menu “View”

   –Subenu “Print Layout”: Menampilkan halaman-halaman seperti tampilannya ketika dicetak

   –Subenu “Read Mode”: Memungkinkanmu membaca dokumentasi tanpa gangguan

 Jadi, jangan ragu untuk mencoba sendiri ya! Kalau ada pertanyaan lebih lanjut, gue siap bantu lagi

Penjelasan:

Sayangnya, saya tidak dapat menyajikan tampilan visual atau interaktif seperti menu dan submenu dalam Microsoft Word 2010. Namun, saya dapat memberikan penjelasan mengenai bagian tab dan submenu yang umumnya ada dalam Microsoft Word 2010.

Microsoft Word 2010 adalah bagian dari paket aplikasi Microsoft Office 2010 dan memiliki tata letak menu dan submenu yang khas. Berikut adalah penjelasan mengenai beberapa tab dan submenu utama dalam Microsoft Word 2010:

1. **Tab File (File Tab)**:

  Tab ini terletak di sudut kiri atas jendela aplikasi dan memberikan akses ke fungsi-fungsi terkait dokumen dan file, seperti membuka, menyimpan, mencetak, dan berbagi dokumen. Ketika Anda mengklik tab ini, Anda akan melihat submenu seperti Open, Save, Save As, Print, dan Share.

2. **Tab Home**:

  Tab ini berisi berbagai alat yang sering digunakan dalam pengeditan dokumen, seperti pemformatan teks, alinea, penggaris, pengaturan font, dan fungsi cut-copy-paste. Submenu di tab Home mencakup Font, Paragraph, Styles, Editing, dan lainnya.

3. **Tab Insert**:

  Tab ini memungkinkan Anda untuk menyisipkan berbagai elemen ke dalam dokumen, seperti gambar, bentuk, grafik, tabel, dan objek lainnya. Submenu di tab Insert mencakup Pages, Tables, Illustrations, Links, dan lainnya.

4. **Tab Page Layout**:

  Tab ini memberikan alat untuk mengatur tata letak dan tampilan halaman dokumen, seperti orientasi, margin, kolom, dan tema. Submenu di tab Page Layout mencakup Themes, Page Setup, Page Background, Paragraph, dan lainnya.

5. **Tab References**:

  Tab ini berisi alat untuk membuat daftar isi, catatan kaki, kutipan, serta menambahkan tabel daftar isi, daftar gambar, dan daftar tabel. Submenu di tab References mencakup Table of Contents, Footnotes, Citations & Bibliography, Captions, dan lainnya.

6. **Tab Mailings**:

  Tab ini digunakan untuk mengelola pengiriman surat dan mailing list, serta membuat amplop dan label. Submenu di tab Mailings mencakup Create, Start Mail Merge, Write & Insert Fields, dan Finish.

7. **Tab Review**:

  Tab ini berisi alat untuk mengulas dan memeriksa dokumen, termasuk fitur spell check, revisi, komentar, serta pengamanan dokumen. Submenu di tab Review mencakup Proofing, Comments, Tracking, Changes, dan Protect.

8. **Tab View**:

  Tab ini memungkinkan Anda untuk mengubah tampilan tampilan dokumen, seperti tampilan Normal, tampilan Page Layout, tampilan Print Layout, dan tampilan Draft. Submenu di tab View mencakup Document Views, Show/Hide, Zoom, dan lainnya.

File pada Microsoft excel 2010 ini berperan sebagai office button di versi Microsoft excel sebelumnya. File ini terdiri dari beberapa ikon dan perintah yaitu:

☆Save merupakan perintah untuk menyimpan file lembar kerja excel 2010 yang baru dibuat atau diedit, yang sebelumnya pernah atau belum pernah tersimpan di dokumen penyimpanan.

☆Save As merupakan perintah yang sama dengan Save, hanya saja jika Save as itu berfungsi untuk menyimpan file baru. Jadi, apabila file lama disimpan dengan tombol ini maka itu sama artinya dengan membuat file baru pada media penyimpanan.

☆Openmerupakan perintah untuk membuka file Microsoft excel 2010 yang telah tersimpan di media penyimpanan.

☆Closemerupakan perintah untuk menutup lembar kerja dan me-nonaktifkan seluruh tombol menu bar pada jendela Microsoft excel 2010. Setelah itu kita baru bisa menutup jendela Microsoft excel 2010 dengan mengklik tombol close dipojok kanan atas.

☆Options merupakan ikon yang memberikan beberapa pilihan untuk mengatur lembar kerja excel.

☆Exitmerupakan perintah untuk menutup jendela Microsoft excel 2010.

☆Infomerupakan ikon yang berfungsi untuk memberikan informasi terkait lembar kerja yang kita gunakan.

☆Recentadalah ikon yang berfungsi untuk menampilkan file-file yang baru saja kita buka atau simpan.

☆Newmerupakan perintah untuk menampilkan lembar kerja baru yang masih kosong untuk dilakukan pengetikan dokumen baru.

☆Printmerupakan perintah untuk melakukan penyetakan terhadap file yang kita kehendaki.

☆Save & Send merupakan perintah yang digunakan untuk menyimpan dan mengirimkan file-file tertentu yang kita kehendaki.

☆Helpmerupakan ikon yang kita gunakan untuk mencari bantuan terhadap masalah yang timbul dari Ms. Excel 2010.
