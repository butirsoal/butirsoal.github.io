5. 1.jelaskan fungsi duration pada animation Pane2.jelaskan mengenai start on clik pada animation Pane​


a.Group Preview

Pada toolgroup ini hanya ada satu tombol yaitu Preview  Animations, digunakan untuk untuk melihat hasil dari animasi yang digunakan pada teks atau gambar.

b.Group Animation

Toolgroup Animation berisikan 2 Icon, yaitu :

Ø  Animation, memiliki fungsi untuk menyediakan pilihan-pilihan efek animasi.

Ø  Effect option, memiliki fungsi untuk menambahkan variasi modifikasi pada efek animasi yang sudah dipilih di slide.

c.Group Advanced Animation

Toolgroup Advanced Animation berisikan 4 Icon, yaitu :

Ø  Add animation, berfungsi untuk menambahkan lagi efek animasi ke objek yang telah diberi efek animasi sebelumnya.

Ø  Animation Pane, memiliki fungsi untuk menampilkan panel (Jendela) penerapan efek animasi.

Ø  Trigger, berfungsi untuk menentukan objek yang telah diberi animasi yang mana yang akan tampil awal daripada objek-objek lain nya pada suatu slide.

Ø  Animation Painter, memiliki fungsi untuk menyalin efek animasi yang telah di terapkan pada objek yang di pilih kepada objek lain dalam presentasi.

d.Group Timing

Toolgroup Timing memiliki 5 icon, yaitu :

Ø  Animation Timing, berfungsi untuk menentukan kapan mulai aktif atau jalan animasi yang telah di terapkan pada objek.

Ø  Animation Duration, berfungsi untuk menentukan lamanya proses aktif efek animasi yang diterapkan pada objek yang dipilih.

Ø  Animation Delay, berfungsi untuk menentukan jeda waktu (Waktu Tunda) sebelum suatu animasi diaktifkan.

Ø  Move Earlier, memiliki fungsi untuk menyusun urutan animasi objek yang urutan akhir menjadi urutan yang awal.

Ø  Maove Later, memiliki fungsi untuk menyusun urutan tampil suatu animasi objek yang urutan awal menjadi urutan akhir.

Implementasi Toolsgroup pada Tab Animation

Pada Tab Animation kalian bisa menggunakan Fungsi dari toolgroup yang sudah kami jelaskan diatas,  Tab Aninmation memiliki fungsi yang kurang lebih sama dengan Tab Transitions, tetapi untuk Tab Animation ini lebih difungsikan untuk membuat animasi untuk teks atau gambar dikarenakan dalam toolgroup Animation terdapat 3 animasi yang berbeda yaitu, Enterance (Animasi Masuk), Emphasis (Animasi Tetap) dan Exit (Animasi Keluar).

             pada Tab ini tentunya akan membuat presentasi kamu semakin menarik untuk diamati oleh audience.

18. Bagian jendela berikut yang merupakan tempat susunan blok kode pogram adalah…. * Stage pane Tab customes Block palette Script area


Jawaban:

D. Script area

Penjelasan:

Script area adalah area di jendela pemrograman visual di mana pengguna dapat menyusun blok-blok kode secara berurutan untuk membuat program atau skrip. Ini adalah tempat di mana logika program diatur dengan menggabungkan berbagai blok kode.

Di dalam Script area, pengguna dapat mengambil blok-blok kode dari Block palette dan menaruhnya secara berurutan untuk membentuk alur eksekusi program. Blok-blok kode dapat disusun, diatur, dan dihubungkan satu sama lain menggunakan ikon-ikon yang ada di blok tersebut. Dengan cara ini, pengguna dapat membentuk instruksi atau alur program dengan cara yang intuitif dan visual.

Setelah blok-blok kode ditempatkan di Script area, mereka dapat dieksekusi berurutan sesuai dengan alur yang telah dibangun. Ini memungkinkan pengguna untuk melihat dan mengontrol jalannya program dengan mudah.

Jadi, Script area adalah bagian jendela yang merupakan tempat susunan blok kode program untuk membuat program atau skrip dalam lingkungan pemrograman visual.

23. 1.jelaskan mengapa pemrograman visual lebih mudah dilakukan dari pada pemrograman biasa 2.jelaskan fungsi dari.a.block paletteb.script areac.staged.stage panee.sprite pane3.sebutkan dan jelaskan fungsi dari 8 jenis blok kode.​​


Jawaban:

1. Pemrograman visual memungkinkan pengembang untuk menulis program dengan memanipulasi elemen grafis. Saya kira menggunakan pemrograman visual harus meningkatkan kualitas kode dan mengurangi bug pemrograman.

2. a. Block Palette

Untuk memprogram sprite. Blocks Palette Scripts adalah Blok yang berisi fungsi atau kode perintah untuk memprogram suatu sprite. Script pada program Scratch divisualisasikan dalam bentuk blok-blok program seperti memasang sebuah puzzle.

b. Script Area

Untuk tempat dimana anda dapat menerapkan script atau kode terhadap Sprite yang anda tampilkan.

c. Stage

Stage/Backdrop adalah gambar latar belakang pada canvas. backdrop bisa di ganti di bagian “Stage” yang terletak sebelah kanan dari tempat sprite.

d. Stage Pane

Untuk menampilkan daftar backdrop(latar)yang digunakan di stage.

e. Sprite Pane

Untuk area antarmuka pengguna grafis dari program Scratch di mana semua sprite yang ada dalam proyek dapat dengan mudah diakses untuk memodifikasi atau memeriksa.

3. a. Unipolar Line Coding adalah kode yang menggunakan hanya satu non-zero dan satu zero level tegangan yaitu untuk logika 0 memiliki level zero dan untuk logika 1 memiliki level non-zero.

b. Polar line coding adalah kode dalam komunikasi data yang menggunakan dua buah level tegangan untuk non-zero guna merepresentasikan kedua level data, yaitu satu positif dan satu negatif.

c. Bipolar line coding adalah kode yang menggunakan dua level tegangan yaitu non-zero dan zero guna menunjukkan level dua jenis data, yaitu untuk logika 0 ditunjukkan dengan level nol, untuk logika 1 ditunjukkan dengan pergantian level tegangan positif dan negatif.

d. 2B1Q coding adalah pengkodean yang dilakukan dengan melakukan pengkodean 2 (dua) biner untuk dijadikan 1 (satu) kuarter. Pola data yang terdiri dari 2 bit dikodekan menjadi sebuah elemen sinyal yang merupakan bagian dari sinyal berlevel empat.

e. Block coding, kode jenis ini beroperasi pada sebuah formasi stream bit informasi. Berikut adalah beberapa hal terkait dengan kode blok yang beroperasi berdasarkan formasi blok bit informasi :

➡Bit redundan ditambahkan ke setiap blok informasi, hal ini dilakukan untuk memberikan kepastian sinkronisasi dan pendeteksian kesalahan.

➡Setiap empat bit data dikodekan menjadi kode 5-bit.

Kode 5-bit normalnya digunakan untuk penggunaan blok invers NRZ.

➡Pemilihan kode 5-bit seperti halnya setiap kode berisi tidak lebih satu bit 0 sebagai bit awal dan tidak ada lagi lebih dari dua buah logika 0.

f. ASCII Coding Kode ini merupakan salah satu sistem sandi dalam komunikasi data standar Amerika yang digunakan untuk menunjuk sebuah karakter yang diberi nama American Standard Code for Information Interchange (ASCII).

g. Data block merupakan pengkodean yang digunakan untuk pengiriman data secara blok yang dilengkapi dengan paritas ganjil atau paritas genap. Data block merupakan cara pengujian lebih baik karena satu blok data akan disertai dengan paritas yang diletakkan pada akhir blok data.

h. Hamming Coding dilakukan mengingat kesalahan data atau kerusakan data yang diterima oleh terminal penerima dalam sistem komunikasi data sering terjadi. Hal yang mendasar sebagai penyebab adalah adanya interferensi sinyal luar yang masuk ke dalam jalur komunikasi, koneksi kawat penghubung, terminal, konektor pada layer terendah yang kurang baik.
