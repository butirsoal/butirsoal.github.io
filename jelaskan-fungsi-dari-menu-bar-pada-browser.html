1. jelaskan fungsi dari menu bar pada browser


menu bar berfungsi untuk mengatur file dokumen seperti menyimpan, membuka, mengatur kertas dokumen, dan sebagainya
Fungsi menu-menu pada menu bar ::

Menu File :

· New teridiri dari submenu Window Message, Post, Contact, dan Internet Call.

· Open untuk membuka alamat baru.

· Edit with microsoft frontpage untuk mengedit tampilan web melalui frontpage.

· Save untuk menyimpan web

· Save as untuk menyimpan web dengan nama dan format sesuai keinginan kita.

· Page setup untuk pengaturan halaman web

· Print Preview untuk melihat tampilan web sebelum dicetak.

· Print untuk mencetak tampilan web.

· Send terdiri dari submenu send page by email, link by email, dan shortcut to desktop.

· Properties untuk melihat properti yang dimiliki oleh web browser.

· Work offline untuk mengaktifkan modus off akses internet.

· Close untuk keluar dari aplikasi web browser.

Menu Edit :

· Cut untuk memotong teks yang terdapat dalam web.

· Copy untuk mengopy teks atau gambar yang terdapat dalam web.

· Paste untuk menempatkan teks atau gambar yang dipotong dan dikopi.

· Select all untuk memblok web yang ada di web browser.

· Find(on this page) untuk mencari teks dalam halaman web.

Menu View :

· Toolbars untuk menampilkan atau menyembunyikan toolbar ikon.

· Status bar untuk menampilkan status bar

· Explorer bar untuk menampilkan explorer bar.

· Go to untuk menuju ke halaman tertentu.

· Stop untuk menghentikan proses akses internet.

· Refresh uyntuk memperbarui kembali proses akses.

· Text size untuk mengubah ukuran teks.

· Encoding untuk menampilkan encoding teks.

· Source untuk menampilkan halaman web menjadi source code.

· Privacy Report untuk menampilkan laporan secara pribadi.

· Script debugger untuk menampilkan pengecekan script HTML.

· Full screen untuk menampilkan halaman link secara penum dalam monitor.

Menu Favorites :

· Add and favorites untuk menyimpan alamat suatu web atau homepage yang kita inginkan ke dalam folder favorites untuk diakses kembali pada waktu lain

· Organize Favorites untuk mengatur alamat web atau homepage yang terdapat dalam folder favorites agar lebih rapih dan mudah mengaksesnya kembali

· Item lain, selain kedua submenu diatas bagian ini berisi dftar alamat yang pernah kita simpan dalam folder favorites. Jumlahnya tergantung dari alamat yang pernah disimpan

Menu Tools :

· Mail and news sebagai menu bantu dalam proses email.

· Synchronize untuk melakukan sinkronisasi website.

· Windows update untuk melakukan update terhadap sisterm operasi windows.

· Show related links untuk menampilkan link yang terhubung

· Internet options untuk pengaturan internet.

Menu Help :

· Contentes and index sebagai isi menu pertolongan

· Tip of the day untuk memunculkan pesan atau saran setiap hari

· For netscarpe users sebagai bantuan untuk pengguna yang menggunakan web browser netscape.

· Online Support untuk melakukan online yang mendukung.

· Send feedback untuk mengirim umpan balik.

· About IE untuk mengetahui informasi tentang IE.
