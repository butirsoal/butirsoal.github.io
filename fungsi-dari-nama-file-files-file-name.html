1. Align Left (CTRL + ALT + L): Sejajarkan tepi kiri dua atau lebih markup ke tepi kiri markup pertama yang dipilih.

2. Sejajarkan Atas (CTRL + ALT + T): Meratakan tepi atas dua atau lebih markup ke tepi atas markup pertama yang dipilih.

3. Rata Kanan (CTRL + ALT + R): Menyelaraskan tepi kanan dari dua atau lebih markup ke tepi kanan dari markup pertama yang dipilih.

4. Sejajarkan Bawah (CTRL + ALT + B): Sejajarkan tepi bawah dua atau lebih markup ke tepi bawah markup pertama yang dipilih.

5. Align Center (CTRL + ALT + E): Sejajarkan pusat vertikal dari dua atau lebih markup ke tengah vertikal markup pertama yang dipilih.

6. Sejajarkan Tengah (CTRL + ALT + M): Sejajarkan tengah horizontal dari dua atau lebih markup ke tengah horizontal markup pertama yang dipilih.

7. Tengah dalam Dokumen: Sejajarkan pusat dua atau lebih markup ke tengah halaman.

8. Align Width: Membuat dua atau lebih markup dengan lebar yang sama dari markup pertama yang dipilih.

9. Align Height: Membuat dua atau lebih markup sama tingginya dengan markup pertama yang dipilih.

10. Align Size: Membuat dua atau lebih markup dengan tinggi dan lebar yang sama dari markup pertama yang dipilih.

11. Distribusikan Secara Horizontal: Mendistribusikan tiga atau lebih markup di sepanjang bidang horizontal.

12. Distribusikan Secara Vertikal: Mendistribusikan tiga atau lebih markup di sepanjang bidang vertikal mereka.

13. Flip Horizontal (CTRL + ALT + H): Membalik satu atau lebih markup secara horizontal.

14. Flip Vertical (CTRL + ALT + V): Membalik satu atau lebih markup secara vertikal.

33. Fungsi Viewing multi FilesYang dapat menunjukkanbeberapa file secara bertumpuk adalah​


Fungsi Viewing multi files yang dapat menunjukkan beberapa file secara bertumpuk adalah​ View side by side.

Pembahasan  

Fungsi Viewing multi files yang kamu tanyakan berkaitan dengan perangkat lunak pengolah angka Excel. Fungsi ini merupakan fitur yang dapat membantu kita untuk membuka beberapa file Excel di saat yang sama dan menampilkannya secara berdampingan atau side by side.

Sebagai perangkat pengolah angka yang mumpuni keluaran Microsoft, Excel dapat dipakai untuk mengolah data dalam jumlah besar. Untuk itu, banyak perusahaan menaruh minat khususnya pada aplikasi ini dan menggunakannya.

Salah satu kebutuhan utama perusahaan dalam mengolah datanya adalah mengonfirmasi keakuratan data yang disimpan agar tidak menimbulkan kekeliruan lebih lanjut dan memengaruhi kredibilitas data secara keseluruhan. Untuk melakukan proses konfirmasi tersebut, data-data yang disimpan perlu dibandingkan. Kadangkala, data-data tersebut tersimpan dalam file Excel yang berbeda. Di sinilah fungsi View side by side berperan untuk membandingkan dua tabel data dalam dua file Excel yang berbeda. Bayangkan saja, tanpa fungsi ini kita lantas harus bolak-balik membuka satu file dan yang lainnya. Cukup memusingkan kan?  

Berikut adalah cara mengoperasikan fitur view side by side:

Klik tombol view di deretan fitur window Klik tombol side by side Ketika tombol side by side dipilih, tombol synchronous scrolling akan otomatis aktif. Fitur ini berfungsi agar kedua file bergerak secara identik ketika digeser. Kita dapat memilih scrolling secara vertikal dan horizontal  

 

Perlu diingat bahwa view side by side digunakan untuk membandingkan dua tabel dalam dua file yang berbeda. Lalu, apakah fungsi tersebut dapat digunakan untuk dua tabel yang berbeda dalam satu file yang sama? Jawabannya adalah tidak. Namun, terdapat fitur lain untuk permasalahan ini, yakni fitur split. Fitur split berfungsi untuk membagi tampilan dokumen kita menjadi beberapa bagian. Jika kita ingin menggeser satu bagian saja, bagian-bagian lain dalam file tersebut tidak ikut tergeser. Dengan fitur ini, kita jadi tidak perlu menyembunyikan (hide/unhide) kolom atau baris tertentu secara berulang-ulang.
