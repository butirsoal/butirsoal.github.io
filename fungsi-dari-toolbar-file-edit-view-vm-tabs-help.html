1. Apa fungsi menu file, edit, view, tools, favorite, help?


fungsi menu 
file :sebuah menu yang berisi hal yang berhubungan dengan penanganan file ex :new,open,edit, dll
edit : berfungsi untuk mengedit atau memanipulasi data dan informasi pada halaman web yang telah aktif
view : berfungsi untuk memanipulasi tampilan pada monitor
favorites :berfungsi untuk menyimpan alamat suatu homepage yang kita inginkan
tools : terdiri atas submenu yang berfungsi untuk mengakses outlook express dan melalukan sikronsasi
help : berfungsi sebagai bantuan manual

35. Sebutkan Kegunaan dari menu file,edit,view,favorites,tools dan help


Menu File di gunakan untuk mengoprasikan hal-hal yang sifatnya fatal dan menyangkut semua hasil kerja pada lembar atau jendela Internet.

Menu Edit secra umum digunakan untuk mengadakan perubahan halaman website, baik yang link atau yang tidak ada pada halaman editor.

Menu View berguna untuk menampilkan atau mengadakan perubahan tampilan pada monitor.

Menu Favorites adalh menampilkan jendela favorit, yaitu mengelompokkan situs-situs tertentu sehingga jika akan mengaksesnya lebih cepat.

Menu Tools merupakan menu secara umum di gunakan untuk mengakses Outlook Express.

Menu Help secara umum digunakan untuk meminta bantuan atau tutorial untuk memeacahkan maasalah operasional dalam Internet Explorer.

38. menu dan fungsi MS world tool-file-edit-layout-view​


Penjelasan:

1. FILE !!

Pada tools di file tab ini, terdapat banyak menu yang memiliki fungsinya masing-masing. Opsi yang disediakan pada file tab ini, yakni:

New Document, berfungsi untuk membuka atau membuat file baru.

Recent, berfungsi untuk membuka file yang sebelumnya telah dikerjakan.

Open, berfungsi untuk membuka file yang tersimpan atau terunggah di folder.

Save/Save As, berfungsi untuk menyimpan file yang baru saja dikerjakan.

Print, berfungsi untuk mencetak dokumen atau laporan yang telah dikerjakan di Microsoft Word.

EDIT !!

Untuk mengedit/mereview sebuah projek yang akan dibuat agar hasilnya maksimal.

view !!

View buttons merupakan tools yang berguna untuk mengubah tampilan dokumen. View buttons ini juga memiliki beberapa tools di dalamnya yang tentu memiliki berbagai macam fungsi, yakni:

Web Layout, yang berguna untuk melihat layout dokumen apabila ditampilkan di laman website.

Full Screen Reading, yang bermanfaat untuk melihat dan juga membaca dokumen dalam ukuran yang maksimal.

Print Layout, yang berguna untuk melihat hasil kerja apabila di cetak.

Outline View, yang bermanfaat untuk melihat hasil kerja yang sesuai dengan urutan dan juga bagannya.

Draft View, yang berguna untuk melihat hasil kerja sesuai dengan konsep, sehingga jika salah bisa cepat dibenarkan.

layout !!

Page layout termasuk salah satu fitur yang terdapat di dalam Microsoft Office, sesuai dengan penjelasan di laman resmi Microsoft Office Word disebutkan bahwa page layout adalah salah satu menu dengan fungsi utama sebagai pengatur atau alat yang digunakan untuk mengatur tampilan pada sebuah dokumen
