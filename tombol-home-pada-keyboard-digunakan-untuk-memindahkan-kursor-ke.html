12. tombol keyboard untuk memindahkan kursor ke kanan atau ke kiri atau ke atas atau ke bawah adalah tombol​ tolong ya kakak yang ganteng sama cantik


Jawaban:

Pertanyaan yang kamu maksud agaknya kurang jelas, terdapat dua yang mendekati maksudnya adalah untuk memindahkan text pointer ke kanan atau ke kiri atau ke atas atau ke bawah atau untuk memindahkan kursor menggunakan keyboard.

Penjelasan:

Jawaban untuk memindahkan text pointer ke ke kanan atau ke kiri atau ke atas atau ke bawah adalah dengan menggunakan arrow keyboard yaitu tombol pada keyboard yang bersimbol panah (kanan, kiri, atas, dan bawah). Tombol tersebutlah yang berfungsi untuk memindahkan text pointer disaat kamu sedang mengetik menggunakan aplikasi office seperti microsoft office word. selain itu juga dapat memindah pointer alternative (alt) yang dapat digunakan dengan keyboard pada icon - icon di suatu software.

Jawaban untuk memindahkan kursor menggunakan keyboard adalah dengan mengaktifkan fungsi MouseKey dengan menggunakan kombinasi Alt (kiri) + Shift (kiri) + Num Lock, kemudian kursor akan dapat digerakkan menggunakan keyboard menggunakan tombol numeric 1, 2, 3, 4, 6, 7, 8, 9. Tombol 4 & 6 untuk bergerak ke kanan dan kiri, 8 & 2 untuk keatas dan bawah, sedangkan lainnya untuk bergerak diagonal sesuai dengan posisi. Kemudian untuk melakukan klik kiri yaitu dengan tombol 5.

25. sebutkan dan jelaskan tombol keyboard untuk memindahkan kursor


Tombol keyboard untuk memindahkan kursor adalah sebagai berikut

Tombol panah kiri. Memindahkan kursor sebanyak satu huruf ke kiri Tombol panah kanan. Memindahkan kursor sebanyak satu huruf ke kanan Tombol panah atas. Memindahkan kursor sebanyak satu huruf ke atas Tombol panah bawah. Memindahkan kursor sebanyak satu huruf ke bawah Tombol Home. Memindahkan kursor ke awal baris Tombol End. Memindahkan kursor ke akhir baris Tombol Ctrl + panah bawah. Memindahkan kursor ke awal paragraf berikutnya Tombol Ctrl + panah atas. Memindahkan kursor ke paragraf sebelumnya Tombol Ctrl + Home. Memindahkan kursor ke awal teks Tombol Ctrl + End. Memindahkan kursor ke akhir teks
