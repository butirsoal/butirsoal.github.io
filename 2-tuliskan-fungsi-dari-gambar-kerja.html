1. empat perangkat utama komputer

Perangkat masukan (input) : perangkat keras yang digunakan untuk memasukan data, misalnya saja keyboard digunakan untuk menginput huruf, angka, simbol dsbPerangkat pemrosesan (processing unit) : perangkat pemroses data utama dalam sebuah komputer, atau otak komputer itu sendiri.Perangkat penyimpanan (storage device) : perangkat keras untuk menyimpan data komputer.perangkat keluaran (output) : perangkat keras yang digunakan untuk mengeluarkan data, kebalikan dari input, contohnya printer

2. caratan : Penjelasan GAMBAR yang terlampir

A. hardware (perangkat keras) : perangkat dari komputer tersebut yang fisiknya terlihat jelas, misalnya motherboard, keyboard, printer, dsb

penjelasan perangkat input, processing, storage dan output ada di jawaban nomer 1 diatas

B. Software (perangkat lunak) : perangkat dari komputer yang tidak kasat mata yang berfungsi untuk memproses permintaan user melalui input dari hardware, misal nya microsoft word yang digunakan untuk membuat

OS (Operating System) :  perangkat lunak sistem yang mengatur sumber daya dari perangkat keras dan perangkat lunak, serta sebagai daemon (melakukan tugas tanpa intervensi user) untuk program komputer. Windows, linux, merupakan contoh dari OS.Aplikasi :  subkelas perangkat lunak komputer yang memanfaatkan kemampuan komputer langsung untuk melakukan suatu tugas yang diinginkan pengguna, misalnya aplikasi grafis, program, database.

C. Brainware : brainware adalah user atau pemakai suatu komputer itu sendiri sesuai kebutuhannya.

System analysis : seseorang yang bertanggung jawab dalam merekomendasikan suatu perangkat lunak atau sistem yang cocok bagi proses bisnis perusahaan.Programmer : seseorang yang bertujuan untuk membuat sebuah sistem, aplikasi menggunakan bahasa pemrograman.Administrator : seorang yamg bertugas untuk mengelola hal-hal yang berhubungan dengan komputer, misal mengatur database.Operator :  Seseorang yang bertanggung jawab atas semua peralatan yang ada dalam sistem komputerisasi, memeriksa dan mencoba komputer dan peralatan lain apakah dapat dipergunakan sebagaimana mestinya dan menonfungsikan peralatan bila tidak dipergunakan
