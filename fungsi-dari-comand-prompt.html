1. Apa Fungsi Comand Prompt (CMD) ?


ada banyak fungsi nya salah satu nya untuk melihat waktu1. meng-handle beberapa masalah saat versi GUI (Grafik User Interface) pada Windows kita bermasalah diakibatkan virus. Seperti hilangnya folder options pada explorer, task manager yang di-disable, tidak bisa membuka msconfig dan sebagainya. Lebih lanjut kita bisa menangani virus lewat cmd. 2. Seperti namanya, “Prompt” sendiri arti harafiahnya adalah Quick atau cepat jadi kita bisa mengeksekusi sebuah perintah lebih cepat dengan menggunakan cmd. Selain lebih cepat cmd ini juga lebih ringan dibanding dengan explorer saat melakukan perintah masuk pada direktori tertentu, atau mencari file tertentu. 3. Kegunaan lain dari Command Prompt ini adalah untuk membiasakan menggunakan command line pada cmd agar nantinya terbiasa dengan server core yang menggunakan OS berbasis text.

16. Jelaskan pengertian dan sebutkan fungsi, internal comand dan eksternal comand


yang saya tau 
Internal command adalah Perintah-perintah yang telah tersimpan dalam memory computer saat pembacaan DOS selesai, seperti perintah Dir, Copy, Del, Ren, Type dan Cls. Internal command adalah bagian dari DOS’ COMMAND.COM file. Jadi jika menghapus COMMAND.COM perintah-perintah dalam DOS tidak bisa digunakan
External Command DOS karena untuk melaksanakan seluruh perintah-perintah berikut ini diperlukan program eksekusi yang berekstensi COM atau EXE. Lokasi eksternal command adalah di direktori Windowssystem32 atau Winntsystem32. Jika eksternal command di direktori tersebut dihapus, di-rename atau di pindah, perintah-perintah tersebut masih bisa ditemukan di MS-DOS.

25. apa perbedaan antara internal comand dan eksternal comand


Internal command adalah Perintah-perintah yang telah tersimpan dalam memory computer saat pembacaan DOS selesai, seperti perintah Dir, Copy, Del, Ren, Type dan Cls. Internal command adalah bagian dari DOS’ COMMAND.COM file. Jadi jika menghapus COMMAND.COM perintah-perintah dalam DOS tidak bisa digunakan.


Disebut External Command DOS karena untuk melaksanakan seluruh perintah-perintah berikut ini diperlukan program eksekusi yang berekstensi COM atau EXE. Lokasi eksternal command adalah di direktori Windowssystem32 atau Winntsystem32. Jika eksternal command di direktori tersebut dihapus, di-rename atau di pindah, perintah-perintah tersebut masih bisa ditemukan di MS-DOS.

38. pengertian internal command dan eksternal comand?


Internal dan external command adalah dua jenis instruksi yang dapat diberikan kepada perangkat komputer.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menyajikan pengertian internal dan eksternal command. Berikut kakak akan mencoba menjawab pertanyaan tersebut.

Internal command merupakan instruksi yang sejak awal ditempatkan dalam perangkat memori.

Contoh: Copy, Color, Cis dan lain sebagainya.

External command merupakan instruksi yang pengeksekusiannya membutuhkan aplikasi eksternal di luar perangkat memori, seperti .exe.
