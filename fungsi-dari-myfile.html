37. jelaskan yang dimaksud dengan a.file executable b.file konvensi​


Jawaban:

File executable adalah jenis file yang dapat dijalankan langsung oleh sistem operasi. Biasanya, file ini memiliki ekstensi .exe atau .com, dan dapat digunakan untuk menjalankan program atau aplikasi di komputer.

File konvensi adalah file yang dibuat sesuai dengan konvensi atau standar tertentu. Misalnya, file teks biasanya memiliki ekstensi .txt atau .doc, dan file gambar biasanya memiliki ekstensi .jpg atau .png. Dengan mengikuti konvensi ini, sistem operasi dapat mengenali jenis file dan menjalankannya dengan program yang tepat.

Jawaban:

A.file executable adalah file yang dapat dijalankan atau dieksekusi oleh komputer. Ini biasanya berisi kode atau instruksi yang dapat dieksekusi oleh sistem operasi untuk melakukan suatu tugas atau menjalankan suatu program.

B.file konvensi adalah file yang memiliki ekstensi tertentu dan dianggap sebagai standar untuk jenis tertentu dari file. Misalnya, file dengan ekstensi .txt biasanya dianggap sebagai file teks biasa, sedangkan file dengan ekstensi .docx biasanya dianggap sebagai file dokumen Microsoft Word. Ini membantu sistem operasi dan aplikasi lain untuk mengetahui bagaimana file tersebut harus dibuka dan ditangani.
