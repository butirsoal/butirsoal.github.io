2. 15. Untuk memberikan warna pada objek dapat dilakukan dengan cepat yaitu…. a. pada color Pallets klik kanan untuk garis luar objek dan klik kiri untuk untuk isi objek b. pada color palette klik kanan untuk isi objek dan klik kiri untuk garis luar objek c. pada toolbox aktifkan flyout fill tool d. pada toolbox aktifkan flyout pattern fill e. pada toolbox aktifkan flyout crop tool


Untuk memberikan warna pada objek dilakukan dengan cepat yaitu pada color pallets klik kanan untuk garis luar objek dan klik kiri untuk isi objek (a).



Pembahasan

Memberi warna pada objek di CorelDraw dapat dilakukan dengan berbagai cara, yaitu:

1. Color pallete

Color Pallete atau pallet warna yang berbentuk sebuah barisan warna, Color pallete bukan tools, melainkan sebuah jendela tampilan yang berisi warna-warna. Color Pallete ini adalah alat yang paling mudah diakses untuk memberi warna pada objek karena tidak memunculkan jendela tampilan ataupun property bar.

Color Pallete ini hanya menampilkan beberapa warna saja. Color Pallete Default biasanya terletak di samping sebelah kanan layar. Untuk mewarnai Objek dengan Color Pallete anda hanya tinggal menyeleksi objek yang akan diwarnai kemudian klik warna yang di inginkan.


2. Fill Color dan Line Color

Fill Color seperti namanya adalah tool untuk mewarnai bagian Isi Objek, dan Line Color adalah untuk mewarnai Garis.

Fill Color dan Line Color terletak di sebelah kanan bawah layar. Di dalam Fill Color tersedia semua warna, sehingga kita bisa memilih warna dengan lebih leluasa.

Terdapat beberapa jenis pewarnaan yang bisa kita aplikasikan dengan menggunakan Fill Color yaitu:

Uniform Fill : Untuk mewarnai dengan satu warna (Polos)Fountain Fill : Untuk mewarnai dengan Gradasi warnaPattern Fill : Untuk mewarnai dengan PatternTexture Fill : Untuk mewarnai dengan tekstur
3. Smart Fill Tool

Tool Ini terletak di bagian tengah Toolbox, Smart Fill Tool berfungsi untuk memberi warna atau menghilangkan warna pada objek atau bagian dari objek.


4. Interactive Fill Tool

Interactive Fill Tool berada di paling bawah Toolbox, fungsinya hampir sama dengan Fountain Fill hanya saja Interactive Fill Tool lebih mudah dalam mengatur arah gradasi warnanya.


5. Mesh Fill Tool

Mesh Fill Tool merupakan alat untuk mewarnai Objek dengan beberapa warna yang berbeda, mesh fill tool akan membentuk grid-grid yang ada di dalam objek yang bisa kita atur jumlah baris dan kolomnya, dimana di dalam setiap grid bisa kita aplikasikan warna yang berbeda. Untuk mengatur arah warnanya kita tinggal mengubah bentuk grid dengan menggerakkan titik-titik grid.
