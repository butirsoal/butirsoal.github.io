38. jelaskan kegunaan fungsi - ctrl - alt - esc - pause break - page up - home - insert

Berguna pada saat proses pengetikan. Menekan tombol Page Up akan mengakibatkan kursor berpindah ke atas sejauh satu layar dari kursor berada.Home = Mengembalikan posisi kursor pada awal kalimat
Insert/ins = Untuk me-replace per karakter.
ctrl alt:menjalankan perintah program.
esc:membatalkan
page up:menggulung layar ke atas seukuran pada monitor
home:memindahkan kursor paling kiri suatu baris
insert:menyisipkan karakter
