15. Hasil dari kode program berikut adalah... #include <iostream>using namespace std; int main() { int x = 157, y = ⁸; int hasil; hasil = x & y; cout<<hasil; return 0;} O 1 O 5 O 2 O 4 O 3 ​


Jawaban:

Kode program tersebut akan menghasilkan output berupa angka desimal, yaitu 8.

Penjelasan:

Baris pertama mendeklarasikan variabel x dengan nilai 157 dan variabel y dengan nilai ⁸. Namun, terdapat kesalahan penulisan pada variabel y yaitu menggunakan karakter superskrip yang salah, sehingga programnya akan mengalami error saat dijalankan.

Namun, bila kita asumsikan karakter superskrip itu sebenarnya adalah angka 8, maka program akan berjalan dan menghasilkan output 8.

Pada baris selanjutnya, nilai variabel hasil dihitung dengan operasi bitwise AND antara nilai variabel x dan y (dalam biner). Nilai biner 157 adalah 10011101, sedangkan nilai biner 8 adalah 00001000. Jika kedua nilai tersebut di-operasikan dengan AND, maka hasilnya adalah 00001000 (dalam biner) atau 8 (dalam desimal).

Program di atas adalah contoh program dalam bahasa pemrograman C++ yang menghitung luas segitiga berdasarkan nilai alas (a) dan tinggi (t) yang dimasukkan oleh pengguna. Berikut adalah penjelasan per baris dari program tersebut:

#include<iostream>: Ini adalah preprocessor directive yang memungkinkan program untuk menggunakan fungsi-fungsi dari pustaka input-output stream (iostream) dalam C++.

using namespace std;: Ini digunakan untuk menghilangkan penggunaan awalan std:: sebelum fungsi-fungsi dari pustaka iostream.

int main(): Ini adalah deklarasi fungsi utama (main function) dari program.

{: Tanda buka kurung kurawal menandai awal dari blok kode fungsi main.

cout << "alas=";: Menggunakan objek cout untuk mencetak teks "alas=" ke layar.

cin >> a;: Menggunakan objek cin untuk menerima input dari pengguna dan menyimpannya di dalam variabel a.

cout << "tinggi=";: Menggunakan objek cout untuk mencetak teks "tinggi=" ke layar.

cin >> t;: Menggunakan objek cin untuk menerima input dari pengguna dan menyimpannya di dalam variabel t.

Luas = 0.5 * a * t;: Menghitung luas segitiga dengan rumus Luas = 0.5 * alas * tinggi dan menyimpan hasilnya dalam variabel Luas.

cout << "jadi luas=" << Luas << endl;: Menggunakan objek cout untuk mencetak teks "jadi luas=" diikuti oleh nilai dari variabel Luas dan tambahan end-of-line (endl) ke layar.

return 0;: Mengembalikan nilai 0 ke sistem operasi untuk menandakan bahwa program berjalan dengan sukses.

}: Tanda tutup kurung kurawal menandai akhir dari blok kode fungsi main.

Program ini akan membaca nilai alas dan tinggi dari pengguna, kemudian menghitung dan mencetak luas segitiga berdasarkan rumus yang diberikan
