32. bantuin pliss pke craa​


Grafik di atas mempotong sumbu ( -6, 0 ) ( 3, 0 ) dan melalui titik ( 2, 8 ) pada grafik, maka persamaannya adalah :

y = a (x - x1) (x - x2)

8 = a (2 - (-6)) (2 - 3)  

8 = a (2 + 6) (2 - 3)

8 = a (8) (-1)

8 = -8a

a = 8/-8

a = -1

Kemudian substitusikan a ke y = a ( x - (-6)) ( x - 3), maka :

y = -1 (x - (-6)) (x - 3)

y = -1 (x²- 3x + 6x - 18)

y = -1 (x² + 3x - 18)

y = -x² - 3x + 18

Jadi persamaan grafik di atas adalah  y = 18 - 3x - x²

Jawaban : B
