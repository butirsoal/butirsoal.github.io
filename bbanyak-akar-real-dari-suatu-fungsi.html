3. AYO DIBANTU KAK CEPETJenis akar dari persamaan x²-4=0 adalah ... a. Tidak real dan akar berbedab. Tidak real dan akar samac. Real dan akar samad. Tidak reale. Real dan akar berbeda​


Jawab:

Jenis akar - akar dari persamaan x² - 4x + 4 = 0 adalah Real Kembar

Pembahasan

Persamaan kuadrat adalah persamaan yang memiliki variabel pangkat tertinggi 2.

Bentuk umum persamaan kuadrat

ax² + bx + c = 0 ; a ≠ 0

Jenis Akar

Untuk menentukan jenis akar pada persamaan kuadrat, Kita harus menentukan nilai diskriminan terlebih dahulu.

Rumus Diskriminan

D = b² - 4ac

Setelah mengetahui nilai diskriminan, maka tentukan jenis akar dengan memperhatikan syarat berikut.

D > 0, maka suatu persamaan kuadrat akan memiliki dua akar real yang tidak sama besar (x1 ≠ x2).

D = 0, maka suatu persamaan kuadrat akan memiliki dua akar real dan kembar.

D < 0, maka suatu persamaan kuadrat tidak memiliki akar real (akarnya imajiner).

D = diskriminan

..

Dari penjelasan tersebut mari selesaikan soal berikut.

Diketahui :

x² - 4x + 4 = 0

Ditanya :

Jenis akar ?

Jawab :

Langkah pertama, mencari nilai diskriminan

x² - 4x + 4 = 0

a = 1

b = -4

c = 4

Substitusi a, b, dan c ke rumus Diskriminan

D = b² - 4ac

D = (-4)² - 4(1)(4)

D = 16 - 16

D = 0

Karena D = 0, maka jenis akarnya memiliki dua akar real dan kembar. (C)

8. Buatlah tiga buah fungsi polinomial yang berderajat 4 dan hanya memilki akar real 2 dan -2 !


Penjelasan dengan langkah-langkah:

Untuk membuat tiga buah fungsi polinomial dengan derajat 4 dan akar real 2 dan -2, kita dapat menggunakan faktor-faktor linier yang terkait dengan akar tersebut.

Pertama, kita tahu bahwa jika suatu angka r adalah akar dari suatu polinomial, maka faktor linier yang terkait adalah x - r.

Dalam hal ini, kita tahu bahwa akar-akar tersebut adalah 2 dan -2, sehingga faktor linier terkait adalah (x - 2) dan (x + 2).

Maka, kita dapat membuat tiga fungsi polinomial dengan derajat 4 yang memiliki akar real 2 dan -2 sebagai berikut:

1.

[tex]f(x) = (x - 2)(x + 2)(x - a)(x - b) [/tex]

2.

[tex] g(x) = (x - 2)(x + 2)(x - c)(x - d) [/tex]

3.

[tex] h(x) = (x - 2)(x + 2)(x - e)(x - f)[/tex]

Di mana a, b, c, d, e, dan f adalah angka bebas, sehingga kita dapat menghasilkan fungsi polinomial yang berbeda dengan menggunakan berbagai kombinasi nilai-nilai tersebut.

1.  D ≥ 0 : akar real/nyata

4.  D < 0 : akar tidak real (imajiner)
