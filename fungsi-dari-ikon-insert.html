4. sebutkan dan jelaskan fungsi dari ikon ikon yg terdapat di menu insert


Didalam microsoft word, terdapat menu "Insert" yang berguna untuk memasukkan ataupun menyisipkan gambar, tabel, ataupun simbol. Berikut merupakan ikon yang terdapat pada menu "Insert" yaitu :

1. Pages :

a) Cover page, untuk mengganti cover halaman.

b) Blank page, untuk membuat halaman baru.

c) Page break, untuk memisahkan halaman yang berbeda.

2. Illustrasions :

a) Picture, untuk menambahkan file gambar.

b) Clip art, untuk menambahkan gambar dari folder clip art.

c) Shapes, untuk menambahkan bentuk pola simple.

d) Smart art, untuk menambahkan bentuk pola yang lebih kompleks.

e) Chart, untuk menambahkan grafik.

3. Link :

a) Hyperlink, untuk menambahkan url website.

b) Bookmark, untuk membuat halaman favorite.

c) Cross-reference, untuk membuat halaman referensi/rujukan.

4. Header dan footer :

a) Header, untuk membuat space ketikan di bagian atas kertas.

b) Footer, untuk membuat space ketikan di bagian bawah kertas.

c) Page number, untuk membuat nomor halaman.

5) Text :

a) Text box, untuk membuat text dalam kotak.

b) Quick parts, untuk mengatur teks secara otomatis.

c) Word Art, untuk mengubah gaya teks.

d) Dropcap, untuk memperbesar huruf di awal paragraf.

e) Signature line, untuk membuat tanda tangan dan nama.

f) Date and time, untuk menambahkan tanggal dan waktu.

g) Object, untuk menambahkan file.

6. Tables, untuk membuat tabel.

7. Symbol :

a) Equation, untuk menambahkan simbol rumus.

b) Symbol, untuk menambahkan simbol lambang.

Pembahasan

Microsoft word adalah salah satu software dalam microsoft office yang digunakan untuk mengolah tulisan. Microsoft word digunakan untuk keperluan seperti :

Membuat dokumen,Membuat surat,Membuat tabel, dan lain-lain.

Didalam microsoft word, terdapat menu "Insert" yang berguna untuk memasukkan ataupun menyisipkan gambar, tabel, ataupun simbol. Terdapat 7 macam grup ikon pada menu "Insert" yaitu pages, tables, illustration, link, header and footer, text, dan symbol.

10. tolong jawab secepatnya- Kegunaan Margins - Cara costum margins - cara print dokumen - fungsi ikon font size - fungsi ikon color - fungsi dari ikon page border - cara membuat page border - fungsi ikon insert picture - Perbedaan insert foto dari explorer dan internet - fungsi dari ikon coloms di page layout - kegunaan explorer - cara membuat folder.


Jawaban:

Kegunaan Margins= Fungsi margin pada microsoft word adalah untuk mengatur jarak antara tepi pengetikan dengan tepi halaman kertas.

Cara costum margins= 1. Pada tab Page Layout, grup Page Setup, klik Margins dan kemudian pilih Custom Margins.

2. Pada kotak dialog Page Setup, tab Margins, isi margin untuk kiri (Left), kanan (Right), atas (Top), dan bawah (Bottom).

3. Klik OK untuk menutup kotak dialog.

Cara print dokumen= 1. Di komputer, buka Chrome.

2. Buka halaman, gambar, atau file yang ingin dicetak.

3. Klik File. Cetak. Atau, gunakan pintasan keyboard: Windows & Linux: Ctrl + p. Mac: ⌘ + p.

4. Di jendela yang muncul, pilih tujuan dan ubah setelan cetak pilihan Anda.

Klik Cetak.

Fungsi ikon font size= Font Size, berfungsi untuk menggantu ukuran font sesuai keperluan pengguna. Ukuran font tersedia dalam rentang skala 8 - 72. Font Color, berfungsi untuk mengganti warna font sesuai keperluan pengguna.

Fungsi ikon color= Pen color berfungsi Untuk memberikan warna pada gambar.Atau memberikan warna pada karakter.

Fungsi dari ikon page border= Page Borders digunakan apabila isi dari suatu halaman ingin diberi bingkai, corak atau warna latar belakang tertentu agar tampilannya lebih indah dan menarik.

Cara membuat page border=

Langkah-langkah:

Pada tab Page Layout (Word 2007 – 2010) atau tab Design (Word 2013 ke atas), Page Background grup, klik Page Borders.

Page Border - Page Background Group MS Word 2010

Akan muncul Page Border tab pada kotak dialog Borders and Shading.

Page Border - Borders And Shading MS Word 2010

Di bagian Setting, pilih bentuk bingkai yang diinginkan, misalnya:

Box: untuk membuat bingkai pada seluruh sisi halaman.

Shadow: untuk membuat bingkai dengan efek bayangan.

3-D: untuk membuat bingkai dengan efek 3-D. Catatan: efek 3-D hanya bisa digunakan pada gaya garis tertentu saja.

Custom: untuk membuat bingkai pada sisi tertentu halaman. Caranya: di bagian Preview, klik tombol atas, bawah, kanan, atau kiri.

Buat pengaturan pada:

Style: untuk menentukan gaya garis bingkai.

Color: untuk menentukan warna garis bingkai.

Width: untuk menentukan ketebalan garis bingkai.

Art: untuk memilih bingkai yang artistik, misalnya gambar bintang, pohon, dan lain-lain.

Fungsi ikon pictures=Untuk menyisip kan gambar dari media lain!

Perbedaan insert foto dari explorer dan internet= Insert foto dari explorer artinya kita menginput sebuah foto/gambar dari internal/dalam yaitu dokumen sendiri atau yang telah disediakan. Sedangkan dari internet artinya kita menginput sebuah foto/gambar dari eksternal/luar sebagai contoh dari Google Images

Fungsi dari ikon coloms di page layout= Columns: digunakan untuk mengatur jumlah kolom teks. Breaks: digunakan untuk mengatur kontinuitas halaman maupun kolom teks. Line Numbers: digunakan untuk mengatur kontinuitas nomor baris teks.

Kegunaan explorer= Dengan Windows Explorer, pengguna dapat melakukan hal-hal berikut: Membuat, memindahkan, menyalin, membuka, menyunting, atau menghapus berkas dan folder di dalam komputer. Mencari berkas-berkas dengan menggunakan query yang kompleks.

Cara membuat folder= 1.Buka File Explorer menggunakan salah satu cara berikut: ...

2.Navigasikan ke tempat yang Anda inginkan untuk membuat folder baru, lalu klik Folder Baru.

3.Ketikkan nama folder Anda, lalu tekan Enter.

24. fungsi menu dan ikon pada tab insert


Fungsi menu dan icon pada tab insert adalah untuk menyisipkan objek kedalam naskah atau tulisan. Pada tab insert kita dapat menyisipkan tabel, gambar, teks, grafik, link dan lainnya.  Icon pada tab insert dikelompokkan kedalam beberapa group.

Pembahasan

Salah satu menu yang terdapat pada microsoft office adalah insert. Tab insert berfungsi untuk menyisipkan objek kedalam lembar kerja. Terdapat beberapa kelompok icon pada tab insert yang terdapat pada microsoft word 2010, yaitu:

Group Pages, terdiri dari Cover Page, Blank Page dan Break Page.Group Tables, terdiri dari Insert Table, Draw Table, Convert Text to Table, Excel Spreadsheet , dan Quick Table.Group Illustrations, terdiri dari Picture, Clip Art,  Shapes, SmartArt, dan Chart.Group Links, terdiri dari Hyperlink, Bookmark, dan Cross-Reference.Group Header and Footer, terdiri dari Header, Footer dan Page Number.Group Text terdiri dari Text Box, Quick Parts, Quick Parts, Drop Cap, Signature Line, Date & Time, dan Object.Group Symbols, Equation dan Symbol.

Jawaban:

menu insert terbagi menjadi 7 grup, yaitu pages, tables,illustrasion, links, header & footer,text, dan symbol.

1. fungsi icon di group pages:

a. cover page : untuk mengganti cover halaman Ms. Word

b. blank page : untuk membuat halaman baru

c. page break : untuk memisahkan halaman yang berbeda

2. fungsi icon di group illustrasions

a. picture : untuk menambahkan/menyisipkan file gambar

b. clip art : untuk menyisipkan/menambahkan gmbar dri folder clip art

c. shapes: untuk menambahkan gmbr sprti: line,bassic shapes,dll

d. smart art: untuk menambahkan icon list,dll.

e. chart : untuk menambahkan grafik column,dll.

3. fungsi icon group Link

a. Hyperlink : untuk menambahkan url website

b. bookmark : untuk membuat halaman favorite

c. Cross-reference : untuk membuat halaman referensi/rujukan

4. fungsi icon group header & footer

a. Header : untuk membuat halaman header di halaman doc atas

b. footer : untuk membuat halaman footer di halaman doc bawah

c. page number : untuk membuat nomer di halaman header dan footer

5. fungsi icon goup text

a. text box : untuk membuat teks dalam kotak

b. quick parts : untuk mengatur teks secara otomatis

c. word art: untk mengubah gaya teks sesuai keinginan kita

d. dropcap: untuk memperbesar huruf di awal paragraf

e. signature line : untuk membuat tanda tangan dan nama

f. date and time : untk menambahkan tgl dan waktu di dokumen

g. object : untk menambhkan file di dokumen

6.fungsi icon pada group tables

a. table : untuk membuat tabel di halaman dokumen

7.fungsi icon pada group symbol

a. equation: untuk menambahkan simbol rumus yang digunakan matematika

b. symbol : untuk menambahkan simbol
