c. lulus dlm 2 mat kul =

= P(A)' x P(B)' x P(C)'

kode program:

def bubble_sort(array):

   for i in range(len(array) - 1):

       for j in range(len(array) - i - 1):

           if array[j] > array[j + 1]:

               array[j], array[j + 1] = array[j + 1], array[j]

   return array

array = [11, 9, 6, 10, 9, 1, 3, 5, 2]

sorted_array = bubble_sort(array)

print(sorted_array)

2. selection sort:

b. start (flowchart)

initialize i = 0

while i < len(array) - 1

   initialize min_index = i

   for j = i + 1 to len(array)

       if array[j] < array[min_index]

           set min_index = j

   swap array[i] and array[min_index]

   increment i by 1

end

print sorted array

kode program:

def selection_sort(array):

   for i in range(len(array)):

       min_index = i

       for j in range(i + 1, len(array)):

           if array[j] < array[min_index]:

               min_index = j

       array[i], array[min_index] = array[min_index], array[i]

   return array

array = [11, 9, 6, 10, 9, 1, 3, 5, 2]

sorted_array = selection_sort(array)

print(sorted_array)

Keharusan pemberian mata kuliah Pendidikan Kewarganegaraan di tingkat perguruan tinggi dapat dievaluasi berdasarkan sejauh mana mata kuliah ini mampu mengatasi dan mencerminkan dinamika permasalahan bangsa yang terjadi akhir-akhir ini. Dalam konteks ini, mata kuliah tersebut harus menyediakan pemahaman yang komprehensif tentang isu-isu ini, mendorong pemikiran kritis, dan mengembangkan kemampuan mahasiswa untuk berpartisipasi dalam masyarakat secara aktif dan bertanggung jawab.

Selain itu, mata kuliah Pendidikan Kewarganegaraan juga dapat menyentuh aspek digital dan teknologi dalam kewarganegaraan, seperti etika digital, privasi data, pengaruh media sosial, dan keamanan siber. Hal ini penting mengingat peran yang semakin besar teknologi informasi dan komunikasi dalam kehidupan sehari-hari, termasuk dalam konteks kewarganegaraan.

Mata kuliah ini mencakup berbagai topik, seperti sistem politik dan pemerintahan, konstitusi dan hukum, demokrasi, hak asasi manusia, pluralisme, keberagaman budaya, ekonomi nasional, globalisasi, lingkungan hidup, serta peran dan tanggung jawab individu dalam masyarakat.

37. bagaimana penerapan viskositas dalam kehidupan sehari-hari, tugas mata kuliah fisika


Viskositas adalah sifat rheology sebuah fluida. Artinya parameter ini menentukan sifat alir dari sebuah fluida. Semakin viscous sebuah fluida maka tekanan yang dibutuhkan untuk mengalirkannya akan semakin besar. Contoh, kita ingin mengalirkan air dari titik A dan titik B yang horizontal. Pompa yang digunakan akan lebih ringan daripada misalnya kita mengalirkan sirup. Hal ini dikarenakan sirup lebih viscous daripada air.
